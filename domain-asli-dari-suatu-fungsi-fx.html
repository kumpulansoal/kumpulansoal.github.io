20. Jika domain (xlx 6,xE asli) tentukan range dari fungsi f:x x+3


Jawaban:

Fungsi f(x) = x + 3 akan memiliki range yang bergantung pada domain yang diberikan. Dalam kasus ini, domain adalah {x | x adalah bilangan asli positif (x > 0)}.

Untuk menentukan range dari fungsi ini, kita perlu mempertimbangkan bagaimana nilai x memengaruhi output fungsi (f(x) = x + 3). Karena x adalah bilangan asli positif, maka nilai minimum x adalah 1. Dengan demikian, nilai minimum dari f(x) adalah:

f(1) = 1 + 3 = 4

Selanjutnya, karena x adalah bilangan asli, nilai x akan terus bertambah. Oleh karena itu, range dari fungsi ini adalah semua bilangan bulat positif yang lebih besar atau sama dengan 4, yaitu {y | y adalah bilangan bulat positif (y ≥ 4)}.

Jadi, range dari fungsi f(x) = x + 3 dengan domain {x | x adalah bilangan asli positif (x > 0)} adalah {y | y adalah bilangan bulat positif (y ≥ 4)}.

Jawaban : Daerah hasil = { -13, -3, 3, 5 }

Perhatikan penjelasan berikut.

Ingat! Daerah hasil atau range adalah himpunan yang berisi pasangan di daerah asal atau domain. Jadi untuk mencari daerah hasil, dapat disubstitusikan nilai domain ke dalam fungsi.

Pembahasan:

Fungsi : f(x) = 5 - 2x²

Domain : { x | -2 ≤ x ≤ 3, x bilangan bulat} = { -2, -1, 0, 1, 2, 3}

Dengan memasukkan masing-masing nilai domain ke dalam persamaan didapatkan:

f(-2) = 5 - 2(-2)² = 5 - 2(4) = 5 - 8 = -3

f(-1) = 5 - 2(-1)² = 5 - 2(1) = 5 - 2 = 3

f(0) = 5 - 2(0)² = 5 - 2(0) = 5 - 0 = 5

f(1) = 5 - 2(1)² = 5 - 2(1) = 5 - 2 = 3

f(2) = 5 - 2(2)² = 5 - 2(4) = 5 - 8 = -3

f(3) = 5 - 2(3)² = 5 - 2(9) = 5 - 18 = -13

Jadi, daerah hasil fungsi tersebut adalah { -13, -3, 3, 5 }
