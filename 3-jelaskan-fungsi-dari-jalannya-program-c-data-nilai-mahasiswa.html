Berikut adalah jawabannya dalam bahasa C++

#include <iostream>

using namespace std;

int main()

{

   int studentScore[10];

   int studentCount = 10;

   int maxScore = 0;

   int occurance = 1;

   for(int i=0; i<studentCount; i++)

   {

       cout << "Insert " <<  i+1 <<" Student Score: ";

       cin >> studentScore[i];

   }

   maxScore = studentScore[0];

   for(int i=0; i<studentCount; i++)

   {

       if(maxScore<studentScore[i])

       {

           occurance = 1;

           maxScore = studentScore[i];

       }

       else if(maxScore == studentScore[i])

       {

           occurance++;

       }

   }

   cout << "The Highest Score: " << maxScore << endl << "Student who have highest score count: " << occurance << endl;

   cin.get();

   return 0;

}

12. Buatlah program untuk menerima daftar nilai mahasiswa sbb : Input : - Jumlah data (n) - Nilai ke-1 s/d Nilai ke-n Output : - Nilai minimal - Nilai maksimal 　　　　 - Nilai rata-rata (rata-rata = nilai total / jumlah data)


Program untuk menerima daftar nilai mahasiswa dengan menggunakan Python 3.10 adalah:

daftar_nilai = []

is_on = True

while is_on:

   nilai_mahasiswa = input('Silakan masukkan nilai (masukkan x untuk mengakhiri):\n')

   if nilai_mahasiswa == 'x':

       is_on = False

   elif nilai_mahasiswa != 'x':

       daftar_nilai.append(int(nilai_mahasiswa))

       

nilai_minimal = min(daftar_nilai)

nilai_maksimal = max(daftar_nilai)

nilai_rata2 = sum(daftar_nilai) / len(daftar_nilai)

print(f'Nilai minimal adalah: {nilai_minimal}')

print(f'Nilai maksimal adalah: {nilai_maksimal}')

print(f'Nilai rata-rata adalah: {nilai_rata2}')

Pembahasan

Program untuk menerima daftar nilai mahasiswa di atas bisa kita rencanakan dengan membuat pseudo-code terlebih dahulu:

1. Buat list kosong untuk diisi dengan daftar nilai.

2. Minta input nilai-nilai dari user.

3. Bila input tersebut 'x' akhiri loop.

4. Bila input bukan 'x' masukkan ke dalam list daftar nilai.

5. Cari nilai terkecil.

6. Cari nilai terbesar.

7. Cari nilai rata-rata dengan membagi jumlah total nilai dengan banyaknya nilai yang dimasukkan.

8. Print masing-masing hasilnya ke layar.

Kemudian kita buat baris kode sesuai pseudo-code di atas ke dalam program Python 3.10 (atau bahasa lainnya). Baris kode di atas sangat mudah dan mendasar, menggunakan teknik loop, list, if-else, dan operasi matematika dasar dalam Python.

25. diketahui data berat badan (kg) dari 100 orang mahasiswa suatu universitas ,yaitu 60-62 sebanyak 15 mahasiswa, 63-65 sebanyak 18 mahasiswa,66-68 sebanyak 32 mahasiswa,69-71 sebanyak 10 mahasiswa,dan 72-74 sebanyak 25 mahasiswa,berapakah nilai modus dari data tersebut


Jadi, nilai modus dari data tersebut adalah 66,62.

Pembahasan

Modus adalah data (nilai) yang paling sering muncul atau mempunyai frekuensi paling tinggi.

Rumus untuk mencari modus data sebagai berikut.

1. Modus data tunggal

  Cara menentukan modus pada data tunggal adalah dengan melihat data yang paling sering muncul atau mempunyai frekuensi paling tinggi.

2. Modus data berkelompok

Untuk data berkelompok, modus dapat dicari dengan menggunakan rumus berikut.

[tex]Mo = L + (\frac{d_1}{d_1 + d_2}) c\\[/tex]

dengan

Mo = modus

L = tepi bawah kelas modus

d₁ = selisih frekuensi kelas modus dengan frekuensi kelas sebelumnya

d₂ = selisih frekuensi kelas modus dengan frekuensi kelas sesudahnya

c = panjang kelas (tepi atas - tepi bawah)

Penyelesaian

diket:

berat badan        frekuensi

         (kg)

   60 - 62                  15

   63 - 65                   18

   66 - 68                   32 ---> kelas modus

   69 - 71                   10

   72 - 74                   25

ditanya:

nilai modus data tersebut....?

jawab:

Kelas modus terletak pada frekuensi terbanyak 32, yaitu pada interval 66 - 68, sehingga

L = 66 - 0,5 = 65,5

c = 68,5 - 65,5 = 3

d₁ = 32 - 18 = 14

d₂ = 32 - 10 = 22

[tex]Mo = L + (\frac{d_1}{d_1 + d_2}) c\\[/tex]

      [tex]= 65,5 + (\frac{14}{14 + 22}) \: . \: 3\\[/tex]

      [tex]= 65,5 + (\frac{14}{38}) \: . \: 3\\[/tex]

      [tex]= 65,5 + 1,12\\[/tex]

[tex]Mo = 66,62[/tex]

Kesimpulan

Jadi, nilai modus dari data tersebut adalah 66,62.
