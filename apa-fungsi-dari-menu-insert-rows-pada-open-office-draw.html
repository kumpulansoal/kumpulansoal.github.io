13. Apa fungsi dari insert rows above pada microsoft word 2007?


Insert digunakan untuk membuat tabel
Columns to the left berfungsi menambahkan kolom pada bagian kiri
Columns to the right berfungsi menambahkan kolom pada bagian kanan
Rows Below berfungsi menambahkan baris pada bagian bawah tabel
Rows Above berfungsi menambahkan baris pada bagian atas
Cells berfungsi untuk menambahkan baris atau kolom
Split cells berfungsi untuk membagi baris dan kolom
Autofit berfungsi untuk menyesuaikan ukuran kolom dan baris secara otomatis
Sort berfungsi untuk menyusun huruf secara alfabet
Formula berfungsi untuk menjumlah angka pada tabel dengan rumus seperti pada Ms Excel.

36. perintah copy dan paste pada libre office draw terdapat pada menu​


Clipboard adalah bagian yang terdiri dari copy, cut, paste, Dan format painter

Menu Edit berfungsi untuk mengedit (melakukan perubahan dengan cara menambah, menghapus, memotong, menyalin, dan lain-lain).

Menu yang ada di Menu Edit :

Undo : Untuk kembali ke perintah sebelumnya.

Redo : Untuk Menuju perintah setelah di Undo.

Repeat : Untuk mengulangi perintah user.

Cut : Untuk memotong teks.

Copy : Untuk Menyalin teks.

Paste : Untuk menampilkan hasil salinan.

Paste Special : Untuk menampilkan hasil salinan dengan perintah tambahan.

Select Text : Untuk menandai teks.

Selection : Untuk mengatur mode penanda.

Select All : Untuk menandai semua teks.

Jadi, jawabannya menu EDIT.


37. Rangkum lah tentang menu "insert" dan sub menu insert pada microsoft officetolong dibantu kak​


Jawaban:

Menu insert berisi tentang berikut ini:

Page, berfungsi untuk menyisipkan halaman.

Tables, berfungsi untuk menyisipkan tabel.

Illustration, berfungsi untuk menyisipkan beberapa hal berikut ini:

Picture (gambar).

Clip Art (gambar yang disediakan satu paket dengan Microsoft Office).

Shapes (objek berbagai bentuk, misalnya segi empat, segi tiga, bintang, lingkaran, kotak, dan lain-lain).

SmartArt (diagram, struktur, matriks, piramida, dan lain-lain).

Chart (grafik).

Link, berfungsi untuk menghubungkan dengan file atau dokumen lain.

Header & Footer, berfungsi untuk menempatkan teks, nomor halaman, judul tulisan, waktu dan tanggal, serta gambar/objek lainnya pada setiap halaman dokumen. Header terletak pada bagian atas dokumen, sedangkan footer berada di bagian bawah dokumen.

Text, berfungsi untuk menyisipkan text box, quick parts, word art, dan drop cap.

Symbols, berfungsi untuk menyisipkan simbol.
