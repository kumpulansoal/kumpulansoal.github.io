1. Apa fungsi dari combo box


Combo Box, fungsi dari combo box yaitu Control yang digunakan untuk memasukkan data dengan cara memilih nilai dari daftar pilihan yang tersedia.Daftar pilihan pada combo box tidak ditampilkan seluruhnya sampai kita mengklik anak panah drop-down pada combo box.Combo box juga memberikan kemudahan untuk memasukkan nilai yang tidak tersedia pada daftar pilihan dengan kata lain combo box merupakan kombinasi antara text box dengan list box.

2. Perbedaan Text Box dan Combo Box


Text box digunakan untuk menulis huruf kalau combo box memilih menu yang disediakan dalam box- Text Box biasanya digunakan untuk memberikan tempat untuk variabel-variabel yang bisa ditulis/ditambahkan karakter saat program dijalankan

- Combo Box digunakan sebagai tempat variabel-variabel karakter yang sudah ditulis/diprogram sebelumnya dan hanya bisa dipilih saat program dijalankan


3. Perbedaan dan persamaan Text Box dan Combo Box


perbedaanya
text box menampilkan semua data berupa text dalam boxnya sedangkan kombo box hanya menampilkan data berbentuk list

persamaannya
sama sama berisi text pada penggunaannya

7. Apa Keuntungan dan Kerugian dalalm Penggunaan List Box dan Combo Box !


Keuntungan dan kerugian dalam memakai  List Box dan Combo Box adalah untuk keuntungan maka kita dapat memberikan opsi untuk memberikan kontrol objek lain. Untuk kekurangan maka diperlukan sebuah algoritma yang dapat menangkap kontrol dari list dan combo box tersebut.

Pembahasan

Berbeda dengan objek kontrol lainnya, kotak kombo lebih sering digunakan sebagai variabel kontrol. Dengan menggunakan variabel kontrol, ini tidak terkait dengan tipe data tertentu (secara default, tipe data kotak kombo adalah CString). Selain itu,  semua fungsi yang terkait dengan kotak kombo, seperti fungsi GetCurSel() dan GetLBText(), dapat diakses menggunakan variabel kontrol. Kotak daftar memiliki fungsi yang hampir sama dengan kotak kombo yang menampilkan daftar. Perbedaannya adalah kotak kombo hanya menampilkan satu opsi sebelum Anda mengkliknya, sedangkan kotak daftar dapat menampilkan beberapa opsi. Kotak kombo adalah kontrol Microsoft Access yang bisa Anda gunakan dalam proses entri data yang memungkinkan Anda memilih data dari  daftar.
