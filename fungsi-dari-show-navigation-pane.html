2. Apa fungsi navigation pane pada microsoft power point 2016


Apa fungsi navigation pane pada microsoft power point 2016. Sebelum membahas mengenai fungsi navigation pane pada microsoft power point 2016, mari kita bahas terlebih dahulu apa itu Microsoft Power Point. Microsoft Power Point adalah suatu software yang digunakan untuk membuat sebuah presentasi, dengan microsoft power point kita bisa membuat prsentasi kita menjadi lebih menarik.

Pembahasan

Setelah mengetahui apa itu microsoft power point, maka selanjutnya adalah membahas mengenai fungsi navigation pane pada microsoft power point, navigation pane pada power sendiri memiliki fungsi yang sama seperti pada microsoft word. Fungsi dari navigation pane sendiri adalah untuk mempermudah kita dalam membuat atau melakukan pengeditan terhadap lembar kerja kita berdasarkan nomor halaman.

Beberapa fungsi lain pada microsoft power point selain dari fungsi navigation pane adalah sebagai berikut
