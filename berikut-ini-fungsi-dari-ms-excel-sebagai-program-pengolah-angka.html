FUNGSI MICROSOFT EXCEL Jawaban

Contoh fungsi Microsoft Excel sebagai program pengolah angka sederhana adalah:

Untuk membuat daftar nilai siswa di sekolah Untuk membuat daftar pemasukan dan pengeluaran Untuk membuat laporan keuangan Untuk membuat daftar gaji karyawan

Pembahasan

Microsoft Office Excel atau sering disebut Microsoft Excel atau Excel saja, adalah sebuah program aplikasi lembar kerja spreadsheet yang dibuat dan didistribusikan oleh Microsoft Corporation. Aplikasi ini dapat dijalankan pada Microsoft Windows maupun pada Mac OS. Aplikasi ini merupakan program spreadsheet paling banyak digunakan, baik di Ms. Windows maupun pada sistem operasi Mac OS.

Kelebihan dan Kekurangan Ms. Excel

Kelebihan

Memiliki daya tamping yang sangat besar. Satu sheet excel terdiri dari 1 juta baris dan 16 ribu kolom. Hal ini memungkinkan kita untuk memasukkan data dalam jumlah yang sangat banyak. Adanya fitur rumus (formula) yang meudahkan pengguna untuk melakukan hitungan aritmatika baik sederhana maupun kompleks. Software yang fleksibel, karena dapat dikonversikan ke berbagai software statistik. Begitupun sebaliknya. Pekerjaan menjadi lebih efektif. Salah satu fitur excel yang bernama pivot tables membuat data yang masuk secara otomatis akan ter-refresh sehingga tabel akan selalu up to date.

Kekurangan

Termasuk aplikasi berbayar yang tidak gratis. Harga sebuah program Ms. Excel cukup mahal untuk kalangan pelajar. Menggunakan RAM dan CPU yang cukup besar. Cukup kompleks bagi pemula. Tidak sembarang orang bisa menggunakan Ms. Excel secara langsung. Perlu adanya latihan untuk menggunakan software ini dengan baik.
