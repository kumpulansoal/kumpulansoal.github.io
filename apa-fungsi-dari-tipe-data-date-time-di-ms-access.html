8. jelaskan 5 tipe data dalam microsoft access


5 tipe data dalam microsoft access yaitu:

Text yaitu tipe data yang digunakan untuk field alfanumeric misalnya nama, alamat, kode pos, dan lainnya.Memo yaitu tipe data yang dapat menampung kurang lebih 64.000 karakter untuk tiap fieldnya.Number yaitu tipe data yang digunakan untuk menyimpan data numeric yang akan digunakan untuk proses perhitungan matematis misalnya pengurangan, penjumlahan, perkalian dan pembagian suatu bilangan.Data/time yaitu tipe data yang digunakan untuk data yang berjenis tanggal, waktu atau penggabungan dari tanggal dan waktu.Currency yaitu tipe data dengan jenis number, yang pada awal angka selalu disertakan simbol currency default sesuai dengan regional setting yang digunakan, misalnya RP. $, dan lainnya.

17. yang tidak termasuk data type yang disediakan ms access adalah


Yang tidak termasuk data type yang disediakan microsof access adalah mari kita bahas terlebih dahulu data apa saja yang ada pada microsoft access. Namun kita akan bahas terlebih dahulu apa itu microsoft access. Microsoft Access adalah sebuah aplikasi buatan Microsoft yang berfungsi untuk database dalam skala kecil dan menengah.

Tipe data yang ada pada microsoft access adalah sebagai berikut

1. Text, tipe data text merupakan salah satu tipe data yang umum digunakan. Tipe data ini biasanya digunakan untuk data nama, alamat dan lainnya. Maksimal karakter adalah 255 karakter.

2. Memo, tipe data yang kedua pada microsoft access adalah memo. Memo tidak berbeda jauh dengan text, perbedaan mendasar adalah jumlah karakter, jika text hanya 255 karakter, memo bisa menampung hingga 64.000 karakter.

3. Number, tipe data ketiga adalah number, pada tipe data ini biasanya berguna untuk melakukan perhitungan. Seperti penambahan, pengurangan, pembagian dan sebagainya.

4. Date Time, tipe data selanjutnya pada microsoft access adalah tipe data date time. Tipe data ini berfungsi sebagai isian data waktu, seperti jam, tanggal dan sebagainya.

5. Currency, tipe data selanjutnya ada;ah currency. Tipe data ini sering digunakan sebagai data mata uang.

Demikian pembahasan mengenai tipe data pada microsoft access, untuk tipe data yang tidak terdapat pada microsoft access adalah unicode dan binary. Tipe data tersebut terdapat di SQL Server.

25. Jelaskan Fungsi tipe data text,type data number,type data date/time,type data autonumber

Tipe data number : untuk menyimpan data numeric / angka yang akan digunakan untuk proses perhitungan matematis.

tipe data pada micrososft acces 2010
1) Text : data yang berupa huruf dan angka, dengan maksimum karakter 255
2) Memo : data yang berupa huruf dan angka seperti Text, tetapi memiliki maksimum karakter 63.999
3) Number : data yang berupa angka untuk kalkulasi matematika
4) Date/Time : data yang berisi penentuan waktu 
5) Currency : data berupa number yang disertai dengan sumbol currency default
6) AutoNumber : data yang tidak dapat kita isi secara manual melainkan ia terisi secara otomati, baik secara menjumlah ataupun random (acak). 
7) Yes/No : data dengan jenis hanya 2 pillihan yaitu Yes (-1 atau True) atau No (0 atau False)
8) OLE Object : data yang diambil dari system OLE
9) Hyperlink : data yang digunakan untuk menyimpan alamat internet atau file yang ditunjukkan melalui alamat URL. 
10. Attachment :data type yang digunakan untuk menyimpan file lampiran yang berformat bebas
