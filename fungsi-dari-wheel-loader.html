3. berikut file dalam dos yang berfungsi sebagai boot loader adalah


file dalam dos yang berfungsi sebagai boot loader adalah mbldr.exe (CMIIW) File tersebut berfungsi sebagai boot loader membawa OS untuk memasuki boot yang ada di dalam sistem operasi.

MBLDR merupakan kependekan atau singkatan dari Master Boot Loader. Mbldr dibuat dengan base OS Boot Select (OS-BS) by Thomas wolfRAM


Pembahasan

Sebagai intermezzo, Boot loader atau dalam bahasa indonesia pemuat boot. Bootloader merupakan program komputer yang memuat sistem operasi (OS) untuk komputer secara sederhananya digunakan untuk memanggil boot sistem operasi dan memilih akan menggunakan Ssistem operasi yang mana.

Bootloader dapat digunakan untuk mengkonfigurasi  dan sarana untuk memilih darimana memulai kernel. Hal ini memungkinkan untuk melakukan aktivitas boot yang memiliki 2 Sistem operasi seperti contoh untuk mengunakan Linux alternatif, atau Windows. Bootloader adalah komponen penting dari proses booting yang harus dipastikan dalam keadaan baik, memori bootloader tersebut tersimpan dalam memori non-volatile contohnya seperti memori flash.

Jawaban:

Untuk memberikan warna pada objek dilakukan dengan cepat yaitu pada color pallets klik kanan untuk garis luar objek dan klik kiri untuk isi objek (a).

Pembahasan

Memberi warna pada objek di CorelDraw dapat dilakukan dengan berbagai cara, yaitu:

1. Color pallete

Color Pallete atau pallet warna yang berbentuk sebuah barisan warna, Color pallete bukan tools, melainkan sebuah jendela tampilan yang berisi warna-warna. Color Pallete ini adalah alat yang paling mudah diakses untuk memberi warna pada objek karena tidak memunculkan jendela tampilan ataupun property bar.

Color Pallete ini hanya menampilkan beberapa warna saja. Color Pallete Default biasanya terletak di samping sebelah kanan layar. Untuk mewarnai Objek dengan Color Pallete anda hanya tinggal menyeleksi objek yang akan diwarnai kemudian klik warna yang di inginkan.

2. Fill Color dan Line Color

Fill Color seperti namanya adalah tool untuk mewarnai bagian Isi Objek, dan Line Color adalah untuk mewarnai Garis.

Fill Color dan Line Color terletak di sebelah kanan bawah layar. Di dalam Fill Color tersedia semua warna, sehingga kita bisa memilih warna dengan lebih leluasa.

Terdapat beberapa jenis pewarnaan yang bisa kita aplikasikan dengan menggunakan Fill Color yaitu:

Uniform Fill : Untuk mewarnai dengan satu warna (Polos)

Fountain Fill : Untuk mewarnai dengan Gradasi warna

Pattern Fill : Untuk mewarnai dengan Pattern

Texture Fill : Untuk mewarnai dengan tekstur

3. Smart Fill Tool

Tool Ini terletak di bagian tengah Toolbox, Smart Fill Tool berfungsi untuk memberi warna atau menghilangkan warna pada objek atau bagian dari objek.

4. Interactive Fill Tool

Interactive Fill Tool berada di paling bawah Toolbox, fungsinya hampir sama dengan Fountain Fill hanya saja Interactive Fill Tool lebih mudah dalam mengatur arah gradasi warnanya.

5. Mesh Fill Tool

Mesh Fill Tool merupakan alat untuk mewarnai Objek dengan beberapa warna yang berbeda, mesh fill tool akan membentuk grid-grid yang ada di dalam objek yang bisa kita atur jumlah baris dan kolomnya, dimana di dalam setiap grid bisa kita aplikasikan warna yang berbeda. Untuk mengatur arah warnanya kita tinggal mengubah bentuk grid dengan menggerakkan titik-titik grid.

16. apa yang anda ketahui tentang boot loader pada linux ?


Bootloader atau boot manager merupakan sebuah aplikasi yang terdapat hampir di seluruh sistem operasi termasuk linux. Boot manager ini digunakan untuk multiple boot. Dengan boot manager, kita bisa mengatur proses booting. Bila kita menginginkan OS yang kita gunakan dibaca oleh BIOS tanpa menggunakan Disk Boot maka kita memerlukan Boot Loader program yang terinstall pada MBR ( Master Boot Record ). Boot loader program dimuat di dalam BIOS komputer dan bertugas untuk membaca kernel yang ada di dalam suatu sistem operasi serta memberi kendali terhadap jalannya sistem pada kernel. Kernel akan dapat melakukan inisiasi pada sistem serta mengendalikannya. Sebagai contoh, saat kita ingin menggunakan 2 sistem operasi pada satu komputer, misal ingin menginstall ubuntu dimana kita telah menginstall windows vista di komputer. Apabila ubuntu telah terinstall di hardisk maka secara otomatis ubuntu akan mengeluarkan pilihan booting untuk multiple boot. Secara singkat, boot loader adalah program software pertama yang berjalan ketika komputer dimulai. Hal ini bertanggung jawab untuk loading dan transfer kontrol ke kernel sistem operasi perangkat lunak (seperti Hurd atau Linux). Kernel, pada gilirannya, menginisialisasi sisa dari sistem operasi (misalnya GNU).

38. definisi istilah berikut: boot loader, boot manager, POST dan MBR dalam sistem operasi


Boot loader program dimuat di dalam BIOS komputer dan bertugas untuk membaca kernel yang ada di dalam suatu sistem operasi serta memberi kendali terhadap jalannya sistem pada kernel.

Boot manager merupakan sebuah aplikasi yang terdapat hampir di seluruh sistem operasi termasuk linux. Boot manager ini digunakan untuk multipleboot. 

POST (Power on Self-Test) yaitu test yang dilakukan oleh PC untuk mengecek fungsi-fungsi komponen pendukung PC apakah bekerja dengan baik.

Partisi Master Boot Record (MBR) (Inggris: MBR-Style Partition) adalah sebuah partisi yang menggunakan struktur Master Boot Record. Di dalam Master Boot Record, terdapat sebuah tabel partisi yang menjelaskan di mana partisi-partisi diletakkan di dalam hard disk.

MBR juga hanya mendukung hingga empat partisi utama yang berarti bahwa pengguna hanya bisa memiliki maksimal empat sistem operasi yang berbeda pada satu hard disk.


Kelas : SMP

Mapel : TIK

Kode : 7.11.7

Kategori : Perangkat Keras Komputer

Kata kunci : MBR
