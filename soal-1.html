14. Soal LatihanHanya 1 soal ​


[tex] \rm \frac{ \frac{1}{a} + \frac{2}{b} }{ab} = \frac{ \frac{1b + 2a}{ab} }{ab} [/tex]

[tex] \rm= \frac{ \frac{2a + b}{ab} }{ab} [/tex]

[tex] \rm = \frac{2a + b}{ab} \div \frac{ab}{1}[/tex]

[tex] \rm = \frac{2a + b}{ab} \times \frac{1}{ab} [/tex]

[tex] \rm = \red{ \frac{2a + b}{ {a}^{2} {b}^{2} } ⟹(b)}[/tex]

.

Jawaban:

[tex] \: \: \: \frac{ \frac{1 }{2} + \frac{2}{b} }{a \: b} [/tex]

[tex] = \frac{ \frac{1b + 2a}{ab} }{ab} [/tex]

[tex] = \frac{ \frac{2a + b}{ab} }{ab} [/tex]

[tex] = \frac{2a + b}{ab} \div \frac{ab}{1} [/tex]

[tex] = \frac{2a + b}{ab} \times \frac{1}{ab} [/tex]

[tex]= \frac{2a + b}{ {a}^{2} {b}^{2} } [/tex]

- Bangun ruang bola tidak memiliki titik sudut.

- Bola memiliki bagian jari-jari yang menghubungkan titik pusat bola dengan titik permukaan bola.
