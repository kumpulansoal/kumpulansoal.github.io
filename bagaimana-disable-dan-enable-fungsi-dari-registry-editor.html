1. sebutkan fungsi registry editor


REGEDIT (Registry Editor).

Regedit adalah database hirarki yang digunakan untuk mengatur informasi yang dibutuhkan oleh Sistem Windows..regedit menampilkan Registry ke dalamformat yang hampir mirip dengan Windows Explorer yang sering anda gunakan. Di dalam regedit tersebut anda dapat mengedit informasi untuk sistem semau anda..tetapi tidak bisa sembarangan. Dan harus mengetahui apa fungsi dari setiap informasi yang ada pada regedit.

Fungsi regedit adalah untuk melihat, membuat atau memodifikasi registry di dalam Windows. Sedangkan Gpedit.msc digunakan untuk melihat, membuat, dan memodifikasi Group Policy, yang digunakan untuk mengelola beberapa pengaturan di dalam setiap pengguna dan lingkungan Windows serta menyimpannya di dalam registry.

2. apa arti enable dan disable​


Jawaban:

Secara bahasa, enable artinya memungkinkan. Disable artinya tidak memungkinkan.

Penjelasan:

Dalam dunia TIK, enable sering digunakan sebagai istilah untuk perangkat yang statusnya (kondisinya) memungkinkan dipakai.

Sedangkan disable ialah istilah sebaliknya. Ada beberapa contoh dimana suatu perangkat, statusnya dapat diubah dari disable menjadi enable. Perhatikan gambar berikut ini (ditunjukkan tanda panah merah).

31. disable auto load image


cara mengembalikannya :

1.Buka browser Firefox Anda. Pada addressbar ketikkan ‘about:config’ kemudian tekan enter. Maka, akan muncul pemberitahuan. Klik pada tombol I’ll be careful, I Promise!
2. Untuk mengaktifkan atau menonaktifkan pilihan Load images automatically, ketikkan ‘permissions.default.image’ di kotak pencarian dan Anda akan mendapat Integer dengan nama tersebut di daftar preferensi
3. Di sana Anda akan melihat value nya 1 yang berarti semua gambar diatur untuk muncul secara otomatis. Jika Anda ingin men-disable image ubah nilainya menjadi 2. Jika Anda ingin mengijinkan gambar loading dari website yang sama tetapi ingin men-disable gambar dari pihak ketiga, ubah value nya menjadi 3.
4. Untuk mengaktifkan dan men-disable pilihan JavaScript, ketikkan ‘javascript.enabled’ di kotak pencarian dan Anda akan melihat boolean dengan nama yang dimaksud yaitu ‘javascript.enabled’ di daftar preferensi.
5. Secara default, value tersebut diatur pada ‘true’ yang berarti JavaScript selalu aktif. Untuk men-disable JavaScript, ubah value nya menjadi ‘false’ dengan cara klik 2x pada nama preferensi tersebut.
