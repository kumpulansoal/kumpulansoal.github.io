17. jlentrehna bedane aksara angka karo aksara jawa liyane?


⏺️ Aksara angka yaiku aksara Jawa kang digunakake kanggo nulis angka Jawa.

⏺️ Aksara murda yaiku aksara sesirah kang sinebut huruf kapital.

⏺️ Aksara rekan yaiku aksara Jawa kang digunakake kanggo mbiyantu pemenuhan saka basa liya.

⏺️ Aksara swara yaiku aksara Jawa kang digunakake kanggo nulis huruf vokal.

Pembahasan

⏺️ Miturut jinising aksara Jawa ana 5 :

1️⃣ AKSARA CARAKAN

yaiku aksara kang ditulis nganggo tulisan Jawa sing durung diwenehi pasangan lan sandhangan apa apa.

>> Aksara carakan cacahe ana 20 :

ha, na, ca, ra, kaha, na, ca, ra, kada, ta, sa, wa, laha, na, ca, ra, kada, ta, sa, wa, lapa, dha, ja, ya, nyaha, na, ca, ra, kada, ta, sa, wa, lapa, dha, ja, ya, nyama, ga, ba, tha, nga

2️⃣ AKSARA MURDA

yaiku aksara kang sinebut huruf kapital. Aksara murda digawe kanggo pakurmatan utawa menghormati lan lumrahe kanggo nulis jenenge wong utawa pangkat lan Drajat sarta papan Panggonan.

>> Aksara murda cacahe ana 8 :

na = ꦟka = ꦑta = ꦡsa = ꦯpa = ꦦnya = ꦘga = ꦓba = ꦨ

3️⃣ AKSARA REKAN

yaiku aksara Jawa kang digunakake kanggo mbiyantu pemenuhan kebutuhan ing panulisan lan ukara ukara sing asale saka basa liya.

>> Aksara rekan cacahe ana 5 :

ka = Kha = ꦏ꦳da = dza = ꦢ꦳ga = Gha = ꦒ꦳pa = Fa = Va = ꦥ꦳ja = Za = ꦗ꦳

4️⃣. AKSARA SWARA

yaiku aksara Jawa kang digunakake kanggo nulis huruf vokal.

>> Aksara swara cacahe ana 5 :

A = ꦄI = ꦇU = ꦈE = ꦌO = ꦎ

5️⃣ AKSARA ANGKA

yaiku aksara Jawa kang digunakake kanggo nulis angka Jawa.

>> Aksara angka cacahe ana 10 :

0 = ꧐1 = ꧑2 = ꧒3 = ꧓4 = ꧔5 = ꧕6 = ꧖7 = ꧗8 = ꧘9 = ꧙
