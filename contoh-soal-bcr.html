5. Perhatikan gambar di atas besar sudut BCR=43 derajat Dan sudut SQR =141 derajat.tentukan besar sudut BSQ


Besar sudut BSQ adalah 82°. Hasil ini diperoleh dengan menentukan sudut-sudut yang sehadap dan bersebrangan, kemudian menentukan sudut-sudut yang berpelurus, jumlah sudut dalam segitiga, dan menghitungnya. Lihat gambar di attachment untuk lebih jelasnya dan simak penjelasan berikut!

Pembahasan

Diketahui

Besar sudut BCR = 43°

Besar sudut SQR = 141°

Perhatikan gambar!

Ditanya

Besar sudut BSQ

Penyelesaian

Perhatikan pada gambar bahwa CR dan BS sejajar, PR dan BC sejajar. Maka,

Sudut ABS dan sudut BCR sehadap

∠ ABS = ∠ BCR = 43°

Sudut QPS dan sudut ABS bersebrangan dalam

∠ QPS = ∠ ABS = 43°

Menghitung sudut PQS

∠ PQS = 180° - ∠ RQS

∠ PQS = 180° - 141°

∠ PQS = 39°

Menghitung sudut PSQ

∠ PSQ = 180° - (43° + 39°)

∠ PSQ = 180° - 82°

∠ PSQ = 98°

Menghitung sudut BSQ

∠ BSQ = 180° - 98°

∠ BSQ = 82°

Kesimpulan

Jadi, besar sudut BSQ adalah 82°.

Jawab:

Penjelasan dengan langkah-langkah:

diketahui :

U4 = 6.900.000

U6 = 9.500.000

ditanya :

a. perkembangan penerimaan tiap tahunnya = b

b. penerimaan tahun ke-1 = U1 = a

c. U8

U4 = 6.900.000

U5 = U4 + b

U6 = U5 + b

     = U4 + b + b

     = U4 + 2 b

oleh karena U4 dan U6 diketahui maka

2 b = U6 - U4

   b = 1/2 ( U6 - U4)

      = 1/2 (9.500.000 - 6.900.000)

      = 1/2 . 2.600.000

      = 1.300.000

dengan demikian kita peroleh perkembangan penerimaan setiap tahunnya adalah sebesar Rp 1.300.000

U1 = a = U2 - b = U3 - b - b = U4 - b - b - b

          = 6.900.000 - 3 (1.300.000)

          = 6.900.000 - 3.900.000

         = 3.000.000

maka kita dapatkan penerimaan pada tahun ke-1 sebesar Rp 3.000.000

Besar penerimaan pada tahun ke-8 atau U8

= a + (8 -1 ) *  b = a + 7b

= 3.000.000 + 7 (1.300.000)

= 3.000.000 + 9.100.000

= 12.100.000

diperoleh penerimaan pada tahun ke-8 sebesar Rp 12.100.000

19. Diketahui ACR, BCR, CCR dengan A={x|x² +5x=0}, B = {x\x²9x=0}, dan C={x|x>0}. Tentukan a. AUBb. BnC C. A-C d. AU(B-C)​


Diketahui A [tex]\subset[/tex] R, B [tex]\subset[/tex] R, C [tex]\subset[/tex] R dengan A={x | x² +5x = 0}, B = {x | [tex]x^{3}[/tex] - 9x = 0}, dan C={x | x > 0}.

a. A U B = {-5, -3, 0, 3}

b. B ∩ C = {3}

C. A - C = {-5, 0}

d. A U (B - C)​ = {-5, -3, 0}

Penjelasan dengan langkah-langkah:

Diketahui:
A = {x | x² + 5x = 0}
c
C = {x | x > 0}

Ditanya:
a. A U B

b. B ∩ C

C. A - C

d. A U (B - C)​

Jawab:
A = {x | x² + 5x = 0}, akan dicari nilai x pada persamaan himpunan A
x² + 5x = 0
x(x + 5) = 0
x = 0 atau x = -5
artinya anggota himpunan A adalah A = {-5, 0}
B = {x | [tex]x^{3}[/tex] - 9x = 0}, akan dicari nilai x pada persamaan himpunan B
[tex]x^{3}[/tex] - 9x = 0
x(x² - 9) = 0
x = 0 atau x² = 9
x = 0 atau x = ±3
artinya anggota himpunan B adalah B = {-3, 0, 3}
C = {x | x > 0}
artinya anggota himpunan C adalah seluruh bilangan real lebih dari 0.

a. A U B = {-5, 0} U {-3, 0, 3}
   A U B = {-5, -3, 0, 3}
b. B ∩ C = {-3, 0, 3} ∩ {x | x > 0}
   B ∩ C = {3}
c. A - C = {-5, 0} - {x | x > 0}
   A - C = {-5, 0}
d. A U (B - C)​ = A U ({-3, 0, 3} - {x | x > 0})
   A U (B - C) = {-5, 0} U {-3, 0}
   A U (B - C) = {-5, -3, 0}
