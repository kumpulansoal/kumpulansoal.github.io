E gabungan F -> anggota F saja + anggota E saja

Jika yang termasuk anggota E ada 31 orang, maka yang murni anggota E saja adalah 31-18 = 13

Jika yang termasuk anggota F ada 22 orang, maka yang murni anggota F saja adalah 22-18 = 4

Jadi E gabungan F adalah 13+4 = 17

P (E) = n (E) / n (S)
  3/5  =  (x² - x + + 1) / [ (x² - x + 1)² - (x² - x + 1) - 1 ]

misal : x² - x + 1 = A
soal dapat ditulis :
3/5 = A / (A² - A - 1)
5A = 3 (A² - A - 1)
5A = 3A² - 3A - 3
  0 = 3A² - 3A - 5A - 3
  0 = 3A² - 8A - 3
  0 = (3A + 1) (A - 3)
  3A + 1 = 0      atau   A - 3 = 0
       3A  = - 1    atau         A = 3
         A  = - 1/3

jadi , x² - x + 1 = - 1/3    atau  x² - x + 1 = 3
