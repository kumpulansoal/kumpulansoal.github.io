8. apa yang dimaksud dengan network address dan broadcast network?​


Broadcast Network Merupakan alamat terakhir pada sbuah jaringan / Network yang tidak bisa digunnakan. sesuai namanya Broadcast address Ip ini digunnakan untuk melakukan Broadcasting kesluruh Host didalam sebuah jaringan.

Network Adress Merupakan alamat pertama pada sebuah jaringan atau network yang tidak dapat digunakan kecuali dalam keadaan tertentu. yang dimaksud keadaan tertenu adalah dimana host dipaksa untuk menggunakann Full Routing sebelum menuju ke Host selanjutnya, jadi Host dipaksa untuk berkomunikasi terlebih dahulu pada router sebelum berkomunikasi pada host lainnya.

30. Diketahui network 192.168.100.0/24 akan di-subnet dengan /27, tentukan :-Jumlah Subnet (Network)-Jumlah Blok Subnet-Jumlah Host per Subnet-Network Address, Broadcast Address dan Host Address (Pertama & Terakhir)-Subnet Mask baru


Jawaban:

1.Jumlah Subnet = 2x ,dimana x adalah banyaknya binary 1 pada octet terkhir subnetmask (2 oktet terakhir untuk kelas B, dan 3 oktet terakhir untuk kelas A). Jadi jumlah subnet adalah 22 = 4 subnet.

Jumlah Host per Subnet = 26 – 2 ,dimana y adalah kebalikan dari x yaitu banyaknya binary 0 pada octet terakhir subnet. Jadi jumlah host per subnet adalah 26 – 2 = 62 host.

Blok Subnet = 256 – 192 (nilai octet terakhir subnetmask) = 64. Subnet berikutnya adalah 64+64=128, 128+64=192, Jadi subnet lengkapnya adalah 0,64,128,192.

34. jika diberikan IP address 172.16.10.33/26 Tentukan network address tersebut


Pertama ubah /26 ke dalam Netmask yaitu 255.255.255.192

Langkah ke dua ubah IP Address dan Netmask ke dalam Biner.

172.16.10.33 = 10101100.00010000.00001010.00100001

255.255.255.192 = 11111111.11111111.11111111.11000000

Lalu untuk menentukan Network Address caranya adalah menggunakan fungsi AND.

Begini caranya, digunakan di antara IP Address dan Netmask :

IP Address = 10101100.00010000.00001010.00100001

Netmask = 11111111.11111111.11111111.11000000

Menjadi = 10101100.00010000.00001010.00000000

Sehingga Kita dapat Network Addressnya : 172.16.10.0 ( Konversi Dari Biner ke Desimal )
