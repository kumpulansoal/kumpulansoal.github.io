10. pada posisi 90 meter dari stasiun sebuah lokomotif bergerak dengan kecepatan 72 kam /jam lokomotif itu haris berhenti di stasiun hitunglah perlambatan lokomotif itu


GLBB (Gerak lurus berubah beraturan)

Analisis Soal

pada posisi 90 m dari stasiun ( s = 90m ) sebuah lokomotif bergerak dengan kecepatan 72 km/jam (V₀ = 72 km/jam). Lokomotif itu harus berhenti di stasiun , Hitunglah perlambatan (a) lokomotif itu.

Yang diketahui dari soal

V₀ = 72 km/jam = 20m/s

S = 90 m

Ditanya

a = ...??

[ Pembahasan ]

" Lokomotif harus berhenti di stasiun"

artinya, Vt = 0

Vt² = V₀² + 2as

0  = (20)² + 2(90)a

0 = 400+ 180a

180a = -400

      a =  -2,22 m/s²

(+a = percepatan, -a = perlambatan )

Perlambatan sebesar 2,22 m/s²

12. 900 m dari stasiun sebuah lokomotif bergerak dengan kelajuan 15m/detik lokomotif itu harus berhenti di stasiun . hitung perlambatan lokomotif


Gerak Lurus Berubah Beraturan.

v² = v₀² + 2aΔx
a = (v² - v₀²) / (2Δx)
   = [(0 m/s)² - (15 m/s)²] / [2(900 m)]
   = -0,125 m/s²

16. Orang yang mendorong lokomotif dan lokomotif tidak bergerak, dalam fisika dikatakan usaha = 0. Mengapa demikian?​


Jawaban:

Benda tidak bergerak ( tidak pindah kedudukan ), artinya nilai s = 0. Sehingga berapapun nilai F akan menghasilkan nilai W (usaha) = nol.

Penjelasan:

W = F . s

dimana; W = usaha ( joule )

              F  = gaya ( newton )

              s  = perpindahan ( meter )

Jika, benda yang dikenai gaya tidak mengalami perpindahan artinya nilai s=0, maka, W (usaha) juga bernilai nol.
