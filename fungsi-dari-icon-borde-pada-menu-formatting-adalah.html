29. Sebutkan icon icon di menu home dan jelaskan fungsinya


Jawaban:

· 1. Clipboard

a. Cut (CTRL +X)

Cut berfungsi untuk menghapus/memindahkan teks atau objek yang dipilih ke clipboard

b. Copy (CTRL +C)

Mengcopy suatu objek yang tersimpan pada clipboard ke dalam lembar kerja kita

c. Paste

· Paste (CTRL +V)

Paste berfungsi untuk menempatkan objek/teks yang ada di clipboard pada posisi titik sisip berada

· Paste Special (ALT + CTRL +V)

Paste Special berfungsi untuk menempatkan objek/teks yang ada di clipboard pada posisi titik sisip berada dengan format yang berbeda

· Paste As Hyperlink

Paste As Hyperlink berfungsi untuk menempatkan teks sebagai hyperlink

d. Format Painter

Menyalin format pada Clipboard untuk diletakkan pada posisi yang diinginkan.

2. Font

a. Fonts (CTRL+SHIFT+F)

Font berfungsi untuk memilih beberapa jenis atau bentuk font/huruf untuk digunakn pada teks.

b. Font Size (CTRL + SHIFT +T)

Font Size berfungsi untuk mengubah ukuran font/huruf pada teks.

c. Grow Font (CTRL +>)

Grow Font berfungsi untuk memperbesar font secara instan

d. Shrink Font (CTRL +<)

Shrink Font berfungsi untuk memperkecil font secara instan

e. Clear Formatting

Clear Formatting berfungsi untuk menghapus pengformatan teks terpilih

f. Bold (CTRL +B)

Bold berfungsi untuk mempertebal teks yang terpilih

g. Italic (CTRL +I)

Italic berfungsi untuk memirinkan teks yang terpilih

h. Underline (CTRL +U)

Underline berfungsi untuk memberikan garis bawah pada teks yang terpilih

i. Strikethrough

Strikethrough berfungsi untuk memberikan garis tengah (garis coret) pada teks yang terpilih

j. Subscript (CTRL +=)

Subscript berfungsi untuk mengetikan karakter pemangkatan (huruf kecil di bawah huruf sebelumnya)

k. Superscript (CTRL +SHIFT ++)

Superscript berfungsi untuk mengetikan karakter pemangkatan (huruf kecil di atas huruf sebelumnya

l. Change Case (SHIFT F3)

Change Case bersungsi untuk mengubah status huruf kapital/huruf kecil/besar kecil pada teks/kata yang terpilih.

m. Text Highlight Color

Text Highlight Color berfungsi untuk memberikan latar belakang warna pada text

n. Font Color

Font Color berfungsi untuk memberi warna teks yang terpilih

3. Paragraph

a. Bullets

Bullets berfungsi untuk memberikan tanda bullet di tiap paragraf terpilih

b. Numbering

Numbering berfungsi untuk memberikan/menghilangkan format penomoran pada paragraph yang terpilih

c. Multilevel List

Multilevel List berfungsi untuk membuat posisi penomoran berupa angka atau nomor urut

d. Decrease Indent

Decrease Indent berfungsi untuk menghapus atau menurunkan posisi paragraph ke kiri

e. Increase Indent

Increase Indent berfungsi untuk menambahkan atau memberi posisi paragraph ke kanan

f. Sort

Sort berfungsi untuk mengurutkan data pada sebuah tabel

g. Show/ Hide¶ (CTRL +*)

Show/ Hide ¶ berfungsi untuk menampilkan/ menyembunyikan koreksi paragraph

h. Align Text Left (CTRL+L)

Align Text Left berfungsi untuk mengatur perataan teks ke kiri.

i. Center (CTRL+E)

Center berfungsi untuk mengatur teks untuk rata tengah

j. Align Text Right (CTRL+R)

Align Text Right berfungsi untuk mengatur teks rata kanan

k. Justify (CTRL+J)

Justify berfungsi untuk mengatur perataan teks ke kanan

l. Line Spacing

Line Spacing berfungsi untuk mengatur jarak antara baris teks atau paragraph

m. Shading

Shading berfungsi untuk memberikan latar belakang warna pada teks yang terpilih

n. Buttom Border

Buttom Border berfungsi untuk meberikan garis tepi pada teks yang terpilih

·

4. Styles

a. Heading Style

Heading Style berfungsi untuk memberi style pada paragraph. Pada Heading Style secara otmoatis terdapat beberapa format style yang siap digunakan seperti : ¶ Normal, ¶ No Spacing, Heading 1, Title, Subtitle, Subtle Emphasis, Emphasis, dan lain-lain Sebagainya

b. Change Styles

Change Styles berfungsi untuk mengubah style paragraph yang sedang aktif

5. Editing

a. Find (CTRL +F)

Find berfungsi untuk mencari kata yang ada pada halaman tesebut

b. Replace (CTRL +H)

Replace berfungsi untuk menggati kata yang ada pada halaman dengan kata yang diinginkan

c. Select

Select berfungsi untuk menyeleksi teks, paragraph atau gambar pada halaman.

38. Carilah fungsi menu dan icon yang ada di microsoft word sebanyak 20 icon


Jawaban:

1. New: New adalah pilihan untuk membuat dokumen baru.

2. Open: Open adalah pilihan untuk membuka kembali dokumen yang telah disimpan.

3. Save: Save adalah pilihan untuk menyimpan dokumen yang telah disimpan setelah diubah.

4. Save As: Save As adalah pilihan untuk menyimpan dokumen baruatau dokumen lama dengan nama yang berbeda.

5. Print: Print adalah pilihan untuk mencetak dokumen.

6. Prepare: Prepare adalah pilihan untuk persiapan publikasi dokumen.

7. Send: Send adalah pilihan mengirimkan dokumen melalui e-mail atau internet faximile.

8. Publish: Publish adalah pilihan untuk mengunggah dokumen ke internet melaui blog atau document managemen server.

9. Close Close: adalah pilihan untuk menutup

10. Paste : Untuk menampilkan hasil perintah cut dan copy

11. Cut: Untuk memindahkan teks

12. Copy: Untuk menyalin teks

13. Format Painter: Untuk menyamakan format sebuah teks sama dengan format teks sebelumnya.

14. Page Number : menambahkan halaman pada work sheet

15. Clip art: menyisipkan gambar yang telah di sediakan Ms Word

16. Insert Table : Menambahkan Tabel pada work sheet

17. Font: Untuk mengganti jenis huruf/gaya huruf

18. Bold : mengubah teks menjadi Tebal

19. Italic : mengubah teks menjadi cetak miring

20. Underline : menambahkan garis bawah pada huruf.
