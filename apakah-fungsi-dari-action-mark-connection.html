5. Action apakah yang berfungsi sebagai link tombol umtuk close atau keluar?


Jawaban:

Action yang biasanya berfungsi sebagai link tombol untuk close atau keluar adalah "on click" atau "onclick". Pada HTML, hal ini dapat diimplementasikan dengan menambahkan atribut "onclick" pada elemen tombol (button) atau tautan (link) dengan value yang berisi kode JavaScript yang akan dijalankan ketika tombol atau tautan tersebut ditekan. Contoh:

Tombol:

<button onclick="closeWindow()">Close</button>

Tautan:

<a href="#" onclick="closeWindow()">Close</a>

Di sini, fungsi JavaScript "closeWindow()" akan dijalankan ketika tombol atau tautan tersebut ditekan. Fungsi ini dapat berisi perintah untuk menutup jendela atau halaman web

16. perangkat keras yang dibutuhkan untuk membuat sambungan dial up connection jelaskan fungsi dari masing masing


Saluran telepon dan modem. Saluran telepon dibutuhkan untuk menghubungkan komputer kita dengan ISP yang digunakan. Selanjutnya, ISP tersebut akan meneruskan koneksi ke internet. Modem dibutuhkan untuk mengubah sinyal digital menjadi sinyal analog (sinyal listrik) agar dapat dikirimkan melalui saluran telepon, dan juga sebaliknya untuk mengubah sinyal listrik menjadi sinyal digital agar dapat dimengerti komputer.

23. Connection oriented protocolnya apa ? dan connection less protocolnya apa ?


Connection-Oriented Protocol (COP) adalah protokol jaringan yang digunakan untuk membuat sesi komunikasi data di mana perangkat endpoint menggunakan protokol awal untuk membangun koneksi ujung ke ujung dan kemudian aliran data berikutnya disampaikan dalam mode transfer sekuensial.

Protokol tanpa koneksi (connectionless protocol) adalah bentuk transmisi data di mana sinyal IT keluar secara otomatis tanpa menentukan apakah penerima siap, atau bahkan apakah ada penerima.

31. Tuliskan fungsi menu action di powerpoint !


Jawaban:

Tombol action atau action button pada aplikasi PowerPoint adalah sebuah tombol yang difungsikan untuk melakukan beberapa aksi tertentu. Diantaranya adalah Tautan dan Menjalankan program.

Penjelasan:

Dalam sebuh presentasi adakalanya kita membutuhkan sebuah interaksi melalui elemen tertentu yang difungsikan sebagai tombol. Nah, disinilah tombol action akan berfungsi.

Tombol action ini sendiri bisa diakses melalui menu bar Insert >> Shapes >> Action Button (lihat lampiran gambar). Pada umumnya aksi yang dilakukan tombol action ini adalah aktivitas yang berkaitan dengan presentasi itu sendiri dan hal eksternal.
