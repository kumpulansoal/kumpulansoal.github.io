12. Cara setting jaringan wlan(wifi/access point)


Jawaban:

Pasang Perangkatnya

Hal pertama yang harus Anda lakukan adalah mencolokkan adaptor dan kabel UTP ke port LAN AP dan ke LAN di komputer atau laptop. Jika sudah berhasil, maka akan muncul lampu indicator hijau yang berkedap-kedip. Pada umumny terdapat 4 indikator, seperti Power, System, LAN, WAN. iIndikator-indikator tersebut tergantung dari jenis dan tipe AP. Pada tutorial kali ini, AP yang digunakan adalah digunakan adalah AP TP-LINK tipe WA701N.

Setting IP di komputer atau Laptop

Langkah selanjutnya adalah melakukan setting IP di komputer atau laptop dengan cara masuk ke Control Panel. Anda perlu memilih Network and Sharing Center, pilih Change Adapter Settings,pilih perangkat jaringan yang sedang aktif, setelah itu akan muncul jendela baru. Kemudian klik Properties, klik dua kali pada Internet Protocol Version 4 (TCP/IPv4).

Jadikan IP komputer Menjadikannya Statistik

Untuk menjadikan IP komputer menjadi statistik, Anda perlu memilih opsi Use the Following IP address, isikan IP address dengan network yang sama, misalnya “192.168.1.10” serta subnet mask-nya, kosongkan yang lain, kemudian pilih OK dan simpan.

Kemudian Anda harus memasukkan username dan password seperti berikut:

Userame: admin

Password: admin

Selanjutnya Anda perlu klik Next, pilih Access Point, Next, isi SSID dan password sesuai dengan keinginan Anda, klik Next.

Pilih Access Point,

Isi SSID dan Password sesuai dengan yang And inginkan, setelah itu klik Next

Langkah selanjutnya adalah pilih Enable pada DHCP Server. Anda juga bisa mengatur IP jika dirasa perlu. Klik YES dan Next.

13. Berikut ini yang bukan merupakan fungsi access layer pada jaringan WAN￼


Salah satu yang bukan merupakan fungsi access layer pada jaringan WAN adalah menyajikan interface antara aplikasi dengan jaringan. Fungsi ini sebenarnya merupakan fungsi dari layer application pada model OSI, yaitu layer teratas yang berhubungan langsung dengan aplikasi pengguna. Layer application bertanggung jawab untuk menyediakan layanan-layanan jaringan, seperti email, web browsing, file transfer, dan lain-lain⁵.

Fungsi access layer pada jaringan WAN adalah sebagai berikut³:

- Menyuplai trafik ke jaringan dan melakukan network entry control.

- Melakukan daftar akses yang didesain untuk mencegah pengguna tidak sah untuk dapat masuk.

- Memberi akses situs jarak jauh kepada jaringan melalui teknologi wide-area, seperti frame relay, ISDN, atau leased lines.

- Mengendalikan akses pengguna atau workgroup ke sumber daya internetwork.

- Mengimplementasikan teknik-teknik pengamanan jaringan, seperti firewall, enkripsi, dan autentikasi.

15. Bagaimana cara access point menciptakan jaringan WLAN


Penggunaan acces point pada jaringan WLAN adalah dengan melakukan pengaturan terlebih dahulu pada sebuah acces point. Untuk melakukan pengaturan, kita bisa menggunakan langkah sebagai berikut

Masuk ke control panelBerikan IP pada perangkat laptop kitaAkses IP default pada access point kita, biasanya tertulis pada bagian bawah acces pointLakukan pengaturan lebih lanjut pada acces point

Setelah access point memiliki alamat IP, kita bisa memberikan alamat IP pada perangkat-perangkat laptop atau komputer kita. Apabila IP bersifat static, maka kita harus memberikan IP satu per satu. Namun kita bisa melakukan pengaturan dengan merubahnya menjadi IP dynamic, perangkat akan mendapatkan IP secara otomatis dari perangkat access point.

Pembahasan

Setelah access point sudah terhubung dengan perangkat kita, kita bisa melakukan test dengan cara melakukan ping. Beberapa kemudahan setelah berhasil membuat jaringan WLAN adalah

17. sumber koneksi jaringan wifi access point sama wifi router


Sumber koneksi jaringan wifi access point dengan wifi router adalah untuk Wifi Acces Point koneksi jaringan tersebut berasal dari router yang terhubung dengan kabel. Nah untuk sumber dari router berasal dari kabel (cable) yang berasal dari tembaga atau fiber.  

Pembahasan

Wireless Access Point adalah perangkat keras atau hardware yang berfungsi untuk menghubungkan peralatan wireless lain (laptop, ponsel) untuk terhubung ke jaringan kabel menggunakan Wi-fi. dan dapat digunakan untuk saling bertukar data antar perangkat wireless dan perangkat kabel yang terhubung pada jaringan.

Sedangkan router adalah suatu perangkat keras atau hardware pada jaringan komputer yang berfungsi untuk menghubungkan beberapa jaringan, baik itu jaringan yang sama maupun jaringan yang berbeda wireless atau kabel.

33. fungsi antena grid/omni dalam jaringan berbasis luas adalah.....


Antena adalah salah satu bagian dari sistem komunikasi berbasis radio yang pada umumnya berfungsi sebagai alat pengirim maupun penerima sinyal dalam bentuk sinyal radio. Pada stasiun penerima, sinyal inilah yang kemudian akan diproses dengan perangkat penerima untuk mengetahui secara detil isi pesan yang disampaikan, sementara pada perangkat pengirim, pesan akan diproses terlebih dahulu sebelum dikirimkan melalui perangkat antena.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menjabarkan fungsi antena grid atau omni dalam jaringan berbasis luas. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

DALAM SEBUAH JARINGAN BERBASIS LUAS, FUNGSI ANTENA GRID ATAU OMNI TERLETAK PADA PENERIMAAN SINYAL DENGAN FREKUENSI 2.4 MHz. PADA TINGKAT FREKUENSI INI, ANTENA DAPAT DIGUNAKAN UNTUK MENGIRIM SINYAL UNTUK BERBAGAI KEPERLUAN, SEPERTI INTRANET MAUPUN INTERNET.
