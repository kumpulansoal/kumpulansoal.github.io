3. jelaskan fungsi dari software remote desktop connections


Remote Desktop Connection adalah salah satu dari sekian banyak software yang dapat mengendalikan komputer lain melalui jaringan komputer (LAN). Fasilitas ini satu paket dengan Win XP ketika pertama sekali menginstall Win XP di komputer. Sayangnya masih banyak yang belum mengerti bagaimana cara menggunakannya, padahal manfaatnya sangat banyak sekali, salah satunya adalah kita dapat memperbaiki komputer lain tanpa harus bersentuhan dengan komputer tersebut melalui jaringan (LAN). Apa yang anda lihat di monitor anda adalah tampak persis seperti yang ditampilkan dikomputer yang sedang kita remote tersebut, namun jika untuk iseng atau untuk mengerjai komputer teman rasanya fasilitas ini bukanlah pilihan yang tepat, karena pada saat sesi remote dalam keadaan 'on', maka komputer yang sedang diremote akan masuk ke tampilan welcome screen.

Untuk penggunaannya, disarankan komputer 1 dan komputer 2 sama-sama Win XP. Terlebih dahulu kita melakukan pengaturan pada komputer yang akan kita remote (jika komputer teman anda yang hendak di remote, minta izinlah terlebih dahulu).

Catatan:- komputer 1 adalah komputer anda- komputer 2 adalah komputer target (yang akan kita remote) Kita dapan memperbaiki komputer lain tanpa menyentuhnya melalui jaringan LAN

16. apa fungsi dari google chrome?


Fungsi dari Google Chrome adalah utamanya untuk browsing. Google Chrome sendiri juga memiliki fungsi tambahan lain yang memungkinkan penggunanya mensinkronisasi akun Google dengan browser tersebut, pengguna dapat memasang ekstensi yang menunjang kinerja, membuka beberapa bentuk file didalam browser, dan melakukan inspect element pada suatu webpage.

Pembahasan

Google Chrome adalah salah satu produk hasil pengembangan perusahaan Google yang tersedia sebagai web browser maupun di android.  Pengguna Google Chrome sendiri kian meningkat tiap tahunnya dikarenakan fitur yang semakin lengkap dan interface yang lebih user-friendly.

Fungsi utama browser ini adalah untuk menjelajah berbagai web untuk kebutuhan berkirim email, layanan sosial media, search engine dan sebagainya.

Fungsi tambahan browser ini antara lain :

→ Memungkinan pengguna untuk mensinkronisasi akun Google dengan Chrome.

Biasanya ketika pengguna pertama kali menggunakan Chrome akan ditawarkan untuk menghubungkan akun Google aktif pengguna agar dapat terhubung sehingga memudahkan dalam mengintegrasikan fasilitas google yang ada pada android dengan browser. Contohnya, Google Calendar, rekomendasi keyword pada kolom search Google, dan login otomatis pada layanan Google lainnya

→ Pengguna dapat memasang ekstensi yang menunjang kinerja

Ekstensi yang tersedia di Chrome sangat beragam dan akan sangat mendukung kinerja penggunanya. Contoh : pemblokir iklan, ekstensi tools perusahaan, ekstensi mempercepat download/upload file dan sebagainya.

→ Membuka beberapa bentuk file didalam browser

Chrome selain sebagai browser untuk menjelajah web, browser ini dapat digunakan untuk membuka file seperti .pdf, docx, .mp3(audio), mp4 (video)

→ Melakukan inspect element pada suatu webpage

Hal ini dapat dilakukan di Chrome dengan tujuan mengedit suatu tampilan web pada susunan HTML / CSS namun hal tersebut hanya dapat dilihat pada komputer pengguna yang melakukan perubahan denga inspect element.

18. Jenis protokol yang digunakan untuk menentukan host remote dan jenis sistem file yang akan diakses dan menempatkan di suatu direktori, serta protokol NFS melakukan I/O pada remote file system adalah...​


Jenis protokol yang digunakan untuk menentukan host remote dan jenis sistem file yang akan diakses dan menempatkan di suatu direktori adalah Network File System (NFS).

NFS merupakan protokol yang digunakan untuk memungkinkan file dan direktori yang disimpan di server untuk diakses dan digunakan oleh komputer klien dalam jaringan. NFS memungkinkan klien untuk mengakses file di server dengan cara yang sama seperti file yang ada di lokal sistem file klien.

NFS menggunakan protokol Remote Procedure Call (RPC) untuk mengirim permintaan dari klien ke server dan sebaliknya. Dalam hal I/O pada remote file system, NFS memungkinkan klien untuk membaca dan menulis file di server melalui protokol NFS.

26. apa fungsi google chrome 1.0.154.53 ?​


Jawaban:

Versi Google Chrome yang dipasang di host jarak jauh lebih lama dari 1.0.154.53. Versi tersebut dilaporkan dipengaruhi oleh beberapa kerentanan:

- Browser menggunakan header Host HTTP untuk menentukan konteks respons CONNECT 4xx/5xx dari server proxy. Ini dapat memungkinkan penyerang man-in-the-middle untuk mengeksekusi kode skrip arbitrer dalam konteks server yang sah. (CVE-2009-2060)

- Browser menampilkan sertifikat yang di-cache untuk halaman respons CONNECT 4xx/5xx dari server proxy. Penyerang man-in-the-middle dapat mengeksploitasi ini dengan menampilkan halaman web palsu dengan sertifikat valid dari situs web yang sah.

32. Apa fungsi dari remote server


Fungsi dari remote server adalah untuk mengakses perangkat komputer lain dengan menggunakan jalur khusus yang sudah disediakan. Remote server identik dengan mengakses sebuah komputer server. Beberapa fungsi dari remote server adalah

Mengakses secara penuh komputer server, baik keyboard, mouse dan semua aktivitas komputer yang diremoteMelakukan perubahan pada komputer server apabila user diberi akses untuk melakukanyaPembahasan

Dengan adanya remote server, komputer client bisa langsung melakukan akses terhadap komputer servernya. Namun sebelum melakukan remote server, terdapat beberapa langkah yang harus dilakukan, yaitu

Menentukan komputer yang akan dijadikan serverBerikan akses pada komputer server agar bisa diakses oleh komputer lainPembuatan user untuk melakukan remotePelajari lebih lanjut

39. Bagian bagian remote dvd,jelaskan fungsi dari bagian2 remote tersebut !
