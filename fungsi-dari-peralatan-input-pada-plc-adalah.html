4. kelompokan alat input dan fungsinya? ​


- Keyboard. Keyboard adalah perangkat yang digunakan untuk memasukkan data berupa huruf, angka, dan simbol serta melakukan perintah-perintah pada komputer. ...

-Mouse. Mouse merupakan sebuah alat masukan(input) yang berguna sebagai penunjuk posisi kursor/pointer pada layar monitor. alat ini sangat berguna untuk sistem operasi yang berbasis grafis-Joystick. ...

-scanner. Scanner merupakan alat input yang memiliki fungsi hampir sama seperti mesin fotocopy dengan prosesnya penggunaannya mencopy data objek lalu di pindahkan kedalam memori yang dapat dilihat hasilnya pada layar monitor.

-joystick.Joystick merupakan alat masukan yang di gunakan untuk memasukkan perintah kepada pemain dalam sebuah permainan pada komputer

-webcam. WebCam merupakan sebutan bagi kamera real time yang gambarnya bisa di akses melalui WWW (World Wide Web), program instant messaging atau aplikasi video call yang dapat memasukkan hasil data dan dibentuk dalam format digital.

-Cd room/DVD room. CD ROM / DVD ROM merupakan perangkat masukan yang berfungsi untuk membaca data atau program data media penyimpanan data CD / DVD.

- flashdick. Flashdisk merupakan alat input (masukan) yang digunakan untuk menyimpan data dengan ukuran tertentu.

9. sebutkan masing masing tiga contoh alat pada sistem PLC


Alat pada PLC (Programmable Logic Controller) digolongkan menjadi 3:
1. Input device (contohnya Push button, Motor starter, Limit switch)
2. Controllers (contohnya Counter, Logic unit, Timer)
3. Output device (contohnya Alarm, Fans, Light)

11. Jelaskan beberapa alat input pada input device yang berfungsi ganda


Alat input yang berfungsi ganda disebut terminal.Terminal dapat digolongkan menjadi 3,yaitu:
1.Non Intelligent terminal hanya berfungsi sebagai alat memasukkan input dan penampil output,dan tidak bisa diprogram karena tidak mempunyai alat pemroses.
2.Smart Terminal mempunyai alat pemroses dan memori didalamnya sehingga input yang terlanjur dimasukkan dapat dikoreksi kembali.
3.Intelligent dapat diprogram oleh pemakai,peralatan yang hanya berfungsi sebagai alat input dapat digolongkan menjadi alat input langsung dan tidak langsung.

19. sebutkan beserta fungsinya apa itu alat input dan output!


Jawaban:

Misalkan pada komputer:

alat input : alat yg berfungsi untuk memasukan data ke dalam komputer.

alat output : alat yg berfungsi sebagai keluaran data pada komputer.

Penjelasan:

Contoh alat input: mouse, keyboard, microphone.

Contoh alat output: monitor, printer, speaker

Jawaban:

°input : adalah perangkat keras yang digunakan untuk memasukkan data serta perintah ke dalam komputer

contohnya:

1. Keyboard : keyboard merupakan papan berisikan tombol huruf dan angka. Keyboard sering digunakan untuk memberi perintah kepada komputer, mengetik dokumen, atau menjalankan proses input lainnya

2. Mouse : alat yang bisa digunakan untuk mengendalikan pergerakan kursor yang ada di layar. Mouse memiliki dua bagian tombol yang ketika diklik bisa menghasilkan perintah tertentu

3. Mikrofon : Alat input ini sering dipakai untuk memasukkan data atau perintah dalam bentuk suara

4. Scanner : Merupakan alat yang sering dimanfaatkan untuk mengubah materi cetak, seperti teks atau gambar, ke bentuk yang bisa dibaca atau dilihat di komputer

5. Web cam : Alat ini sering dipasang di bagian atas komputer atau laptop. Web cam dimanfaatkan untuk mengambil dan gambar atau video dan kemudian dikirim ke orang lain

6. Joy stick : Adalah salah satu alat input komputer yang bisa digerakkan ke berbagai arah

•output merupakan perangkat keras yang digunakan untuk menyampaikan informasi kepada satu orang atau lebih

Contohnya :

1. Monitor : Monitor adalah bagian unit komputer yang menampilkan visual data, gambar atau program yang sedang dijalankan. Monitor terdiri atas berbagai jenis, ada yang LCD, flat dan lain sebagainya

2. Speaker : Speaker sering juga disebut pengeras suara. Biasanya speaker dipakai untuk mendengarkan musik, suara ataupun bentuk audio lainnya

3. Printer : alat output komputer yang sering dipakai untuk memproduksi data dalam bentuk fisik. Data yang bisa dihasilkan lewat printer diantaranya teks, gambar dan tangkapan layar video

4. Plotter : alat output komputer yang dimanfaatkan untuk mencetak hasil desain grafis dalam ukuran besar. Fungsinya hampir sama seperti printer, hanya saja plotter berfokus untuk mencetak hasil grafis dengan ukuran besar

5. Proyektor : alat output komputer yang dimanfaatkan untuk mencetak hasil desain grafis dalam ukuran besar. Fungsinya hampir sama seperti printer, hanya saja plotter berfokus untuk mencetak hasil grafis dengan ukuran besar

6. Headset : Headset adalah peralatan output yang bisa digunakan di laptop, komputer ataupun gadget. Fungsi alat ini adalah untuk mendengarkan suara dan berbicara
