10. Diketahui fungsi f(x) = 2 – 3x dengan domain {-2, -1, 0, 1, 2}. Daerah hasil fungsi f tersebut adalahTolong jawab dengan caranya ​


Jawaban:

Relasi fungsi

fungsi f(x) = 2 - 3x

Domain fungsi (Df) = {-2, -1, 0, 1, 2}

Maka:

f(x) = 2 - 3x

f(-2) = 2 - 3(-2) = 2+6 = 8

f(-1) = 2 - 3(-1) = 2+3 = 5

f(0) = 2 - 3(0) = 2 - 0 = 2

f(1) = 2 - 3(1) = 2 - 3 = -1

f(2) = 2 - 3(2) = 2 - 6 = -4

Jadi, daerah hasil fungsi/Range (Rf) tersebut adalah {8, 5, 2, -1, -4}.

16. domain fungsi g(x) = 3/√x²+3x+2 adalah...​


✇ Fungsi RasionaL

-

Domain fungsi g (x) = 3 / √(x² + 3x + 2) adalah x ∈ (-∞ , -2) ∪ (-1 , ∞)

PEMBAHASAN :

Perhatikan bahwa ada 2 syarat dalam menentukan domain pada kasus diatas , yaitu sebagai berikut :

Penyebut √(x² + 3x + 2) ≠ 0 Penyebut √(x² + 3x + 2) < 0

Oleh karena syarat diatas , maka diperoleh :

Dg (x) = { x | x² + 3x + 2  > 0 }

Dg (x) = { x | (x + 1)(x + 2) > 0 }

Dg (x) = { x | x < -2 atau x > -1 }

Bentuk Lain :

Dg (x) = x ∈ (-∞ , -2) ∪ (-1 , ∞)

32. carilah domain fungsi dari akar x kuadrat - 3x + 2


[tex]f(x)=\sqrt{x^2-3x+2}[/tex]
Domainnya adalah ketika hasil dari pemetaan oleh x tidak negatif karena sifat akar pangkat 2 akan menghasilkan bukan bilangan real jika negatif. Maka:
[tex]\sqrt{x^2-3x+2}\geq 0 \\ x^2-3x+2\geq 0 \\ (x-1)(x-2)\geq 0 \\ x\leq 1$ atau $x\geq 2[/tex]

Maka, domain dari fungsi tersebut adalah:
[tex]D=\{x|x\leq-1$ atau $x\geq 2,x\in R\}[/tex]
