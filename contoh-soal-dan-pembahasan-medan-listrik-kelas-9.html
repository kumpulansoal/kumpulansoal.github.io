29. mohon bantuannya kak soal medan listrik


ABCD persegi, dengan AC = 4 cm
Qa = Qb = Qc = Qd = Q = 0,5 C
Fc = __?
Eo = __?
Vo = __?

Gaya pada muatan Qc
Fc = Fcb + Fca cos 45° + Fcd cos 45°
Fc = Fcb + 2 Fca cos 45°
Fc = k Qc Qb / CB² + 2 k Qc Qa / AC² cos 45°
Fc = k Q² (1/CB² + 2 × 1/CA² • cos45°)
Fc = 9×10^9 (0,5)² (1/(4×10^-2)² + 2 × 1/(2√2×10^-2)²) • ½√2
Fc = 5,38×10¹² N ←

Kuat medan di O
Eo = 0 ←

Potensial di O
Vo = 4 k Q/a
Vo = 4 × 9×10^9 (0,5) / 2×10^-2
Vo = 9×10¹¹ V ←
