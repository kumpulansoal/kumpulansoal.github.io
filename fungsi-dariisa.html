34. Uang ahmad 1/5 uang darisa. Uang darisa 1 1/3 uang takenaba. Jika selisih uang ahmad dan takenaba adalah Rp 99.000,00 maka tentukan uang mereka masing masing!


Jawab:

uang Ahmad = Rp36.000,00

uang Darisa = Rp180.000,00

uang Takenaba = Rp135.000,00

Penjelasan dengan langkah-langkah:

Ahmad = A

Darisa = B

Takenaba = C

A = 1/5 B  ---> 5A = B

                      A : B = 1 : 5

B = 1 1/3 C ---> B = 4/3 C

                       3B = 4C

                       B : C = 4 : 3

buat perbandingan ketiganya

A : B = 1 : 5     | × 4

B : C = 4 : 3    | × 5

sehingga

A : B = 4 : 20

B : C = 20 : 15

pada bagian B perbandingannya sudah sama yaitu 20, maka perbandingan ketiganya adalah

A : B : C = 4 : 20 : 15

Selisih A dan C = Rp99.000,00

- mencari uang masing-masing

 uang Ahmad = 4 / (15 - 4) x Rp99.000,00

                        = 4/11 x Rp99.000,00

                        = 4 x Rp9.000,00

                        = Rp36.000,00

- uang Darisa = 20 / (15 - 4) x Rp99.000,00

                       = 20/11 x Rp99.000,00

                       = 20 x Rp9.000,00

                       = Rp180.000,00

- uang Takenaba = uang Ahmad + Rp99.000,00

                              = Rp36.000,00 + Rp99.000,00

                              = Rp135.000,00

Jawab:

Uang ahmad : 36.000

Uang darisa : 180.000

Uang takenaba : 135.000

Penjelasan dengan langkah-langkah:

Perbandingan uang ahmad & darisa:

Ahmad 1/5 : 1 darisa

Ahmad 1 : 5 darisa

Perbandingan uang darisa & takenaba:

Darisa 4/3 : 1 takenaba

Darisa 4 : 3 takenaba

Perbandingan uang 3 orang

Ahmad : darisa : takenaba

    1      :  5  :  4  :       3

    4     : 20 : 20 :      15

    4     :     20    :       15

uang ahmad:

=> 99.000×4/15-4

=> 99.000×4/11

=> 36.000

Uang darisa:

=> 20/4×36.000

=> 180.000

Uang takenaba:

=> 15/4×36.000

=> 135.000
