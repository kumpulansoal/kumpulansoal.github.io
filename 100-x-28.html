Untuk menyelesaikan perhitungan ini, kita dapat menggunakan urutan operasi matematika BIDMAS, yaitu:

1. Kurung ()

2. Pangkat (eksponen)

3. Perkalian dan Pembagian (kiri ke kanan)

4. Penjumlahan dan Pengurangan (kiri ke kanan)

Dalam persamaan ini, tidak ada kurung atau pangkat, jadi kita langsung melakukan perkalian dan pembagian terlebih dahulu, lalu diikuti dengan penjumlahan. Sehingga:

(100) (-20) x 3 + 28 = -2000 x 3 + 28

Selanjutnya, kita lakukan perkalian antara -2000 dan 3, lalu diikuti dengan penjumlahan 28. Sehingga:

-2000 x 3 + 28 = -6000 + 28 = -5972

Jadi, hasil dari perhitungan tersebut adalah -5972.
