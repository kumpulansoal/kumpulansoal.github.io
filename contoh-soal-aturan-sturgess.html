1. Boleh minta soal aturan sturgess dalam matematika Bab Statitiska ?


Contoh soal penggunaan aturan Sturgess.

Diketahui data-data sebagai berikut:

42 62 59 46 53 60 65 72 51 53 61 63 69 64 65 67 72 91 52 59 69 72 88 65 65 75 67 78 81 47 79 55 62 32 71 67 62 54 66 69

N = 40 dataData terkecil = 32, data terbesar = 91Range = 91 - 32 = 59Banyak kelas k = 1 + 3,3 log 40 = 6,3. Dibulatkan k = 6 kelasPanjang interval kelas = ⁵⁹/₆ = 9,8 dibulatkan menjadi 10Batas bawah kelas pertama = 32, batas atas kelas pertama = 41

Setelah itu seluruh data berurutan dimasukkan ke dalam tabel distribusi frekuensi sesuai kelas intervalnya. Tabel dapat dilihat pada gambar terlampir.

Pembahasan

Dalam pembentukan tabel frekuensi data berkelompok, kita mulai dengan menetapkan banyak kelas. Kelas yang terlalu banyak ada kemungkinan terdapat kelas yang tidak mengandung pengamatan, atau jika kelas terlalu sedikit dapat mengakibatkan hilangnya sebagian keterangan. Oleh karena itu digunakan suatu cara atau langkah-langkah dalam membuat tabel sebaran frekuensi.

Jangkauan (range) = nilai data terbesar - nilai data terkecil.Banyak kelas dengan aturan Sturgess, yaitu k = 1 + 3,3 log N. Nilai k sebagai bilangan bulat dan N sebagai banyak data.Panjang interval kelas = jangkauan dibagi oleh banyak kelas.Siapkan batas bawah kelas pertama yang dapat diambil dari nilai pengamatan terkecil atau data minimum.Pelajari lebih lanjutMenentukan modus dari data histogram https://brainly.co.id/tugas/10180825Mencari median dari tabel data berkelompok brainly.co.id/tugas/2320008

4. Diketahui 100 buah data dengan nilai tertinggi 214 dan nilai terendah sebesar 120. menggunakan aturan sturgess tentukan: a. banyaknya kelas b. interval/panjang kelas


Penjelasan dengan langkah-langkah:

Diketahui :

n = 100

Xmax = 214

Xmin = 120

Ditanya :

a). Banyaknya kelas?

b). Interval/panjang kelas?

Jawab :

Langkah pertama tentukan jangkauan terlebih dahulu.

[tex]jangkauan = xmax - xmin[/tex]

Jangkauan = 214 - 120

= 94

Langkah kedua menentukan banyak kelas dengan aturan sturgess

[tex]k = 1 + 3.3 log(n) [/tex]

k = 1 + 3.3 log 100

= 1 + 3.3 (2)

= 1 + 6.6

= 7.6 (dibulatkan menjadi 8)

Langkah ketiga menentukan panjang interval kelas.

[tex]interval \: kelas \: = \frac{jangkauan}{banyak \: kelas} [/tex]

interval kelas = 94 : 8

= 11.75 (dibulatkan menjadi 12)

10. Sekumpulan data yang berjumlah 100 datum memiliki datum terbesar 90 dan datum terkecil 35. jika data tersebut akan disusun dalam tabel distribusi frekuensi kelompok dengan bantuan aturan sturgess, panjang interval kelasnya adalah.....


Jawab:

Panjang interval kelas adalah 7.

Penjelasan dengan langkah-langkah:

Aturan Sturges:

Range = Data terbesar - data terkecil

Jumlah Kelas (k) = 1 + 3,3log(n)

Panjang Interval Kelas (p) = Range / k

Sehingga, dari soal diperoleh:

Range = 90 - 35 = 55

Jumlah kelas (k) = 1 + 3,3log(100) = 1 + (3,3 × 2) = 1 + 6,6 = 7,6 = 8 (dibulatkan ke atas)

Panjang Interval Kelas (p) = 55 / 8 = 6,875 = 7 (dibulatkan ke atas)

Pembahasan

nilai ulangan matematika dari 100 siswa mempunyai nilai terendah 30 dan nilai tertinggi 90

n = 100

data terbesar = 90

data terkecil = 30

Langkah langkah mencari panjang kelas (intervalnya) adalah

jangkauan = data terbesar - data terkecil

j = 90 - 30

i = j/k

i = 60/7,6

i = 7,89

i = 8 (dibulatkan)

19. Dari 60 buah data diketahui data tertinggi 62 dan terendah 27. Jika data tersebut disusun dalam distribusi frekuensi dengan bantuan aturan sturgess,maka interval (panjang kelas) adalah..


Dari 60 buah data diketahui data tertinggi 62 dan terendah 27. Jika data tersebut disusun dalam distribusi frekuensi dengan bantuan aturan sturgess, maka interval (panjang kelas) adalah 5. Untuk menyajikan data dalam bentuk tabel distribusi frekuensi, dapat kita gunakan aturan Sturgees yaitu

1. Menentukan jangkauan/rentang dari data

J = data terbesar – data terkecil

2. Menentukan banyak kelas

K = 1 + 3,3 log n

dengan n adalah banyaknya data

3. Menentukan panjang interval tiap kelas

P = [tex] \frac{Jangkauan}{Banyak \: kelas} = \frac{J}{K}[/tex]  

Catatan: Untuk pembulatan bilangannya bisa dibulatkan ke atas atau dibulatkan ke bawah, tetapi biasanya dibulatkan ke atas.


Pembahasan

Diketahui

n = 60

data terbesar = 62

data terkecil = 27


Ditanyakan

Interval (panjang kelas) = ...?


Jawab

Langkah 1. Menentukan jangkauan

J = data terbesar – data terkecil

J = 62 – 27

J = 35


Langkah 2. Menentukan banyak kelas

K = 1 + 3,3 log n

K = 1 + 3,3 log 60

K = 1 + 3,3 (1,7781)  

K = 1 + 5,90073

K = 6,9007 (dibulatkan ke atas)

K = 7  


Langkah 3. Menentukan panjang kelas

P = [tex]\frac{J}{K}[/tex]

P = [tex]\frac{35}{7}[/tex]

P = 5


Jadi interval (panjang kelas) data tersebut adalah 5

jangkauan = data terbesar - data terkecil

j = 183 - 145

37. Dari 80 data, diketahui datum tervesar 92 dan datum terkecil 31. jika data tersebut akan disusun dalam tabel distribusi frekuensi kelompok, dengan bantuan aturan sturgess, panjang interval kelasnya adalah... petunjuk: log= 1,903


Menentukan Tabel Distribusi Frekuensi dengan Aturan Sturges

Menentukan banyaknya kelas (k) yaitu dengan rumus,

k = 1 + 3,322 log n , di mana

n = banyaknya data

r = selisih nilai terbesar dan nilai terkecil = nilai terbesar - nilai terkecil

Menentukan panjang interval (i), rumusnya yaitu :

i = [tex]\frac{r}{k}[/tex]

Pembahasan

n = 80

k = 1 + 3,322 log 80

  = 1 + 3,322 (1,903)

  = 1 + 6,321766

  = 7,321766

k = 7 (diambil yang kecil) dan k = 8 (diambil yang besar)

r = 92 - 31 = 61

Panjang interval (i) untuk k = 7 yaitu :

i = [tex]\frac{61}{7}[/tex] = 8,71428 ... = 9

Panjang interval (i) untuk k = 8 yaitu :

i = [tex]\frac{61}{8}[/tex] = 7,625 = 8
