17. Buat lah sebuah 3 fungsi overloading yang memiliki 1 parameter - Jika parameternya int maka outputnya “Tipe data int, hasilnya [parameter * 100]” - Jika parameternya double maka outputnya “Tipe data double, hasilnya [parameter * 1000]” - Jika parameternya string maka outputnya “Tipe data string, nilainya Hello [parameter]” contoh pada fungsi main : int main() { overloadingFunc(10; overloadingFunc(10.0); overloadingFunc("World); return 0; } outputnya : Tipe data int, hasilnya 10000 Tipe data float, hasilnya 1000 Tipe data string, nilainya Hello World


Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}

37. Tay 12. Tipe data yang digunakan manipulasi string adalah..... a. alfanumerik b. bit-tunggal C. logikal d numerik TO LONG DIJAWAB ​


Tipe data yang digunakan untuk manipulasi string adalah alfanumerik.
(opsi a)
 

Pembahasan

“String“ adalah tipe data majemuk dari karakter, yang merupakan barisan hingga dari kumpulan simbol yang diambil dari himpunan karakter, yang meliputi karakter huruf (alfabet), angka (numerik), simbol lain, dan karakter kontrol atau karakter khusus.

Pada program/aplikasi, input atau output data string pada umumnya meliputi karakter alfabet dan numerik. Sebagai contoh, data nama, alamat, dsb.
Manipulasi string (pengubahan atau modifikasi string) terjadi pada pemrosesan, sehingga dari data input, program dapat menyajikan informasi yang sesuai, atau menentukan tindakan yang telah ditetapkan untuk suatu kondisi input tertentu.

Gabungan/kombinasi dari himpunan karakter alfabet dan numerik adalah: alfanumerik.

∴  Dengan demikian, tipe data yang digunakan untuk manipulasi string adalah alfanumerik.
