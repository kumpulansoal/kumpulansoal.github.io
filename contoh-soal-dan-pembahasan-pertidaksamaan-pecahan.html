2. 4/5 - 2/4 = 6/20 = 3/10

5 = 5

4 = 2 x 2

Kpk = 2 x 2 x 5 = 4 x 5 = 20

=> ( 20 : 5 x 4 ) - ( 20 : 4 x 2 ) per ( / ) 20
