2. Sebutkan struktur program dalam VB net​


Jawaban:

Sebelum kita mempelajari blok bangunan dasar bahasa pemrograman VB.Net, mari kita lihat struktur program VB.Net yang minimal sehingga kita dapat menganggapnya sebagai referensi dalam bab-bab yang akan datang.

Program VB.Net pada dasarnya terdiri dari beberapa bagian berikut:

Deklarasi namespace

Kelas atau modul

Satu atau lebih prosedur

Variabel

Prosedur utama

Pernyataan & Ekspresi

Komentar

Mari kita lihat sebuah kode sederhana yang akan mencetak kata-kata "Hello World":

1

2

5

Imports System

Module Module1

'This program will display Hello World

Sub Main()

Console.WriteLine("Hello World")

Console.ReadKey()

End Sub

End Module

Bila kode diatas dikompilasi dan dieksekusi, maka menghasilkan hasil sebagai berikut:

Hello, World !

Mari kita lihat berbagai bagian dari program di atas:

Baris pertama dari program Imports System digunakan untuk memasukkan namespace System dalam program ini.

Baris berikutnya memiliki deklarasi Modul, Modul Modul1. VB.Net benar-benar berorientasi objek, jadi setiap program harus berisi modul kelas yang berisi data dan prosedur yang digunakan program Anda.

Kelas atau Modul umumnya mengandung lebih dari satu prosedur. Prosedur berisi kode yang dapat dieksekusi, atau dengan kata lain, mereka mendefinisikan perilaku kelas. Prosedur bisa berupa salah satu dari berikut ini:

Function

Sub

Operator

Set

AddHandler

RemoveHandler

RaiseEvent

Baris berikutnya ('Program ini) akan diabaikan oleh compiler dan telah ditambahkan komentar tambahan di program ini.

Baris berikutnya mendefinisikan prosedur utama, yang merupakan titik masuk untuk semua program VB.Net. Prosedur Utama menyatakan apa yang akan dilakukan modul atau kelas saat dijalankan.

Prosedur Utama menentukan perilakunya dengan pernyataan tersebut.

Console.WriteLine ("Hallo World") WriteLine adalah metode kelas Console yang didefinisikan dalam namespace Sistem. Pernyataan ini menyebabkan pesan "Hallo, World!" Untuk ditampilkan di layar. Baris terakhir Console.ReadKey () adalah untuk Pengguna VS.NET. Ini akan mencegah layar agar tidak berjalan dan menutup dengan cepat saat program diluncurkan dari Visual Studio.NET.

Compile & Execute Program VB.Net:

Jika Anda menggunakan Visual Studio.Net IDE, lakukan langkah-langkah berikut:

Mulai Visual Studio.

Pada bilah menu, pilih File, New, Project.

Pilih Visual Basic From Template

Pilih Console Application

Tentukan nama dan lokasi proyek Anda dengan menggunakan tombol Browse, kemudian pilih tombol OK

Proyek baru muncul di Solution Explorer.

Tulis kode di Editor Kode.

Klik tombol Run atau tombol F5 untuk menjalankan proyek. Jendela Command Prompt muncul yang berisi baris Hello World.

Anda dapat mengkompilasi sebuah program VB.Net dengan menggunakan command line daripada Visual Studio IDE:

Buka editor teks dan tambahkan kode yang disebutkan di atas.

Selanjutnya, ketik helloworld untuk menjalankan program anda.

Anda akan dapat melihat "Hello World" yang tercetak di layar.

6. jelaskan fungsi coding dalam dev C++​


Jawaban:

Apa itu Fungsi?

Fungsi adalah sub-program yang bisa digunakan kembali baik di dalam program itu sendiri, maupun di program yang lain.

Fungsi dapat menerima input dan menghasilkan output.

Contoh fungsi yang sering kita buat adalah fungsi main().

Fungsi ini memang wajib ada di setiap program C++, karena fungsi inilah yang akan dieksekusi pertama kali saat program berjalan.

Seperti yang saya bilang tadi:

“Kalau kita menumpuk semua kode program di dalam fungsi main(), maka program akan menjadi sulit terbaca.”

Karena itu, kita perlu membuat fungsi tersendiri agar program tidak berantakan.

Bagaimana cara buatnya?

Silahkan lanjutkan membaca:

Cara Membuat Fungsi pada Bahasa C++

Fungsi pada C++ dapat kita buat dengan cara seperti ini:

Cara Membuat Fungsi pada C++

Fungsi biasanya akan mengembalikan sebuah nilai dari hasil prosesnya. Karena itu, kita harus menentukan tipe data untuk nilai yang akan dikembalikan.

Apabila fungsi tersebut tidak memiliki nilai kembalian, maka kita harus menggunakan tipe void untuk menyatakan kalau fungsi tersebut tidak akan mengembalikan nilai apa-apa.

Contoh:

void nama_fungsi(){

cout << "Ini adalah sebuah fungsi\n";

}

Lalu untuk parameter bersifat opsional, boleh ada boleh tidak.

Tergantung dari fungsi yang dibuat. Jika fungsi itu membutuhkan input, maka kita harus membuatkan paramter.

Tapi kalau tidak menerima input apapun, ya tidak perlu dibuat.

Fungsi yang tidak menerima input, kadang juga disebut dengan prosedur.

Okee..

Sekarang mari kita coba membaut fungsi pada program C++.

Silahkan buat file baru bernama contoh_fungsi.cpp kemudian isi dengan kode berikut:

#include <iostream>

using namespace std;

// membuat fungsi say_hello()

void say_hello(){

cout << "Hello Selamat Datang!\n";

}

int main(){

// memanggil fungsi say_hello()

say_hello();

return 0;

}

Hasilnya:

Program fungsi say_hello

Fungsi say_hello() dapat kita panggil berulang kali pada fungsi main().

Contoh:

#include <iostream>

using namespace std;

// membuat fungsi say_hello()

void say_hello(){

cout << "Hello Selamat Datang!\n";

}

int main(){

// memanggil fungsi say_hello()

say_hello();

say_hello();

say_hello();

return 0;

}

Maka hasilnya:

Program fungsi say_hello

Jadi, cukup buat fungsi satu kali. Kita bisa panggil berkali-kali.

20. Tuliskan 5 fungsi yang ada dicoding scratch selain event!​


Jawaban:

Berikut ini adalah 5 fungsi umum yang ada di Scratch selain event:

1. Motion: Fungsi Motion digunakan untuk mengatur pergerakan objek pada layar. Fungsi ini memungkinkan pengguna untuk menggerakkan objek ke kiri, ke kanan, ke atas, dan ke bawah, serta memutar objek atau mengubah ukurannya.

2. Looks: Fungsi Looks digunakan untuk mengatur tampilan objek pada layar. Fungsi ini memungkinkan pengguna untuk mengubah warna, ukuran, dan transparansi objek, serta memasukkan gambar atau teks pada objek.

3. Sound: Fungsi Sound digunakan untuk menambahkan efek suara pada proyek Scratch. Fungsi ini memungkinkan pengguna untuk menambahkan suara yang telah disediakan oleh Scratch atau merekam suara mereka sendiri.

4. Control: Fungsi Control digunakan untuk mengatur alur program dan mengontrol tindakan pengguna. Fungsi ini memungkinkan pengguna untuk mengatur perulangan, pengaturan kondisi (if-else), serta menjalankan program pada saat yang ditentukan.

5. Pen: Fungsi Pen digunakan untuk menggambar garis atau bentuk pada layar. Fungsi ini memungkinkan pengguna untuk mengatur ukuran, warna, dan gaya garis yang digunakan untuk menggambar. Fungsi Pen dapat digunakan untuk membuat animasi yang menarik atau untuk menunjukkan proses penghitungan pada program
