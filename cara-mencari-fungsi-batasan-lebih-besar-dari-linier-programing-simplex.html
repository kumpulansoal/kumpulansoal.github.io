1. PERBEDAAN ANTARA LINIER PROGRAMING METODE GRAFIK DENGAN LINIER PROGRAMING METODE SIMPLEKS. JELASKAN.​


Perbedaan metode grafik dan simpleks adalah metode grafik hanya sanggup dipakai untuk dua masalah saja. Sementara, metode simpleks sanggup menyelesaikan kendala lebih dari dua kendala, yang berarti metode ini mempunyai setidaknya tiga kendala didalamnya.

Pembahasan

Program linear merupakan suatu program yang memiliki fungsi menyelesaikan berbagai masalah yang berhubungan dengan suatu variabel dan memiliki batasan dengan pertidaksamaan linear. Program linear juga disebut dengan metode matematika yang berfungsi memecahkan masalah berkaitan dengan optimasi linear (maksimum/minimum). Persoalan yang berkaitan dengan program linier selalu terkait dengan fungsi objektif berdasarkan kondisi yang membatasinya. Dalam hal ini, optimasi berupa memaksimalkan atau meminimalkan fungsi objektif.Metode grafik merupakan metode yang digunakan untuk menyelesaikan permasalahan program linear yang sering dipakai, karena mudah dan tidak menghabiskan banyak waktu. Hanya saja, metode ini hanya bisa dipakai untuk menyelesaikan permasalahan program linear dengan dua variabel saja. Metode simpleks merupakan teknik penentuan solusi optimal yang dipakai pada pemrograman linier yang dilakukan dengan memeriksa titik ekstrim satu persatu dengan perhitungan interaktif. Sehingga, penentuan solusi optimal dengan metode simpleks tersebut dilakukan dengan tahap demi tahap yang kita kenal dengan iterasi.

Persoalan linear programing yang diselesaikan dengan cara metode grafik dapat diselesaikan juga dengan metode simpleks. Namun sebaliknya, persoalan yang hanya dapat diselesaikan dengan metode simpleks tidak akan bisa diselesaikan dengan metode grafik.

9. Jenis-jenis epitelium simplex beserta letak dan fungsinya​


1. Epitel pipih selapis

terletak di alveoulus (paru paru), dinding kapiler, ginjal (Kapsula Bowman ginjal), ruang-ruang jantung.

Fungsi : tempat terjadinya difusi

2. Epitel kubus selapis

terletak di retina mata, ovarium

Fungsi : proteksi (pelindung), absorbsi (penyerapan), sekresi

3. Epitel silindris selapis

terletak di saluran pencernaan, bronkus (paru-paru), rahim, serta beberapa kelenjar lainnya.

Fungsi : proteksi (pelindung), absorbsi (penyerapan), sekresi

4. Epitel silindris bersilia

terletak di Trakea (tenggorokan), bronkus (paru-paru), dinding dalam (ovidak), rongga hidung.

Fungsi : proteksi (pelindung), penghasil mucus, tempat terjadinya difusi, dan absorpsi.

5. Epitel silindris berlapis semu

terletak di Rongga hidung, trakea

Fungsi : proteksi (pelindung), sekresi, dan gerakan zat yang melewati permukaan

17. Tugas kalian mencari 2 masalah sehari hari kemudian selesaikan dengan menggunakan PROGRAM LINIER (Boleh mencari google ataupun youtube); Langkah langkahnya : 1. Model matematika (batasan batasan dan Fungsi Tujuan) berupa Pertidaksamaan Linier 2 Variabel 2. Daerah Penyelesaian 3. Titik-titik penyelesaian 4. Nilai optimum​


Penerapan Program Linear dengan masalah sehari-hari.

Soal :

Untuk menghasilkan barang A seharga Rp.120.000 diperlukan bahan baku 30kg dan waktu kerja mesin 18 jam, Produk B seharga Rp.135.000 memerlukan bahan baku 20kg dan waktu kerja mesin 30 jam. Jika banyaknya produk A dan B masing-masing x dan y dibuat selama 774 jam waktu kerja mesin dan 750 kg bahan baku, maka pendapatan maksimum yg diperolah adalah ...

Penjelasan dengan langkah-langkah

Diketahui :

Barang A memerlukan bahan 30 kg dan waktu 18 jam.

Barang B memerlukan bahan 20 kg dan waktu 30 jam.

Banyaknya produk A dan B dibuat selama 774 jam dan 750 kg bahan baku.

Harga barang A Rp 120.000 dan barang B seharga Rp 135.000.

Ditanya :

Pendapatan maksimum.

Jawab :

1. Model matematika

Maksimum fungsi tujuan

f (x, y) = 120.000 x + 135.000 y dengan kendala :

30x + 20y ≤ 750   ⇔   3x + 2y ≤ 75

18x + 30y ≤ 774     ⇔   3x + 5y ≤ 129

Banyak barang A dan B yang diproduksi selalu bernilai positif maka

x ≥ 0 dan y ≥ 0.

2. Daerah Penyelesaian

Pertidaksamaan 3x + 2y ≤ 75

x   |    y    |

0   | 37,5 |

25  |    0   |

Titik potong (0, 37,5) dan (25, 0)

Pertidaksamaan 3x + 5y ≤ 129

x  |    y    |

0  | 25,8 |

43 |    0   |

Titik potong (0, 25,8) dan 43 , 0)

Apabila koefisiennya positif dan mempunyai tanda ≤, maka daerah arsirannya ke bawah.

3. Titik-titik penyelesaian

Eliminasi

3x + 2y = 75

3x + 5y = 129  _

     -3y = -54

        y = -54/-3

        y = 18

Subtitusi

3x + 2y = 75

3x + 2 (18) = 75

3x + 36 = 75

      3x = 75 - 36

      3x = 39

        x = 39/3

        x = 13

Titik potong dari kedua pertidaksamaan adalah (13, 18).

4. Nilai optimum​

Menentukan pendapatan maksimum

Uji titik f (x, y) = 120.000 x + 135.000 y

(0, 25,8) ⇒ 120.000 (0) + 135.000 (25,8) = 3.483.000

(13, 18)    ⇒ 120.000 (13) + 135.000 (18)

               = 1.560.000 + 2.430.000 = 3.990.000   (MAKSIMUM)

(25, 0)   ⇒ 120.000 (25) + 135.000 (0) = 3.000.000

Jadi pendapatan maksimum yang diperolah adalah Rp 3.990.000.

19. x=-2y+12 dan x=2y-3 carilah titik potong kedua buah fungsi linier tersebut dan gambarkan dalam grafik fungsi linier


Penjelasan dengan langkah-langkah:

titik potong terjadi ketika x dan y dari kedua fungsi tersebut sama

maka pertama tama kita mencari nilai x dan y yang memenuhi kedua fungsi tersebut.

Karena

x = -2y + 12

maka

2y - 3 = -2y + 12

2y + 2y = 12 + 3

4y = 15

y = 15 / 4 = 3,75

karena y = 3,75 maka

x = -2y + 12

x = -2(3,75) + 12

x = -7,5 + 12

x = 4,5

titik potong kedua fungsi terletak pada (4,5 ; 3,75) atau dalam pecahan (4 1/2, 3 3/4)

mencari titik potong sumbu x dan y masing2 fungsi

untuk fungsi x = -2y + 12

titik potong sumbu x nya (y = 0)

x = -2(0) + 12

x = 12

jadi titik potong (12,0)

titik potong sumbu y (x= 0)

x = -2y + 12

0 = -2y + 12

2y = 12

y = 6

titik potong sumbu y (0,6)

untuk fungsi x = 2y - 3

titik potong sumbu x (y = 0)

x = 2(0) -3

x = -6

titik potongbsumbu y

0 = 2y - 3

2y = 3

y = 1,5

33. sebuah fungsi linier dan fungsi invers linier serta gambar satu koordinat


sebuah fungsi linier dan fungsi invers linier serta gambar satu koordinat

JAWAB

Inver dari fungsi linier : f(x) = ax + b
maka invers nya adalah : f-¹(x) = (x - b)/ a
sehingga jika
f(x) = 4x + 2 , maka
f-¹(x) = (x - 2)/4 = ¼x - ½

koordinat:
◆ untuk f(x) = 4x + 2
x = 0 maka f(x) = y = 2, titiknya (0,2)
y = 0 maka 4x + 2=0 → 4x = -2 → x = -½, titiknya (-½, 0)

◆ untuk f-¹(x) = ¼x - ½
x = 0 maka f(x) = y = -½, titiknya (0,-½)
y = 0 maka ¼x - ½ = 0 → ¼x = ½ → x = 2, titiknya (2, 0)

Silahkan digambar di koordinat kartesius menggunakan titik2 ini.
