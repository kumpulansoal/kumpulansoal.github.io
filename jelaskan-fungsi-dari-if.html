16. Carilah 3 jenis fungsi logika selain fungsi IF . Berikan penjelasannya!​


Jawaban:

Fungsi OR ✔  

Fungsi OR adalah fungsi bawaan di Excel yang dikategorikan sebagai Fungsi Logika. Fungsi OR adalah fungsi logis untuk menguji beberapa kondisi pada saat bersamaan. Rumus OR akan menghasilkan TRUE jika beberapa argument bernilai benar dan FALSE jika semua argument salah.  

Rumus fungsi OR:  

=OR(logical1; logical2; ...)

Fungsi AND ✔  

AND merupakan rumus fungsi logika pada Excel yang berguna untuk menghasilkan nilai TRUE, jika semua argument yang diuji bernilai benar. Jika FALSE, maka semua atau salah satu argument bernilai salah.  

Rumus fungsi AND:  

=AND (logical1; logical2; ...)

Fungsi NOT ✔  

Kebalikan dari rumus logika pada Excel AND, rumus NOT berguna untuk menghasilkan TRUE jika kondisi yang diuji salah dan FALSE jika yang diuji benar.  

Rumus fungsi NOT:  

=NOT(logical)

19. jelaskan fungsi dari rumus IF,HLookup,dan VLookup?​


Fungsi IF ialah salah 1 fungsi di Ms. Excel yang digunakan untuk menghasilkan nilai berdasarkan uji logika (benar/salah) tertentu.

Fungsi HLOOKUP ialah salah 1 fungsi di Ms. Excel untuk mencari sebuah nilai pada kolom(column) pertama sebuah data tabel sehingga didapat sebuah nilai tertentu pada baris lain yang berada pada kolom (column) yang sama dengan hasil pencarian. Data pada fungsi ini disusun secara horizontal.

Fungsi VLOOKUP adalah salah 1 fungsi pada Ms. Excel untuk mencari sebuah nilai pada baris (row) pertama sebuah data tabel sehingga didapat sebuah nilai data tertentu pada kolom lain yang berada pada baris (row) yang sama dengan hasil pencarian. Data pada fungsi ini disusun secara vertikal.

Tuliskan salah satu contoh fungsi If yang sudah pernah kamu lakukan dalam praktikum.​


Jawaban:

1. Fungsi IF adalah fungsi logika dalam berbagai bahasa pemrograman dan perangkat lunak seperti Excel yang digunakan untuk melakukan pengujian kondisi. Fungsi ini memungkinkan program atau spreadsheet untuk menjalankan tindakan tertentu berdasarkan apakah suatu kondisi atau ekspresi bernilai benar (true) atau salah (false).

2. Kegunaan fungsi logika IF antara lain:

- Menggunakan kondisi untuk mengambil keputusan dalam program atau spreadsheet.

- Menampilkan hasil yang berbeda berdasarkan kondisi yang diberikan.

- Menghitung nilai berdasarkan kondisi tertentu.

3. Fungsi IF tunggal adalah penggunaan fungsi IF dengan satu kondisi atau ekspresi saja, seperti IF(kondisi, nilai_jika_benar, nilai_jika_salah).

4. Fungsi IF majemuk adalah penggunaan beberapa fungsi IF dalam satu rumus untuk mengatasi beberapa kondisi yang berbeda. Contohnya, IF(kondisi1, nilai1, IF(kondisi2, nilai2, nilai_default)).

5. Fungsi IF bertingkat adalah penggunaan beberapa pernyataan IF yang bersarang (nested) dalam satu rumus. Misalnya, IF(kondisi1, IF(kondisi2, nilai1, nilai2), nilai_default).

6. Lambang-lambang logika pada Microsoft Excel:

- "=" digunakan untuk perbandingan kesetaraan (equal).

- "<" digunakan untuk perbandingan kurang dari (less than).

- ">" digunakan untuk perbandingan lebih dari (greater than).

- "<=" digunakan untuk perbandingan kurang dari atau sama dengan (less than or equal to).

- ">=" digunakan untuk perbandingan lebih dari atau sama dengan (greater than or equal to).

- "<>" digunakan untuk perbandingan tidak sama dengan (not equal to).

7. Beberapa operator logika yang pernah dipelajari adalah "AND" (dan), "OR" (atau), "NOT" (bukan), "IF" (jika).

8. Fungsi logika adalah perangkat matematika atau perangkat lunak yang digunakan untuk mengambil keputusan berdasarkan kondisi atau perbandingan tertentu.

9. Operator logika adalah simbol atau kata kunci yang digunakan untuk menggabungkan kondisi atau ekspresi dalam suatu pernyataan logika. Lambang logika adalah karakter yang digunakan untuk menggambarkan operasi perbandingan seperti kesetaraan, lebih besar, atau kurang dari.

10. Contoh fungsi IF dalam Excel:

Dalam sebuah spreadsheet Excel, Anda dapat menggunakan fungsi IF untuk menentukan apakah suatu siswa lulus ujian dengan rumus seperti IF(nilai >= 60, "Lulus", "Tidak Lulus"). Rumus ini akan menghasilkan "Lulus" jika nilai siswa sama dengan atau lebih besar dari 60, dan "Tidak Lulus" jika nilai di bawah 60.

34. Jelaskan perbedaan fungsi IF Tunggal, IF Ganda dan IF Majemuk!


Penggunaan fungsi IF dalam logika memiliki beberapa syarat yaitu sebagai berikut

= : sama dengan

< : lebih kecil dari

<= : lebih kecil atau sama dengan

> : lebih besar dari

>= : lebih besar atau sama dengan

< > : tidak sama dengan

Fungsi IF dari dari 2 jenis utama, yaitu IF Tunggal dan IF Ganda/Majemuk. Perbedaan fungsi IF Tunggal, IF Ganda dan IF Majemuk terletak pada jumlah kondisi yang harus dipenuhi.

Pada IF Tunggal, terdapat 1 kondisi yang harus dipenuhi sehingga akan memiliki 2 hasil, yaitu hasil jika kondisi terpenuhi dan hasil jika kondisi tidak terpenuhi.

IF Tungggal memiliki syntax sebagai berikut (kita gunakan Excel sebagai acuan)

=IF(Kondisi,Hasil Sesuai Kondisi, Hasil Tidak Sesuai Kondisi)

Pada IF Ganda/Majemuk, terdapat lebih dari 1 kondisi yang harus dipenuhi sehingga akan memiliki lebih dari 2 hasil, yaitu hasil jika kondisi pertama, kedua dan seterusnya terpenuhi dan hasil jika semua kondisi tidak terpenuhi.

IF Ganda/Majemuk memiliki syntax sebagai berikut (kita gunakan Excel sebagai acuan)

=IF(Kondisi1,Hasil Sesuai Kondisi 1,IF(Kondisi2,Hasil Sesuai Kondisi 2, Hasil Tidak Sesuai Kondisi 2))

Mapel : TIK

Kategori : Operasi Dasar pada Sistem Operasi

Kata kunci : IF Tunggal, IF Ganda, IF Majemuk

36. Bisakah fungsi IF digabungkan dengan fungsi logika (AND, NOT atau OR)? Bila “Bisa” tuliskan contohnya, dan bila “Tidak” jelaskan alasannya!


Bisa jika dalam soal/data ada yg berhubungan dengan "jika", lalu ada "dan" atau "atau". Karena terkadang suatu data diolah dengan cara menentukan logika (IF) yang isinya merupakan gabungan data (AND) atau salah satu dari gabungan data tsb (OR) (Kalau NOT belum pernah dicoba)

Misal:
Nilai Akhir (ada di kolom A):
Sel A2 isinya 60
Sel A3 isinya 80
Nilai Kehadiran (ada di kolom B):
Sel B2 isinya 60
Sel B3 isinya 95
Ditanya: Keterangan Lulus/Tidak Lulus, jika:
- Jika nilai akhir dibawah 75, dan nilai kehadiran dibawah 80, maka Tidak Lulus
- Jika nilai akhir diatas 75, dan nilai kehadiran diatas 80, maka Lulus
Maka rumusnya (gabungan IF dengan AND) (ditulis di sel C2. Patokan nilai akhir A2 & patokan nilai kehadiran B2):
=IF(AND(A2<=75;B2<=80);"TIDAK LULUS";"LULUS")
Maka, hasilnya Tidak Lulus.
Kalau kurang dimengerti, bisa lihat hasilnya di lampiran/gbr

38. Jelaskan kegunaan fungsi logika dari: IF; AND; OR; dan NOT(masing-masing contohnya)


Logika IF : memberikan pernyataan suatu kondisi.
Contoh : "=IF(B5>75,"Lulus",Gagal")"

Logika NOT : membalikkan suatu pernyataan dengan output TRUE atau FALSE.
Contoh = "=NOT(50 > 70)" Maka akan bernilai TRUE(kebalikan dari False)

Logika AND : memberikan nilai TRUE jika semua inputannya bernilai benar. Jika salah satu inputannya salah, maka menghasilkan False
Contoh = B5 = 80 maka "=AND(B5>1, B5<85)"

Logika OR : Memberikan nilai TRUE jika salah satu inputannya bernilai benar. Jika kedua inputan salah, maka bernilai False. 
Contoh = B1 = 25 dan B2 = 10 "=OR(B1>10, B2>15)"
