2. tombol bertanda logo windows pada keyboard adalah tombol shortcut yang berfungsi untuk...


Tombol bernama Windows Logo Key itu ternyata mempunyai fungsi yang bisa memudahkan pengguna untuk mengakses menu-menu lain yang ada di komputer bila dipadukan dengan tombol lain (Shortcut). Tapi shortcut yang ada hanya akan berjalan apabila komputermu menggunakan sistem operasi Windows.

Windows Logo Key berada di antara tombol Alt dan Fn/ Ctrl yang berada di sebelah kiri bawah bagian keyboard. Nah, berikut beberapa fungsi shortcut yang bisa digunakan dengan bantuan Windows Logo Key. 

1. Windows: untuk menampilkan Start menu.

2. Windows Logo + D: untuk menampilkan desktop, meminimasi aplikasi windows yang terbuka atau mengembalikan (restore) semua aplikasi windows yang terbuka.

3. Windows Logo + E: untuk membuka Windows Explorer.

4. Windows Logo + F: untuk menampilkan menu pencarian file (Search Results).

5. Windows Logo + Ctrl + F: untuk menampilkan menu pencarian "Search Results - Computer".

6. Windows Logo + F1: untuk membuka menu "Help and Support Center".

7. Windows Logo + R: untuk membuka kotak dialog "Run".

8. Windows Logo + break: untuk membuka kotak dialog "System Properties".

9. Windows Logo + M: untuk meminimasi semua aplikasi windows yang sedang dibuka atau dengan kata lain kembali ke desktop.

10. Windows Logo + shift + M: kebalikan dari Windows Logo + M, yakni untuk meng-undo semua windows yang diminimasi.

11. Windows Logo + L: untuk mengunci workstation

12. Windows Logo + U: untuk membuka menu "Utility Manager". 

13. Windows + tab, layar komputer akan terganti menjadi 3D.

14. Windows + R lalu ketik psr.exe, untuk  memulai rekaman.

15. Tekan windows +R dan masukan osk, di layar monitor akan muncul keyboard virtual

16. Windows ditambah "+" atau "-" , maka kaca pembesar akan muncul di layar, dan kita bisa memperbesar atau memperkecil gambar di layar sesuai keperluan.

Itulah beberapa fungsi tombol Windows Logo Key yang bisa dipadukan dengan beberapa tombol lain. Selamat mencoba!

4. Tombol yang berfungsi mengubah fungsi tombol penunjuk menjadi tombol numerik adalah....


Jawaban:

Tombol NumLock

Penjelasan:

Perhatikan contoh gambar keyboard berikut ini (khususnya di bagian yg ditunjukkan tanda panah kuning):

- bila tombol NumLock dalam kondisi aktif, maka tombol penunjuk berubah menjadi tombol numerik.

- bila tombol NumLock dalam kondisi nonaktif, maka tombol numerik berubah menjadi tombol penunjuk.

Contohnya dapat diamati dengan seksama:

- tombol numeric 8 dapat berubah jadi tombol penunjuk panah atas

- tombol numeric 6 dapat berubah jadi tombol penunjuk panah kanan

- tombol numeric 4 dapat berubah jadi tombol penunjuk panah kiri

- tombol numeric 2 dapat berubah jadi tombol penunjuk panah bawah

8. tombol pada keyboard yang berfungsi untuk mengaktifkan dan menonaktifkan referensi/tanda absolute($) yaitu​


Pada keyboard Microsoft Excel tombol yang digunakan untuk mengaktifkan dan menonaktifkan referensi absolut adalah tombol "F4". Saat kursor di posisi sel tertentu, tekan tombol F4 untuk mengaktifkan atau menonaktifkan referensi absolut untuk sel tersebut. Setiap kali tombol F4 ditekan, referensi akan berganti antara absolut, relatif, dan kombinasi dari keduanya. Selain itu juga, kamu bisa menambahkan tanda $ (dollar) sebelum nomor kolom dan baris untuk menjadikan referensi sel sebagai absolut

35. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.
