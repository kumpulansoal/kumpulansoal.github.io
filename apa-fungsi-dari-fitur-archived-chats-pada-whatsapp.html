KOMPONEN-KOMPONEN SCRATCH

Sebelum kita membuat aplikasi sederhana, pada tahap awal ini kita akan membahas bagian dari program scratch yang akan kita gunakan. Pengenalan fungsi-fungsi pada scratch sangat diperlukan agar dalam pembuatan animasi atau game menjadi lebih mudah

BAGIAN SCRATCH FUNGSI

LANGUAGE Untuk mengubah bahasa

SAVE Untuk menyimpan project

SHARE Untuk berbagi file

BLOCK PALETTE Blok untuk memprogram sprite

SCRIPTS AREA Tempat untuk meletakkan blok. Drag blok kedalam, tempelkan bersama script lainnya

SPRITE LIST Tempat semua sprite yang digunakan. Klik untuk memilih dan mengedit sebuah sprite

NEW SPRITE BUTTONS Membuat sebuah karakter baru atau objek untuk project kita.

STAGE Tempat untuk melihat cerita, animasi atau game yang telah dibuat

STOP SIGN Untuk menghentikan semua script

GREEN FLAG Untuk menjalankan script yang telah dibuat

PRESENTATION MODE Untuk menampilkan project kita

VIEW MODE Untuk mengubah stage menjadi besar atau kecil.

B. Scratch Block

Scratch block terdiri dari delapan kategori yang berbeda dan ditandai oleh delapan warna yang berbeda. Kategori dalam scratch meliputi Motion, Looks, Sound, Pen, Control, Sensing, Operators dan Variables.

11. apasaja fitur ²dlm menggunakan wa dan apa fungsi fitur trsb​


Jawaban:

Ada banyak fitur yang tersedia di WhatsApp, namun di antara yang paling populer dan umum adalah:

Pesan Teks: Fitur pesan teks memungkinkan pengguna untuk mengirim dan menerima pesan teks secara instan ke dan dari kontak mereka di WhatsApp.

Panggilan Suara dan Video: Fitur ini memungkinkan pengguna untuk melakukan panggilan suara dan video secara gratis ke kontak WhatsApp mereka dari seluruh dunia dengan koneksi internet yang stabil.

Status: Fitur ini memungkinkan pengguna untuk memposting foto, video, atau pesan teks yang menghilang dalam waktu 24 jam.

Berkirim pesan suara: Pengguna dapat merekam dan mengirim pesan suara instan ke kontak mereka di WhatsApp.

Berkirim Lokasi: Fitur ini memungkinkan pengguna untuk berbagi lokasi real-time mereka dengan kontak mereka di WhatsApp.

Berkirim dokumen dan file: Pengguna dapat mengirim dokumen dan file seperti PDF, dokumen Word, presentasi PowerPoint, dan file media seperti foto dan video ke kontak mereka di WhatsApp.

Grup: Fitur grup memungkinkan pengguna untuk membuat grup dan berkomunikasi dengan banyak orang secara bersamaan.

WhatsApp Web: Pengguna dapat mengakses WhatsApp mereka dari browser web di komputer desktop mereka.

Panggilan suara dan video grup: Fitur ini memungkinkan pengguna untuk melakukan panggilan suara dan video dengan lebih dari satu orang di dalam grup.

WhatsApp Business: Aplikasi ini dirancang khusus untuk pemilik bisnis kecil dan menengah untuk berkomunikasi dengan pelanggan mereka.

Fungsi dari fitur-fitur di atas bermacam-macam, namun pada dasarnya fitur-fitur tersebut digunakan untuk memudahkan pengguna dalam berkomunikasi dengan orang lain secara instan, mudah, dan gratis, serta memungkinkan pengguna untuk berbagi informasi, dokumen, dan file dengan mudah.

Antarmuka Aqua
Salah satu perbedaan besar dari Mac OS Klasik dan Mac OS X adalah penggunaan Aqua, sebuah antar muka grafis dengan elemen seperti air, pada rilis pertama Mac OS X. Setiap elemen window, teks, grafis, tombol, dll ditampilkan menggunakan teknologi antialiasing. ColorSync, tekonolgi yang telah dikembang sebelumnya, ditingkatkan dan dimasukkan ke dalam core drawing engine, sehingga dapat menyetarakan warna untuk multimedia dan percetakan. Selain itu, bayangan ditambahkan di sekitar window dan beberapa teks untuk memberikan efek kedalaman. Elemen antarmuka baru ditambahkan, seperti sheets (kotak dialog yang terikat pada window tertentu) dan drawers, yang dapat keluar dan memberikan opsi tambahan.

Penggunaan tepi yang membulat, warna translucent dan garis-garis, serupa dengan desain perangkat untuk iMac pertama, membawa lebih banyak tekstur dan warna dibandingkan dengan antarmuka "Platinum" pada Mac OS 9 dan Mac OS X Server. Bruze Tognazzini (yang mendirikan Grup Apple Human Interface) mengatakan bahwa antarmuka Aqua merupakan kemunduran dalam usability dibandingkan dengan antarmuka pada Mac OS awal. Pengembang pihak ketiga mulai memproduksi tema untuk sistem operasi lain agar menyerupai tampilan Aqua. Apple pernah melakukan tindakan hukum kepada pihak yang membuat atau mendistribusikan perangkat lunak dengan antarmuka yang menurut Apple menyontek desain mereka.

Appa terus melakukan perubahan tampilan dan desain macOS, terutama dengan perbaikan pada tampilan windows dan menu bar. Sejak 2012, Apple menjual banyak model dengan layar Retina (layar dengan resolusi tinggi), dan macOS memiliki dukungan luas untuk mendukung layar dengan resolusi tinggi. Menurut pengulas, dukungan Apple untuk resolusi tinggi lebih baik dibandingkan Windows.

Panduan antarmuka yang dipublikasikan Apple untuk macOS diikuti oleh banyak aplikasi, memberikan tampuilan dan shortcut yang konsisten. Selain itu, banyak layanan baru untuk aplikasi ditambahkan, seperti pemeriksa ejaan dan tata bahasa, special characters pallate, color picker, font chooser, dan dictionary; fitur ini tersedia untuk semua aplikasi berbasis Cocoa, menambah konsistensi.

Dukungan multibahasa
Terdapat 39 bahasa sistem tersedia untuk macOS pada saat pemasangan; bahas sistem digunakan pada seluruh sistem operai. Metode input untuk berbagai huruf dapat dipilih secara terpisah. Pembaruan terikini menambahkan dukungan untuk karakter bahasa Cina.

Metode pembaruan
macOS dapat diperbarui melalui panel Software Update di System Preferences atau menggunakan "softwareupdate" pada antarmuka baris. Hingga OS X 10.8 Mountain Lion, terdapat Software Update terpisah untuk melakukan hal ini. Pada Mountain Lion dan setelahnya, hal ini digabungkan dengan aplikasi Mac App Store, meski mekanisme pembaruan tidak berubah dan berbeda dengan mekanisme unduh untuk aplikasi pada App Store. Pada macOS 10.14 Mojave, fungi pembaruan dipindahkan lagi ke Software Update.

31. apa saja fitur-fitur yang ada di dalam corel draw? apa fungsi dari weld dan distorsi?


Tool Bar

Top Bar merupakan sekumpulan tombol shortcut yang memiliki fungsi untuk menjalankan perintah dengan cepat. ShortCut. Ini adalah bagian yang paling sering digunakan oleh desainer grafis dalam melakukan sesuatu. CorelDraw memberikan fitur dalam toolbar yang powerfull namun mudah sekali digunakan.

Menu Bar

Menubar berisi tombol-tombol text yang memiliki banyak sekali fungsi yang berbeda sesuai dengan namanya masing-masing. Desainer pun sering menggunakan fasilitas ini biasanya dalam memberikan suatu effect berhubungan dengan vector dan bitmap.

Property Bar

Property Bar berisi sekumpulan tombol yang isinya itu menyesuaikan dengan tools yang sedang kita pilih di dalam tool box. Untuk lebih jelasnya kita akan pelajari lebih detil dipertemuan yang akan datang.

Docker

Docker merupakan kotak dialog yang di dalamnya berisi berbagai tombol perintah untuk mengatur, memberi effect kepada sebuah objek maupun teks.

Rulers

Rulers merupakan alat ukur berupa penggaris dibagian atas dari drawing page, dan di bagian kanan drawing page, rullers digunakan sebagai acuan yang mampu menunjukkan ukuran suatu objek atau dokumen, serta tata letak suatu desain.

Status Bar

Status Bar terletak dibagian bagian bawah area program sebelah kanan. Status Bar berisi informasi mengenai objek (teks) seperti ukuran, bentuk, warna, outline, format gambar, jenis huruf beserta  ukurannya. Anda bisa melakukan perubahan terkait bentuk, warna, outline dan format gambar menggunakan status bar ini.

Drawing Page

Drawing Page adalah adalah sebuah area kerja utama di dalam Corel Draw di mana Anda dapat mendesain suatu objek di dalamnya dan merupakan area cetak suatu desain.

Color Palette

Color Palette bagian dari CorelDraw yang berisi koleksi warna di dalam sebuah bar dan secara default/bawaan terletak di sebelah kanan dokumen / program.

Di pertemuan yang akan datang, kita akan bahas satu persatu fitur dari toolbox disertai dengan contoh kasusnya. Semoga bermanfaat.
