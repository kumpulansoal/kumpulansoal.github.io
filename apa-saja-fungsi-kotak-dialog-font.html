1. apa saja fungsi kotak dialog font



FUNGSI MENU YANG TERDAPAT DALAM KOTAK DIALOG (FONT)
Font : mengajur jenis huruf yang digunakan dan mempunyai nama misalnya, arial, times, courier, dll.Font Style : merubah gaya tampilan huruf yang didalamnya terdapat :- Regular : normal- Italic : memberikan efek miring pada huruf- Bold : memberikan efek tebal pada huruf- Bold italic : memberikan efek tebal dan efek miring pada hurufSize : mengatur ukuran huruf yang didalamnya terdapat berbagai ukuran huruf, misalnya ialah 8, 9, 10, 11, dst.Font Color : memberikan warna pada huruf yang didalamnya terdapat berbagai macam warna, misalnya warna merah, kuning, hijau, dll.Underline Style : memberikan efek garis bawah pada huruf dan terdapat berbagai macam jenis garis bawah.Underline Color : memberikan warna pada garis bawah yang didalamnya terdapat berbagai macam warna, misalnya warna merah, kuning, hijau, dll.Effects : memberikan pencetakan khusus pada huruf

23. sebutkan beberapa efek yang terdapat pada kotak dialog font?


Beberapa efek dalam kotak dialog font (lihat gambar lampiran), saya urutkan dari atas kiri ya penjelasannya
1. Font: untuk mengatur jenis tulisan
2. Font size: untuk mengatur ukuran tulisan
3. Grow font: untuk membuat tulisan menjadi lebih besar
4. Shrink font: untuk membuat tulisan menjadi lebih kecil
5. Change case: mengganti tulisan apakah menjadi kapital semua, atau kecil semua, atau kapital di bagian huruf depan
6. Clear formatting: menghapus format teks sehingga teks menjadi normal
7. Bold: menebalkan teks
8. Italic: memiringkan teks
9. Underline: menggarisbawahi teks
10. Strikethrough: mencoret teks
11. Subscipt dan superscript: membuat pangkat di atas atau di bawah
12. Text effect: membuat efek tulisan menjadi berbayang, terefleksi, atau bersinar
13. Text highlight color: mewarnai teks (menghighlight teks)
14. Font color: mengganti warna teks

27. Apa fungsi preview yang terdapat pada submenu font dalam kotak dialog format cells??


Fungsi submenu font dalam format cells adalah:
1. Mengubah font.
2. Mengubah font style menjadi italic,bold,bold italic, maupun regular.
3. Mengubah ukuran font.
4. Memberi efek pada teks, seperti strikethrough,superscript,subscript, memberi underline dan
5. Mengubah warna.
