Materi : Satuan dan Bilangan

A = { x | 201 ≤ x ≤ 600 }

A = { 201, 202, ..., 599, 600 }

n(A) = 400 bilangan

Dibagi habis 4

Ket : [ ] = pembulatan kebawah

Misal : 150,9 => 150

[ 600/4 ] = 150

[ 201/4 ] = 50,25 => 50

Maka selisihnya adalah 100 bilangan

Dibagi habis 7

[ 600/7 ] = 85,714... => 85

[ 201/7 ] = 28,714 => 28

Maka selisihnya adalah 57 bilangan

Kesimpulan

Dibagi habis 4 adalah 100 bilangan

Dibagi habis 7 adalah 57 bilangan

Maka totalnya = 157 bilangan

yang tidak dapat dibagi habis oleh 4 dan 7 di himpunan 201 ≤ x ≤ 600 adalah 243 bilangan

9. tentukan dP/dQ dan dQ/dP ketika 30P+30-81 = 0​


Jawaban:

Diketahui :

Qd = 4 - P

Qd = 3

Ditanya :

Ed = ?

Jawab :

jika kita mengetahui fungsi permintaan dalam bentuk Qd = a - bP, maka kita dapat menghitung koefisien elastisitas permintaan dengan menggunakan rumus :

Ed = Q' × P/Q

ket :

Ed = Koefisien elastisitas permintaan

P = tingkat harga

Q = kuantitas pada tingkat harga tertentu

Q’ = turunan pertama dari fungsi

karena kita belum mengetahui harganya (P) maka kita cari dahulu dengan mensubtitusikan Q yang telah kita ketahui ke dalam fungsi permintaan :

Qd = 4- P

3 = 4 - P

P = 4 - 3

P = 1

kemudian kita cari juga turunan pertama fungsi permintaannya :

Q = 4 - P

Q' = -1

setelah mengetahui harganya (P), barulah kita dapat menghitung koefisien elastisitas permintaannya :

Ed = Q' × P/Q

Ed = -1 × 1/3

Ed = -0,34

gunakan nilai mutlak, Jadi, Ed = 0,34

Pembahasan

Pengertian Elastisitas Permintaan

Elastisitas permintaan adalah sebuah ukuran seberapa besar derajat kepekaan permintaan terhadap perubahan harga.

Jenis-jenis Elastisitas Permintaan

Elastisitas permintaan ada 5 jenis, yaitu:

Permintaan elastis, yaitu apabila koefisien elastisitasnya lebih besar dari 1 (Ed > 1)

Permintaan inelastis, yaitu apabila koefisien elastisitasnya lebih kecil dari 1 (Ed < 1)

Permintaan elastis uniter, yaitu apabila koefisien elastisitasnya sama dengan 1 (Ed = 1)

Permintaan elastis sempurna, yaitu apabila koefisien elastisitasnya sama dengan tidak terhingga (Ed = ∞)

Permintaan inelastis sempurna, yaitu apabila koefisien elastisitasnya sama dengan nol (Ed = 0)

Menghitung Koefisien Elastisitas Permintaan

rumus untuk menghitung koefisien elastisitas permintaan adalah :

Ed = ∆Q/∆P×P/Q

Ket :

Ed : koefisien elastisitas permintaan

ΔP : perubahan harga (selisih P1 dan P2)

ΔQ : perubahan kuantitas (selisih Q1 dan Q2)

P : harga awal

Q : kuantiatas awal yang diminta

Jika diketahui fungsi permintaan, maka koefisien elastisitasnya di hitung dengan cara :

Jika diketahui fungsi permintaan Qd=a –bP, maka rumusnya: Ed=Q'×P/Q

Jika diketahui fungsi permintaan Pd=a –bQ, maka rumusnya: Ed=1/P'×P/Q

Ket :

Ed = Koefisien elastisitas permintaan

P = tingkat harga

Q = kuantitas pada tingkat harga tertentu

P’ = turunan pertama dari fungsi

Q’ = turunan pertama dari fungsi

32. 5,9,7,…201 tentukan banyak suku jika suku terakhir adalah 201


Barisan Aritmatika

Runus suku ke-n

Un = a + (n - 1)b

a = U1 = suku pertama

b = beda = U2 - U1

soal diperbaiki

5, 7, 9, ... , 201

a = 5

b = 7 - 5 = 2

Un = 201

a + (n - 1)b = 201

5 + 2(n - 1) = 201

5 + 2n - 2 = 201

2n + 3 = 201

2n = 201 - 3

2n = 198

n = 198/2

n = 99

Banyak suku (n) jika suku terakhirnya 201,

adalah 99 suku.

38. Biaya total (9Q² - 18Q + 12) dQ?


Diketahui :

TC = Q³ - 14Q² + 69Q + 128

Ditanya :

a. Fungsi AC ?

b. Fungsi AVC?

c. Fungsi MC ?

Jawab :

a. Fungsi AC ?

AC = TC/Q

AC = Q³ - 14Q² + 69Q + 128

AC = Q² -14Q + 69 + 128/Q

b. Fungsi AVC ?

AVC = VC/Q

AVC = Q³ - 14Q² - 69Q / Q

AVC = Q² - 14Q - 69

c. Fungsi MC ?

MC = TC' (turunan pertama fungsi TC)

TC = Q³ - 14Q² + 69Q + 128

TC' = 3Q² - 28Q + 69 = MC

Pembahasan

Macam-Macam Biaya

1. Biaya Tetap (Fixed Cost) adalah biaya yang dikeluarkan untuk memproduksi sejumlah barang. Jumlah biayanya selalu tetap dan tidak berubah berapapun jumlah barang yang diproduksi.

2. Biaya Variabel (Variable Cost) adalah biaya yang dikeluarkan untuk memproduksi sejumlah barang. Jumlah biayanya selalu berubah secara konstan sesuai jumlah barang yang diproduksi.

3. Biaya Total (Total Cost) adalah total keseluruhan biaya yang dikeluarkan untuk memproduksi sejumlah barang (output) baik biaya tetap maupun biaya variabel.

rumus : TC = FC + VC

4. Biaya Rata-Rata (Average Cost) adalah biaya yang dikeluarkan setiap memproduksi satu unit barang (output).

rumus : AC = TC/Q

5. Biaya Marjinal (Marginal Cost) adalah tambahan biaya yang dikeluarkan setiap memproduksi satu unit tambahan.

rumus : MC = ΔTC/ΔQ

6. Biaya Tetap Rata-Rata (Average Fixed Cost) adalah biaya tetap yang dikeluarkan setiap memproduksi satu unit barang (output).

rumus : AFC = FC/Q

7. Biaya Variabel Rata-Rata (Average Variable Cost) adalah biaya variabel yang dikeluarkan setiap memproduksi satu unit barang (output).

rumus : AVC = VC/Q
