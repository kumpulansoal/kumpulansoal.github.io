Penjelasan:

Tombol Enter

Tombol Enter merupakan tombol yang penting pada keyboard dan memiliki beberapa fungsi. Tombol keyboard ini dapat digunakan untuk memindahkan kursor ke  baris di bawahnya saat  mengetik. Fungsi lainnya adalah saat sebagai perintah saat  menyetujui suatu program yang ada pada komputer atau laptop.

Fungi lain tombol Enter adalah untuk memulai baris, atau paragraf baru. Bisa dipakai baik dalam dokumen yg memakai tulisan atau bahkan gambar.

Ketika kita metekan tombol Enter, maka otomatis kursor akan berpindah pada baris berikutnya dalam kertas kerja. Karena itu, ketika membuat sebuah tulisan yg hanya terdiri dari kalimat atau kata-kata saja yg harus ada disetiap baris, maka tombol Enter ini akan sangat sering dipakai dalam pekejaan tsb.

Jawaban singkat:

b. =Min(Blok angka) enter

Penjelasan panjang:

Misalkan diterapkan di Ms. Excel, beberapa fungsi yg ada antara lain:

Max: untuk menampilkan data angka tertinggi

Min: untuk menampilkan data angka terendah

Sum: untuk menjumlahkan semua data angka

Average: untuk mencari nilai rata² data angka

Berdasarkan soal di atas, maka fungsi yg tepat ialah Min karena selanjutnya akan menampilkan angka terendah dari angka² yg terblok (yaitu 28)
