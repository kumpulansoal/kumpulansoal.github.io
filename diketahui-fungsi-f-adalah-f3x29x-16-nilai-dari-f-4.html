10. 19. Perhatikan gambar berikut. Y F₂ = 130 N F3x Bosinur. 45° Fry: 180 CCJ 45 F₁ = 30√/2 N F₁ = 30√/2 N X Resultan ketiga gaya pada gambar di atas adalah .... A. 125 N B. 100 N C. 70 N D. 50 N E. 25 N​


Jawaban:

217.45 N.

Penjelasan:

kita perlu menghitung komponen horizontal dan vertikal dari setiap gaya.

Diberikan:

F₁ = 30√/2 N

F₂ = 130 N

F₃ = 180 N

Komponen horizontal F₁ (F₁x) = F₁ * cos(45°) = 30√/2 * cos(45°) = 30 N

Komponen vertikal F₁ (F₁y) = F₁ * sin(45°) = 30√/2 * sin(45°) = 30 N

Komponen horizontal F₂ (F₂x) = 0 (berarah tegak lurus ke arah horizontal)

Komponen vertikal F₂ (F₂y) = F₂ = 130 N

Komponen horizontal F₃ (F₃x) = F₃ * cos(45°) = 180 * cos(45°) = 180√/2 N

Komponen vertikal F₃ (F₃y) = F₃ * sin(45°) = 180 * sin(45°) = 180√/2 N

Selanjutnya, kita dapat menjumlahkan semua komponen horizontal dan komponen vertikal:

Komponen horizontal total = F₁x + F₂x + F₃x = 30 N + 0 N + 180√/2 N = 30 N + 180√/2 N

Komponen vertikal total = F₁y + F₂y + F₃y = 30 N + 130 N + 180√/2 N = 30 N + 130 N + 180√/2 N

Untuk mencari resultan (R) dari kedua komponen tersebut, kita terlebih dahulu menggunakan rumus pythagoras:

R = √[(komponen horizontal total)^2 + (komponen vertikal total)^2]

R = √[(30 N + 180√/2 N)^2 + (30 N + 130 N + 180√/2 N)^2]

Setelah melakukan perhitungan, resultan dari ketiga gaya pada gambar di atas adalah sekitar 217.45 N.

FUNGSI INVERS KOMPOSISI

Menentukan nilai dari (fogoh)(x)

= (g o h)(x)

= g(h(x))

= 11 - 3x

= (f o (g(h(x)))

= [tex] \frac{(-3x + 11 + 2}{33 - 9x - 4} [/tex]

= [tex] \frac{(-3x + 13)}{-9x + 29} [/tex]

Nilai (f o g o h)(x) = [tex] \frac{(-3x + 13)}{-9x + 29} [/tex]

Misalkan; (f o g o h)(x) = y

(f o g o h)(x) = [tex] \frac{(-3x + 13)}{-9x + 29} [/tex]

y = [tex] \frac{(-3x + 13)}{-9x + 29} [/tex]

y(-9x + 29) = -3x + 13

-9xy + 29y = -3x + 13

-9xy + 3x = 13 - 29y

x(-9y + 3) = (13 - 29y)

x = [tex] \frac{13 - 29y}{-9y + 3} [/tex]

x = [tex] \frac{-29y + 13}{-9y + 3} [/tex]
