7. 24. Shortcut yang digunakan untuk menutup file database yang sudah dijalankan, pada aplikasi pengolah data adalah .... A. tekan tombol Alt + V B. tekan tombol Alt + F4 C. tekan tombol Alt + alt + del D. tekan tombol Alt + del E. tekan tombol Alt +


Shortcut yang digunakan untuk menutup file database yang sudah dijalankan pada aplikasi pengolah kata adalah tombol Alt+F4.

Pembahasan

Shortcut adalah kombinasi dari beberapa tombol yang terdapat di keyboard untuk memberi suatu perintah kepada komputer untuk melakukan sesuatu. Shortcut diciptakan untuk memudahkan dan mempercepat pekerjaan manusia dalam menggunakan komputer. Nah, yang akan dibahas kali ini adalah shortcut pada aplikasi pengolah kata (Ms.Office word).

Macam macam shortcut pada aplikasi pengolah kata dan pembahasannya.

1. Ctrl+A

Shortcut ini digunakan untuk menandai seluruh teks dan gambar yang ada di layar, jika cursor berada di tabel, maka shortcut ini akan menandai seluruh teks yang ada di dalam tabel. Huruf "A" disini memiliki arti "All"

2. Ctrl+B

Ctrl+B adalah shortcut yang digunakan untuk menebalkan tulisan. Shortcut ini merupakan salah satu shortcut terpenting di aplikasi pengolah kata. Huruf "B" memiliki arti "Bold"

3. Ctrl+C

Ctrl+C adalah shortcut yang digunakan untuk meng-copy (menyalin) teks dan gambar yang sudah ditandai sebelumnya. Huruf "C" memiliki arti "copy"

4. Ctrl+D

Ctrl+D adalah shortcut yang berfungsi menghapus teks dan gambar yang telah ditandai sebelumnya. Huruf "D" memiliki arti "Delete"

5. Ctrl+V

Ctrl+V berfungsi untuk menempel / paste seluruh teks dan gambar yang telah di salin (copy) atau di potong (cut) sebelumnya.

6. Ctrl+Z

Ctrl+Z memiliki nama lain undo yang digunakan untuk membatalkan perintah yang terakhir dilakukan oleh pengguna. shortcut ini merupakan shortcut yang paling banyak digunakan oleh pengguna.

7. Alt+F4

Shortcut ini digunakan untuk menutup file database yang sudah dijalankan. Jadi jika kita menekan shortcut ini, maka aplikasi pengolah kata akan otomatis tertutup.

Jadi, jawaban yang paling tepat adalah B. Alt+F4.

13. Mengaktifkan Menu Help dengan cara... a. ALT + H b. ALT + E c. ALT + L d. ALT + P


Jawaban:

A. Alt + H

Penjelasan:

Agar anda paham, silakan terapkan instruksi berikut :

1. Buka program Ms. Office. Misalnya Ms. Word. Pastikan berada di Home/Beranda.

2. Tekan tombol Alt + H di keyboard

3. Jika benar, akan muncul semua kombinasi hurup di menu, submenu maupun tab.

Jika hanya menekan Alt, maka kombinasi hurup yang muncul lebih sedikit (hanya poin penting saja). Dengan kombinasi Alt + H, semua menu/fitud yang didukung shortcut ditampilkan.

Karena semua kombinasi hurup muncul, artinya sistem telah membantu anda. Inilah yang dimaksud Help.

Jika anda melanjutkan, kombinasi berikut berfungsi :

Alt + H = kembali ke Home/Beranda jika sebelumnya berada di tab lain

Alt + E = membukan kunci akses shortcut. Misal Alt + E + B + 1 = Bold (menebalkan huruf tulisan)

Alt + L = membuka kunci akses shortcut. Misal alt + L + B + 1 (menebalkan huruf tulisan)

Alt + P = menampilkan menu Page Layout / Tata Letak

Jadi menampilkan menu Help adalah Alt + H
