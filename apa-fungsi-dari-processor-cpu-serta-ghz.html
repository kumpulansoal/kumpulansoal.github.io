4. Jelaskan Central Processor Unit (CPU)


Unit Pemroses Sentral (UPS) (bahasa Inggris:Central Processing Unit/Processor; CPU), merujuk kepada perangkat keras komputeryang memahami dan melaksanakan perintahdan data dari perangkat lunak. Istilah lain, pemroses/prosesor (processor), sering digunakan untuk menyebut CPU. Adapun mikroprosesor adalah CPU yang diproduksi dalam sirkuit terpadu, seringkali dalam sebuah paket sirkuit terpadu-tunggal. Sejak pertengahan tahun 1970-an, mikroprosesorsirkuit terpadu-tunggal ini telah umum digunakan dan menjadi aspek penting dalam penerapan CPU. CPU Merupakan bagian utama dari komputer karena processor berfungsi untuk mengatur semua aktifitas yang ada pada komputer. Kecepatan eksekusi processor tergantung apalagi pada frekuensinya, satuan adalah MHz (MegaHertz) atau GHz (1 GigaHertz = 1000 MegaHertz).

5. Sebutkan kegunaan CPU, motherboard,keyboard, hardisk processor​


Pertanyaan:

Sebutkan kegunaan CPU, motherboard,keyboard, hardisk processor​

Jawaban & Penjelasan:

Kegunaan 9 Perangkat keras : processor, motherboard, ram, harddisk,  power supply,​  Fan cooler, GPU, dan Keyboard

1. Prosessor (CPU)

Kegunaan Prosessor adalah membagi pekerjaan pengolahan data kepada seluruh komponen komputer yang dilakukan dengan speed atau kecepatan yang begitu tinggi. Itulah sebabnya suhu pada processor sangat panas, dan di samping itu ada perangkat kipas pendingin yang berguna untuk menurunkan suhu processor sehingga tidak terlalu panas.

2. RAM

Kegunaan RAM adalah sebagai menyimpan file primer atau sebagai memori utama saat software sedang berjalan.

3. Storage

Kegunaan Storage adalah menyimpan file secara permanen, menyimpan file besar, dll

4. Motherboard

         Tugas motherboar adalah menghubungkan beberapa komponen seperti; BIOS (Basic Input Output System), Chipset (Pengatur koneksi input output), RAM (memory penyimpanan data), VGA Card (memori penyimpanan data grafis), Prosesor, harddisk dan SSD.

5. Keyboard

Kegunaan keyboard adalah sebagai alat input data berupa huruf, angka, atau simbol.

6. Solid State Drive (SSD)

         Kegunaan SSD untuk membantu meningkatkan kecepatan Booting pada Laptop atau PC kamu. Meskipun harganya di pasaran relatif mahal, tetapi, dari segi kecepatan , SSD jauh melampaui dari kecepatan yang di tawarkan oleh HDD.

7.  Graphic Processing Unit (GPU)

          Kegunaan GPU adalah membantu kerja Prosessor, untuk melakukan render pada gambar, animasi serta video yang akan ditampilkan pada layar laptop atau desktop Anda.

8. Fan Cooler

           Kegunaan Fan Cooler adalah sebagai alat penjaga prosesor agar suhu tetap setabil, namun biasanya fungsi ini terganggu karena terhalang oleh banyaknya debu yang menempel pada bagian baling-baling kipas. Semakin tebal debu yang menghinggapi, maka dapat dipastikan akan memperlambat kinerja sistem.

9. Power Supply

Kegunaan Power Supply adalah sebagai alat yang mampu memberikan sebuah suplai arus listrik kepada semua komponen komputer yang sudah terpasang dengan baik, dimana arus listrik yang dihasilkan merupakan arus AC dan selanjutnya akan dirubah menjadi arus DC.

14. Kenapa Processor (CPU) komputer mempunyai kecepatan yang jauh lebih tinggi daripada RAM ?


Jika processor membutuhkan suatu data, pertama - tama ia akan mencarinya pada cache.
Jika data ditemukan, processor akan langsung membacanya dengan delay yg sangat kecil.
Tetapi, jika data yg dicari tdk ditemukan, processor akan mencarinya pada RAM yg kecepatannya lebih rendah.
Pada umum nya cache dpt menyediakan data yg dibutuhkan oleh processor sehingga pengaruh kerja RAM yg dpt dikurangi.
Dengan cara ini maka memory bandwidth akan naik & kerja processor menjadi lebih efisien.
Selamat sore,
Alasan Processore lebih cepat dari pada RAM adalah Processore memiliki ruang pentimpanan sementara tersendiri yang disrbut chace, chace pada processore sekarang adalah L3Chace yang memiliki penimpanan lebih banyak mrncapai 25MB lebih dan kecepatan yang lebih baik dari pada versi chace yang terdahulu yakni L2Chace.

Cache berfungsi sebagai tempat penyimpanan sementara untuk data atau instruksi yang diperlukan oleh processor. Secara gampangnya, cache berfungsi untuk mempercepat akses data pada komputer karena cache menyimpan data/informasi yang telah diakses oleh suatu buffer, sehingga meringankan kerja processor.
