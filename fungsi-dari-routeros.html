1. Tampilan RouterOS akan berupa ...​


Jawaban:

Tampilan RouterOS akan berupa CLI (Command Line Interface) atau Antarmuka Baris Perintah.

Penjelasan:

RouterOS merupakan suatu sistem operasi jaringan keluaran Mikrotik. Biasanya, dipanggil lengkap Mikrotik RouterOS. RouterOS dapat merubah Desktop menjadi sebuah server atau router yang diinginkan. RouterOS dapat dipasang dengan mengikuti Instalasi Mikrotik RouterOS pada desktop kita dan akan berupa tampilan Antarmuka Baris Perintah (CLI).

1. Pernyataan yang salah tentang konfigurasi switch di atas adalah: B. no. 4 dan 5, pengguna memasuki fastethernet0/1 dan mengubah port 2 menjadi mode access. Selain itu, kata "switcport" pada nomor 5 juga merupakan kesalahan penulisan yang seharusnya adalah "switchport".

3. Berdasarkan penggunaan perangkat yang diberikan, yaitu 2 unit PC/laptop untuk ruangan Direktur, 4 unit PC/laptop untuk bagian keuangan, dan 4 unit PC/laptop untuk ruang kepegawaian, maka diperlukan 3 VLAN untuk memisahkan perangkat-perangkat tersebut secara logika. 4. Aplikasi mode GUI yang digunakan untuk meremote dan mengkonfigurasi MikroTik RouterOS adalah: B. Winbox. Winbox adalah aplikasi berbasis GUI yang dikembangkan khusus oleh MikroTik untuk mengkonfigurasi perangkat mereka.

5. Username default pada MikroTik adalah: A. user. Ketika pertama kali mengakses perangkat MikroTik, username default yang digunakan adalah "user", sedangkan password defaultnya kosong.

9. In mikrotik routeros, layer-3 communication between 2 hosts can be achieved by using an address subnet of ?


From mikrotik routers layer-3 communication between 2 hosts can achived using address subnet /29 and /30.

Pembahasan:

RouterOS adalah sistem operasi untuk perangkat keras RouterBOARD. Ini memiliki semua fitur yang Anda butuhkan untuk perutean ISP, firewall, manajemen bandwidth, titik akses nirkabel, tautan backhaul, gateway hotspot, server VPN, dan banyak lagi. Instalasi cepat dan mudah serta antarmuka yang ramah pengguna. Semua router telah dikonfigurasikan sebelumnya dengan alamat IP pabrik 192.168.1.1. 88.1/24 pada port ether1. Nama pengguna default adalah admin tanpa kata sandi. Netinstall adalah alat yang dirancang untuk sistem operasi Windows yang menginstal ulang perangkat MikroTik yang menjalankan RouterOS (kecuali perangkat selain MikroTik x86). Netinstall memformat ulang hard drive perangkat dan menyalin file baru ke hard drive sistem. Ini dapat memecahkan beberapa masalah jika perangkat tidak berfungsi dengan baik.
