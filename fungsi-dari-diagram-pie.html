9. Sebuah Pie Mangga Memiliki Fungsi:f(x) = -x² + 4x + 5, Tentukan Titik Tertinggi Yang Dapat Dicapai Pie Mangga Tersebut!#PieMangga​


Jawaban:

titik tertinggi yang dapat dicapai = (2 , 9)

Penjelasan dengan langkah-langkah:

f(x) = -x² + 4x + 5

a = -1 , b = 4 , c = 5

titik tertinggi = titik puncak (Xp,Yp)

Xp = - b/2a

Xp = - 4/2(-1)

Xp = -4/(-2)

Xp = 2

Yp = f(Xp)

Yp = f(2)

Yp = -2² + 4 × 2 + 5

Yp = -4 + 8 + 5

Yp = 4 + 5

Yp = 9

Titik puncak = (2,9)

Jawaban:

(2,9)

Penjelasan dengan langkah-langkah:

ada dua cara menentukan titik maksimum dari sebuah fungsi kuadrat.

Cara pertama dengan menggunakan rumus

[tex]x_{maks}=\frac{-b}{2a} [/tex]

[tex]y_{maks}=\frac{D}{-4a} = \frac{ {b}^{2} - 4ac }{-4a}[/tex]

untuk suatu fungsi kuadrat y=ax²+bx+c

pada soal di atas dapat diketahui bahwa a=-1, b=4, dan c= 5, sehingga diperoleh

[tex]x_{maks}=\frac{-b}{2a} = \frac{-4 }{2(-1)} = \frac{-4}{-2} = 2[/tex]

[tex]y_{maks}=\frac{D}{-4a} = \frac{ {b}^{2} - 4ac }{-4a} = \frac{ {4}^{2} - 4( - 1)(5) }{ - 4( - 1)} = \frac{16 + 20}{4} = \frac{36}{4} = 9[/tex]

Cara kedua menggunakan turunan

nilai maksimum diperoleh saat y'=0, sehingga

y= -x²+4x+5

y'=-2x+4

-2x+4=0

-2x=-4

x=-4/-2

[tex]x_{maks}=2[/tex]

untuk mendapatkan y maks kita substitusi x=2 ke dalam persamaan y= -x²+4x+5

diperoleh

[tex]y_{maks}=-2²+4(2)+5=-4+8+5=9[/tex]

26. Bagaimana cara membuat diagram pie menggunakan nilai derajat??


Jawaban:

Untuk membuat sebuah diagram lingkaran yang telah diketahui datanya, maka kita harus dapat mengubah satuan masing-masing jumlah data yang telah diketahui nilainya. Jika dibuat dalam bentuk diagram lingkaran persen, maka kita harus mengubah datanya ke bentuk persen, sedangkan jika ingin dibuat dalam bentuk diagram lingkaran derajat, maka diubah menjadi satuan derajat.

Namun, untuk membuat diagram lingkaran bentuk persen atau derajat sebenarnya memiliki konsep yang sama. Dimana untuk menentukan besar sudut pada tiap-tiap ruas bagian lingkaran menggunakan rumus sebagai berikut:

Ruas sudut = (nilai data / total data) x 360º

Sementara itu, jika data yang telah diketahui akan diubah ke bentuk persen (%), maka rumus yang digunakan untuk mengubah nilai tiap-tiap data ke persen adalah sebagai berikut:

Nilai persen = (nilai data / total data) x 100%

Dari penjelasan tersebut, maka langkah-langkah untuk membuat diagram lingkaran dalam bentuk persen atau derajat adalah sebagai berikut:

Menghitung jumlah total data dari semua nilai dataMenghitung besar sudut pada masing-masing bagian data dengan rumus: (nilai data / total data) x 360ºJika nilai data akan dirubah ke bentuk persen, maka gunakan rumus: (nilai data / total data) x 100%Membuat lingkaran penuh dan membagi lingkaran menjadi beberapa ruas dengan besar sudut sesuai masing-masing data
