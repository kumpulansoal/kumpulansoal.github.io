There are over 2,000 computers connected to the mesh, The broadband internet connection is putting the mesh under great load. At present, the system seems to handle the load without any increase in latency or packet-loss. It is clear that scalability will become an issue if we continue to use a single radio channel. To solve this problem, new mesh routers with multiple radio channel support are being developed and tested in Dharamsala, with an emphasis on products that meet our technical require- ments and our economically viable. The initial results are very promising.The mesh network is based on recurring deployments of a hardware device, which is designed and built locally – known as the Himalayan-Mesh-Router332 Chapter 11: Case Studies(http://drupal.airjaldi.com/node/9). Ekstansi adalah data client Voip, misalnya komputer ke 1 ekstansinya 001 , komputer ke 2 ekstansinya 002 , komputer ke 3 ekstansinya 003 dan seterusnya.

Semua ekstensi diatas ditulis dalam sebuah file konfigurasi yaitu file extensions conf. Setiap ekstensi memiliki komponen dan tahapan perintah yang tersimpan didalam file tersebut. Didalam file tersebut ditulis dalam format exten = Extensions, priority, Command [ parameter ]. Untuk setiap extensions / ekstensi, dapat didefinisikan sekumpulan perintah. Komponen yang membangun tahapan perintah extension atau Command line adalah sebagai berikut.

1.] Ekstension adalah label dari extension, Dapat berupa sebuah string [ angka, huruf dan simbol yg diijinkan ] Atau pola yg harus di evaluasi secara dinamik untuk dengan banyak kemungkinan nomor telepon, Setiap command line yang menjadi bagian dari extension tertentu harus mempunyai label yg sama.

2.] Priority, biasanya berupa angka integer merupakan urutan dari perintah yg harus dijalankan dalam sebuah extension. Perintah pertama yang akan di jalankan harus dimulai dengan prioritas 1, jiia tidak ada prioritas 1, maka asterisk tidak akan menjalankan perintah extension setelah prioritas 1 dijalankan, Asterisk akan menambah prioritas ke prioritas 2 dan seterusnya, Tentunya jika tidak ada perintah yg menentukan prioritas mana yg selanjutnya harus dijalankan. 15. Berikan penjelasan bagaimanakah keterkaitan antara IP-PBX dengan Sistem Operasi Asterisk​


IP-PBX (Private Branch Exchange) adalah sistem telepon yang digunakan di dalam jaringan perusahaan atau organisasi. Ini menyediakan fitur seperti pemindahan panggilan, panggilan ganda, dan antar grup panggilan. IP-PBX menggunakan teknologi VoIP (Voice over Internet Protocol) untuk mengirimkan panggilan melalui jaringan internet.

Sistem Operasi Asterisk adalah sistem operasi open-source yang dapat digunakan untuk membuat sistem telepon IP-PBX. Asterisk menyediakan berbagai fitur telepon, seperti panggilan masuk dan keluar, panggilan conferens, voicemail, dan lainnya. Asterisk juga dapat digabungkan dengan perangkat lunak atau perangkat keras lainnya untuk menambahkan fitur tambahan, seperti integrasi dengan aplikasi CRM atau sistem antrian panggilan.

Keterkaitan antara IP-PBX dan Sistem Operasi Asterisk adalah bahwa IP-PBX menggunakan Sistem Operasi Asterisk sebagai dasarnya untuk menyediakan fitur-fitur telepon yang diinginkan pada jaringan perusahaan atau organisasi. Asterisk menyediakan kode sumber yang dapat dikustomisasi untuk menyesuaikan dengan kebutuhan khusus perusahaan atau organisasi.

19. penjelasan asterisk interface pstn?


Penjelasan asterisk interface pstn? Untuk bisa menjelaskan apa itu asterisk ingterface pstn, kita harus mengetahui terlebih dahulu  mengenai istilah tersebut. Istilah asterisk interface pstn berkaitan dengan teknologi VOIP. VOIP sendiri merupakan singkatan dari Voice over Internet Protocol, VOIP merupakan salah satu teknologi yang bisa kita gunakan untuk melakukan panggilan suara atau telepon menggunakan internet.

Setelah mengetahui apa itu VOIP, maka kita akan membahas apa itu PSTN, PSTN sendiri merupakan singkatan dari Public Switched Selephone Network, suatu jalur atau jaringan telepon menggunakan kabel.

Untuk pengertian asterisk sendiri adalah software yang digunakan untuk melakukan dan berbagai macam pengaturan terkait panggilan VOIP, agar bisa melakukan panggilan VOIP ke panggilan telepon biasa yang masih menggunakan kabel, kita memerlukan suatu perantara yaitu jadi asterisk interface pstn.

Demikian penjelasan mengenai asterisk interface pstn, suatu software yang bias kita gunakan untuk melaukan VOIP dan panggilan telepon biasa, selain software kita juga memerlukan perangkat keras seperti PABX agar kita bisa menggabungnkan antara VOIP dengan panggilan telepon biasa.

26. apa yang dimaksut dengan asterisk dengan interface ptsn


Pengertian 
Asterisk adalah software implementasi dari telepon private branch exchange (PBX) yang memungkinkan telepon yang tersambung untuk melakukan panggilan ke satu sama lain, dan untuk terhubung ke layanan telepon lainnya
Asterisk dirilis dengan model lisensi ganda, menggunakan GNU General Public License (GPL) sebagai lisensi perangkat lunak bebas dan lisensi perangkat lunak proprietary untuk mengizinkan pemegang lisensi untuk mendistribusikan proprietary , komponen sistem yang tidak dipublikasikan. 
Asterisk diciptakan pada tahun 1999 oleh Mark Spencer dari Digium . Awalnya dirancang untuk Linux, Asterisk berjalan pada berbagai sistem operasi, termasuk NetBSD, OpenBSD, FreeBSD, MacOS, dan Solaris, dan dapat diinstal dalam sistem embedded berdasarkan OpenWRT dan flash drive.
