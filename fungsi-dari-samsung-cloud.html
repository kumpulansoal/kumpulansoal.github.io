DEGREE OF COMPARISON

Degree of comparison adalah perbandingan antara satu noun dengan noun lain berdasarkan perubahan pada bagian adjective. Terdapat 3 jenis dalam degree of comparison, yaitu;

POSITIVE DEGREE   (Kedua noun sebanding)

Rumus :  (noun + as + adj/adv as + noun)

COMPARATIVE DEGREE (Membandingkan 2 noun)

Rumus : [Adjective + er + than] Rumus saat berakhiran y : [Adjective + ier + than] Rumus saat syllable lebih dari 2 : [More + Adjective + than] Irregular form : [contoh : bad --> worse + than]

SUPERLATIVE DEGREE (Membandingkan 1 noun terhadap banyak noun)

Rumus : [Adjective + est] Rumus saat berakhiran y : [Adjective + iest] Rumus saat syllable lebih dari 2 : [Most + Adjective] Irregular form : [contoh : bad --> worst]PENJELASAN

5. apa yang dimaksud cloud computing?dan jelaskan skema dan model bisnis cloud technology​


Cloud computing ialah gabungan pemanfaatan teknologi komputer dalam suatu jaringan pengembangan berbasis internet yang memiliki fungsi untuk menjalankan program maupun aplikasi melalui komputer-komputer yang terkoneksi pada waktu yang sama, akan tetapi tidak semua yang terkoneksi dengan internet menggunakan cloud computing.

Pembahasan  

Manfaat dari teknologi cloud computing :

Semua data tersimpan di server secara terpusat

Keamanan data

Fleksibilitas dan skabilitas yang tinggi

Investasi jangka panjang

Skema & model bisnis layanan cloud computing

IaaS (Infrastructure as a Service) : model komputasi awan dimana penyedia layanan memberikan jasa organisasi infrastruktur seperti server, hardware jaringan serta data stirage yang bisa digunakan konsumen.

PaaS (Platform as a Service) : model komputasi awan yang didesain agar memberikan developer sebuah platform yang memungkinkan developer tersebut untuk membuat dan mengembangkan aplikasi baru diatasnya.

Saas (Software as a Service) : model layanan komputasi awan yang ada dalam bentuk aplikasi yang diap untuk dipakai.

Jawaban:

Cloud storage adalah ruang penyimpanan di internet yang dapat digunakan oleh pengguna untuk menyimpan file-file miliknya. Teknologi ini memungkinkan kita sebagai pengguna menempatkan file-file yang ada di internet. Cara ini akan memungkinkan pengguna memiliki backup file dan dapat mengaksesnya jika diperlukan.

Cloud adalah gabungan pemanfaatan teknologi komputer ('komputasi') dan pengembangan berbasis Internet ('awan'). Cloud adalah metafora dari internet, sebagaimana awan yang sering digambarkan di diagram jaringan komputer. Sebagaimana awan dalam diagram jaringan komputer tersebut, Cloud dalam Cloud Computing juga merupakan abstraksi dari infrastruktur kompleks yang disembunyikannya.

Perbedaan:

• Cloud storage membutuhkan kapasitas penyimpanan yang lebih besar, sedangkan cloud computing membutuhkan keberadaan memori dan cpu yang lebih banyak untuk melakukan pemrosesan.

• Cloud computing lebih banyak digunakan pada kebutuhan tingkat lanjut, baik bisnis atau aplikasi tertentu. Sedangkan cloud storage penggunaan penyimpanannya lebih ke data pribadi, kelompok atau pekerjaan.

• Dalam hal pengelolaannya, cloud storage lebih sederhana untuk dilakukan. Sedangkan untuk cloud computing harus memiliki atribut tambahan dalam pengelolaannya seperti akses pengguna, common line interface dan lainnya.

Mengenal Apa itu Cloud Computing : Defenisi, Fungsi, dan Cara Kerja

Cloud Computing merupakan istilah yang sudah sering kita dengar. Namun, masih sedikit yang mengerti apa itu Definisi , Fungsi dan cara kerja Cloud Computing dalam kehidupan sehari – hari. Pada perusahaan – perusahaan maupun organisasi besar pengaruh Cloud Computing untuk memudahkan dalam berinteraksi dari satu komputer dengan komputer lainnya tanpa memakan banyak biaya.

Cloud Computing membuat semua pekerjaan serba praktis dan tidak perlu menggunakan Infrastruktur yang besar. Cloud Computing Bahkan dapat meminimalisir biaya pengeluaran dalam perusahaan. Tanpa kita sadari sehari – hari kita telah bertukar informasi dan data menggunakan Cloud Computing.

Dalam penerapannya, Cloud Computing hanya membutuhkan jaringan internet, hal ini dikarenakan Cloud Computing memiliki server yang terpusat. Berkut penjelasan mengenai Cloud Computing, Yuk kita simak penjelasan berikut ini !

Defisini Cloud Computing

Cloud Computing merupakan istilah dari Cloud diartikan sebagai internet dan Computing diartikan sebagai komputer. Definisi dari Cloud Computing adalah sebuah proses pengolahan daya komputasi melalui jaringan internet yang memiliki fungsi agar dapat menjalankan program melalui komputer yang telah terkoneksi satu sama lain pada waktu yang sama.

Cloud Computing merupakan sebuah teknologi yang menjadikan internet sebagai pusat server untuk mengelola data dan juga aplikasi pengguna. Cloud Computing memudahkan penggunanya untuk menjalankan program tanpa harus menginstall aplikasi terlebih dahulu dan memudahkan pengguna untuk mengakses data dan informasi melalui internet.

Keuntungan dari penggunaan aplikasi berbasis cloud adalah keamanan data dan kita tidak dipusingkan untuk melakukan perawatan terhadap aplikasi tersebut karena sudah menjadi tanggung jawab penyedia layanan cloud tersebut.

32. 11. Komponen dari dua data atau lebih infrastruktur cloud disebut... a. Hybrid cloud b. Community cloud c. On-site-cloud d. Private cloud e. Public cloud​


Jawaban:

A. Hybrid cloud

Penjelasan:

Hybrid cloud adalah jenis infrastruktur cloud yang terdiri dari dua atau lebih komponen dari jenis infrastruktur cloud yang berbeda, seperti public cloud, private cloud, atau on-premises cloud. Dalam hybrid cloud, setiap jenis infrastruktur cloud tetap terpisah, tetapi dihubungkan melalui jaringan yang aman dan dapat berkomunikasi dengan lancar satu sama lain. Dengan menggunakan hybrid cloud, organisasi dapat memanfaatkan keuntungan dari setiap jenis infrastruktur cloud yang tersedia, misalnya keamanan dan kontrol yang lebih baik dari private cloud dan skalabilitas dan elastisitas dari public cloud. Hybrid cloud juga memberikan fleksibilitas bagi organisasi untuk memilih infrastruktur cloud yang paling cocok dengan kebutuhan bisnis mereka serta memungkinkan organisasi untuk memindahkan data dan aplikasi dari satu infrastruktur cloud ke infrastruktur cloud lainnya jika diperlukan.
