4. Diketauhi fungsi f(×)=2×-1 dan g(×)= 3×^2-×+5.komposisi fungsi (gof) adalaah


f(x) = 2x-1

g(x) = 3x² - x + 5

komposisi fungsi (gof) adalah

(gof)(x) = g(f(x)) = g(2x - 1)

            = 3(2x - 1)² - (2x - 1) + 5

            = 3(4x² - 4x + 1) - 2x + 6

            = 12x² - 12x + 3 - 2x + 6

            = 12x² - 14x + 9

Jadi, (gof)(x) = 12x² - 14x + 9

Jawab:

Komposisi fungsi (gof)(x) adalah 12² - 14x + 9

Penjelasan dengan langkah-langkah:

f(x) = 2x - 1

g(x) = 3x² - x + 5

(gof)(x)

= g(f(x))

= g(2x - 1)

= 3(2x - 1)² - (2x - 1) + 5

= 3(4x² - 4x + 1) - 2x + 1 + 5

= 12x² - 12x + 3 - 2x + 1 + 5

= 12² - 14x + 9

8. 15 fungsi fungsi dasar CMD ialahn?


1. ping

Perintah cmd pertama yang paling sering digunakan dalam mengatur jaringan adalah ping, ya perintah ini fungsinya adalah untuk mengetes atau mengecek koneksi pada jaringan lokal atau internet, contoh penggunaan : ping www.google.com

2. ipconfig /all

Perintah yang kedua yaitu ipconfig /all , perintah ini fungsinya adalah untuk menampilkan informasi konfigurasi ip address yang kita lakukan, seperti informasi lan card, nomer ip address, DNS type, host name, ethernet LAN dan informasi lainnya.

3. ipconfig /flushdns

Perintah yang ketiga ipconfig /flushdns ini fungsinya adalah untuk menghapus dan juga mereset DNS cache di Windows, dan akan memaksa Windows mendapatkan alamat baru dari DNS server setiap kita akan mengakses situs baru.

4. ipconfig /registerdns

Perintah ipconfig /registerdns ini adalah kebalikan dari flusdns, yaitu digunakan untuk merefresh kembali DNS Server, den meregister ulang kembali koneksi DNS.

5. ipconfig /displaydns

Perintah ipconfig /displaydns ini digunakan untuk menampilkan DNS cache.

6. ipconfig /release

Perintah ipconfig /release ini fungsinya adalah untuk menghapus semua koneksi ip address yang ada di jaringan.

7. ipconfig /renew

Perintah ipconfig /renew ini digunakan untuk membuat dan mengkonfigurasi ip address yang baru

8. ipconfig /showclassid

Perintah ipconfig /showclassid ini digunakan untuk menampilkan informasi kelas DHCP

9. ipconfig /setclassid

Perintah ipconfig /setclassid ini digunakan untuk mengubah id kelas DHCP.

10. tracert

Perintah tracert ini digunakan untuk melihat jalur yang dilewati paket data ke alamat jaringan atau situs yang di tuju, perintah ini mirip dengan perintah ping, hanya saja perintah tracert ini memberikan informasi jalur data yang dilewati.

11. route

Perintah route ini digunakan untuk membuat jalur baru (routing) pada komputer ke suatu jaringan, biasanya pada LAN atau WAN . Kemudian untuk mengetahui proses lalu lintas jaringan, mengetahui informasi jalur host, gateway, dan network destination Anda untuk paket yang akan dikirim.

contoh :

route add 172.16.0.0 mask 255.255.0.0 172.16.1.1 ( perintah ini digunakan untuk menambah routing agar koneksi internet stabil, namun seting ini tidak permanen, jika Anda ingin permanen gunakan perintah route -p )

12. netstat

Peritah nestsat adalah singkatan dari network status, perintah netstat ini digunakan untuk menampilkan koneksi yang masuk maupun keluar di dalam jaringan atau koneksi jaringan yang sedang aktif,

netstat -n untuk menampilkan ip dan port yang sedang di akses silahkan gunakan perintah

netstat -b untuk melihat aplikasi atau program yang aktif dalam jaringan gunakan perintah

netstat -s untuk melihat statistik jaringan gunakan perintah

13. hostname

Peintah hostname ini digunakan untuk menampilkan nama komputer Anda.

14. control netconnections

Perintah control netconnections ini digunakan untuk menampilkan pengaturan network connections.

15. nslookup

Perintah nslookup ini digunakan untuk menampilkan ip address dari domain yang anda ingin ketahui.

16. diketauhi suatu fungsi lintas f(×):2×+m tentukan bentuk fungsi tersebut jika f(3):4​


Jawaban:

Diketahui suatu fungsi lintas f(x) = 2x + m dan f(3) = 4.

Untuk mencari bentuk fungsi tersebut, kita dapat menggunakan informasi f(3) = 4, yaitu:

f(3) = 2(3) + m = 6 + m = 4

Dari sini, kita dapat mencari nilai m:

6 + m = 4

m = 4 - 6

m = -2

Jadi, bentuk fungsi lintas f(x) = 2x - 2.

33. diketauhi fungsi f(x)=(x-1)²(2x+3) maka turunan fungsinya Adalah


~Turunan

f(x) = (x - 1)²(2x + 3)

f'(x) = 2(x - 1)(2x + 3) + 2(x - 1)²

f'(x) = 2(2x² + x - 3) + 2(x² - 2x + 1)

f'(x) = 4x² + 2x - 6 + 2x² - 4x + 2

f'(x) = 6x² - 2x - 4

TURUNAN

f(x) = (x - 1)² (2x + 3)

Misalkan :

u = (x - 1)²

u'= 2(x -1)

v = 2x + 3

v' = 2

Maka,

uv' + u'v

= 2(x -1)² + (2x - 2) (2x + 3)

= 2(x² - 2x + 1) + (4x² + 2x - 6)

= 2x² - 4x + 2 + 4x² + 2x - 6

= 6x² - 2x - 4

= 3x² - x - 2

Penjelasan dengan langkah-langkah:

diketahui:

f(x) = 3x - 5

ditanya : nilai f(12) ?

jawab:

• f(x) = 3x - 5

f(12) = 3(12) - 5

= 36 - 5

= 31

jadi nilai dari f(12) adalah 31
