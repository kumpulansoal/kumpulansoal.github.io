15. Apa perbedaan keyboard USB dengan keyboard PS2. Jelaskan!


Jawaban:

→ Perbedaan dari segi sambungan kabel:

- keyboard USB disambungkan ke komputer dg port USB (berbentuk persegi panjang).

- keyboard PS2 disambungkan ke komputer dg port PS2 (berbentuk lingkaran)

→ Perbedaan dari segi pemakaian:

- keyboard USB dapat langsung disambungkan ke komputer saat komputer menyala (dan keyboard dapat langsung terdeteksi/ dipakai)

- keyboard PS2 disambungkan ke komputer saat komputer mati (karena port PS2 tidak dapat membuat keyboard langsung terdeteksi).

24. Tuliskan dan jelaskan menurut anda, keyboard serial, keyboard ps/2, keyboard wireless, keyboard usb ! bantu jawab ya kak ​


Jawaban:

keyboard serial : keyboard yang memakai koneksi port berjenis serial untuk koneksinya biasa digunakan pada perangkat lama.

Keyboard Ps/2 : Keyboard yang menggunakan konektsi port berjenis ps/2, dalam media koneksi ini keyboard ada yang digabung dengan mouse dan ada yang tidak.

keyboard wireless : keyboard yang menggunakan media nirkabel seperti bluetooth, radiowave,dll untuk berfungsi, keyboard ini memerlukan recharge ketika habis baterai.

keyboard usb : keyboard yang paling umum digunakan saat ini, keyboard ini menggunakan port usb sebagai konektor.

31. tulisan shortcut keyboard untuk software blender dalam bahasa Indonesia secara lengkap yah Terimakasih ​


Blender adalah software yang berfungsi untuk membuat model tiga dimensi. Shortcut keyboard untuk software Blender, yaitu:

Tombol TAB, untuk mengubah antara “edit mode” dan “object select mode”.Tombol “O”, untuk mengaktifkan “proportional vertex editing” pada “edit mode”.Tombol “A”, untuk memilih semua vertices yang terdapat pada sebuah obyek.Tombol “B”, untuk memilih banyak objek dengan menggunakan windows drag.Tombol Space atau spasi, untuk menampilkan menu tool.Tombol angka, untuk mengontrol view, yaitu tombol “7” untuk “top”, tombol “1” untuk “front”, tombol “3” untuk “side”, tombol “0” untuk “camera”, tombol “5” untuk “perspective”, tombol “.”, untuk memperbesar objek yang telah dipilih, tombol “+” dan “-“ untuk memperbesar atau memperkecil.Tombol Shift, untuk memilih objek lebih dari satu.Tombol Arrow atau panah, untuk pembuatan frame dalam sebuah animasi.Tombol “R”,untuk merotasi sebuah objek.Tombol “S”, untuk mengatur skala sebuah objek.Tombol “G”, untuk memindahkan sebuah objek.Tombol “P”, untuk memisahkan objek yang dipilih menjadi sebuah objek dalam “edit mode”.Tombol Shift + “D”, untuk menduplikat sebuah objek.Tombol “E”, untuk meng-extrude objek yang telah dipilih di dalam “edit mode”.Tombol “U”, untuk menampilkan Single User menu, sedangkan di dalam “edit mode” berfungsi sebagai “undo”.Tombol “M”, untuk memindahkan objek ke layer lain, dan di dalam “edit mode” berfungsi sebagai “Mirror”.Tombol “Z”, untuk merubah view dari wireframe menjadi solid.Tombol Alt + “Z”, untuk merubah view tekstur.Tombol “P”, untuk memulai mode game.Tombol ALT / CTRL + “P”, untuk membuat atau menghapus hubungan Parent/Child.Tombol “N”, untuk menampilkan info sebuah obyek dalam bentuk numeric atau angka.Tombol Ctrl + “J”,  untuk menggabungkan objek.Tombol Alt + “A”, untuk menjalankan animasi.Tombol “F”, untuk membuat sebuah sisi pada “edit mode”.Tombol “W”, untuk menggabungkan atau mengiris pada 2 objek atau lebih.Tombol “X” atau tombol delete untuk menghapus objek, vertices, atau sisi.Tombol Function yaitu F1 untuk Load File, F2 untuk Save, F3 untuk Save Image, F4 untuk Lamp Buttons, F5 untuk Material Buttons, F6 untuk Texture Buttons, F7 untuk Animation Buttons, F8 untuk Real Time Buttons, F9 untuk Edit Buttons, F10 untuk Display Buttons, F11 untuk Last Render, F12 untuk Render.Tombol “I”, untuk memasukkan animasi.Tombol ALT + “U”, untuk menampilkan “Global Undo Command”.Tombol ALT + “C”, untuk mengkonversi menjadi meshes, text dan curves.Tombol Shift + Space, untuk menampilkan seluruh layar atau hanya pada viewport yang aktif saja.Tombol Ctrl + “0”, untuk memilih sebuah camera.

Pembahasan

Blender merupakan aplikasi atau software berfungsi untuk membuat karya tiga dimensi (3D) atau model tiga dimensi. Aplikasi Blender bersifat gratis dan  open source, dan dapat digunakan di berbagai sistem operasi, seperti Windows, macOS, serta Linux.

Aplikasi Blender memiliki berbagai macam fitur, yaitu untuk 3D modelling, rigging, fluid dynamics, keyframe animation, photorealistic rendering dan video editing.
