Microsoft Office adalah software paket aplikasi perkantoran buatan Microsoft dan dirancang untuk dijalankan di bawah sistem operasi Microsoft Windows dan Mac OS X. Alasan terbesar banyak orang memakai Microsoft Office adalah karena hampir semua tempat kerja memakai software ini. Tampilan Microsoft Office juga sangat user friendly, dengan banyak format pengeditan file akan membuat anda nyaman menggunakan software ini dibanding software lain.

Beberapa contoh aplikasi di dalam Microsoft Office yang terkenal adalah Word, Excel, dan PowerPoint. Dalam artikel ini, saya akan menjelaskan pengertian dan juga fungsi dari masing-masing aplikasi tersebut.

Microsoft Word

Microsoft Word adalah aplikasi pengolah kata yang sangat populer pada saat ini. Microsoft Word dapat memudahkan kerja dalam melakukan pengetikan surat maupun dokumen lain.

Dalam perkembangannya Microsoft Word mengalami banyak perkembangan dari tahun ke tahun mulai dari Microsoft Word 1998, Microsoft Word 2003, Microsoft Word 2007, Microsoft Word 2010 dan Microsoft Word 2013. Dengan perkembangan tersebut microsoft telah menambahkan database dan tool yang baru untuk menyempurnakan agar Microsoft Word lebih mudah untuk digunakan.

Fungsi Microsoft Word :

Membuat surat-menyurat baik untuk keperluan resmi maupun tidak resmi.

Membuat dokumen dengan cepat serta tepat.

Membuat dokumen dengan berbagai variasi agar lebih menarik.

Menambahkan gambar pada dokumen.

Membuat table.

Microsoft Power Point

Microsoft Office PowerPoint atau PowerPoint adalah aplikasi yang digunakan untuk presentasi. Dengan bantuan software ini, seseorang bisa membuat bentuk presentasi profesional dengan mudah bahkan presentasi tersebut dapat digunakan sebagai bahan pembelajaran. Aplikasi ini sangat banyak digunakan, apalagi oleh kalangan perkantoran dan pebisnis, guru, siswa, dan trainer.

Dimulai pada versi Microsoft Office System 2003, Microsoft mengganti nama dari sebelumnya Microsoft PowerPoint saja menjadi Microsoft Office PowerPoint. Lalu, pada Office 2013, namanya cukup disingkat PowerPoint. Versi terbaru dari PowerPoint adalah versi 15 (Microsoft Office PowerPoint 2013) yang tergabung ke dalam paket Microsoft Ofice 2013.

Fungsi Microsoft Power Point :

Memudahkan pengguna mengatur materi yang hendak disampaikan.

Membuat audience lebih gampang memahami materi presentasi karena hanya menampilkan poin-poin utama yang disuguhkan dalam bentuk slide.

Memudahkan pengguna dalam menuangkan imajinasinya karna format pengeditan relatif lengkap dibanding software lain.

Membuat penyajian materi lebih berkesan, apalagi jika pengguna menambahkan animasi-animasi di dalamnya. Sebab, pada kasus yang sering ditemui, audience kurang fokus dan bosan apabila materi nan ditampilkan monoton.

Microsoft Excel

Microsoft Office Excel atau Excel adalah aplikasi pengolah data / angka. Menggunakan spreadsheet yang terdiri dari baris dan kolom untuk mengeksekusi perintah. Aplikasi ini memiliki fitur kalkulasi dan pembuatan grafik. Microsoft Excel tidak hanya tersedia dalam platform Windows, Microsoft Excel juga tersedia di MacOS, Android dan Apple.

Aplikasi ini merupakan bagian dari Microsoft Office System, dan versi terakhir adalah versi Excel 2013 yang diintegrasikan di dalam paket Microsoft Office 2013.

Membuat, mengedit, mengurutkan, menganalisis, meringkas, dan memformat data serta grafik.

Membuat catatan keuangan dan anggaran keuangan.

Menghitung dan mengelola investasi, pinjaman, penjualan, inventaris, dll.

Melakukan perhitungan statistika.

tata cara menggunakan Microsoft Windows 10

1. Touchpad Gesture untuk Akses Windows Lebih Cepat

Kamu perlu mengecek laptopmu dengan masuk ke Settings > Change PC Settings > PC and Device > Mouse and touchpad. Jika fitur ini aktif, maka kamu akan menemukan tulisan “your PC has a precision touchpad.”

Dengan cara ini, kamu bisa mencoba swipe tiga jadi ke atas untuk melihat task view. Tiga jari swipe ke bawah untuk melihat desktop, tiga jari kekanan atau kekiri untuk masuk ke halaman berikutnya atau aplikasi lainnya. Dan men-tap tiga jari pada touchpad akan membukan fitur pencarian Cortana.

 

2. Menekan tombol Windows dan tombol kiri atau kanan

Tips ini membuat window yang kamu buka akan memisahkan diri ke kanan atau ke kiri layar. Tidak hanya 2 window yang bisa kamu split. Kamu bisa men-split 4 window berbeda. Fitur ini sangat berguna untuk kamu yang gemar multitasking di perangkat Windows.

3. Menekan tombol ALT dan Tab atau tombol Windows dan Tab

Tips ini akan  membuka semua Windows. Tekan Tab secara berulang untuk berpindah jendela. Lepaskan tombol Tab untuk membuka Windows yang dipilih.

Baca juga: Bug Windows 10 Mobile Buat Sistem Android Bisa Diinstal di Windows Phone

 

4. Virtual Desktop di Windows 10

Fungsi virtual desktop di Windows 10 ini adalah untuk memberikan kamu desktop lain sesuai kebutuhan. Misal, desktop untuk kerja dan desktop untuk keperluan pribadi. Kamu bisa membuat 4 virtual desktop sekaligus. Untuk mengaktifkan virtual desktop, kamu cukup menekan Ctrl + Windows + D. Untuk menghapus virtual desktop, kamu cukup menekan tombol Windows + Ctrl + F4

5. Aktifkan Hey Cortana

Kalau kamu mengucap Hey Cortana dan tidak ada respon, sebaiknya kamu coba masuk ke Settings > Cortana and Search. Pastikan fitur cortana kamu sudah aktif. Namun, jangan selalu mengaktifkan fitur ini karena cukup boros baterai.

6. Menonaktifkan fitur update Windows

Salah satu keunggulan sistem ini dibanding Windows lainnya ialah adanya fitur update yang berjalan otomatis. Untuk mematikannya, kamu cukup masuk ke Settings > update and security > windows update > advanced options > chose how update are delivered dan matikan pilihannya.

Baca juga: Tablet Windows 12 Inci dengan Windows 10 Masih Misteri

7. Klik kanan pada bagian tertentu dan pilih Ask Cortana

Bila kamu browsing dan menemukan sesuatu, kamu bisa klik kanan pada bagian tersebut dan pilih Ask Cortana. Nantinya Cortana akan menyajikan informasi lain yang berkaitan. Namun fitur ini untuk sementara hanya bisa di browser Microsft Edge.

 

8. Gunakan PIN lebih baik daripada password di Windows 10

Windows 10 kini dapat menggunakan PIN untuk proteksi perangkat Windows 10 kamu. Kalau biasanya kamu menggunakan password yang menyulitkan, kamu sekarang bisa menggunakan PIN yang tentunya lebih gampang diingat. Kamu cukup tekan tombol Windows, ketik PIN dan klik Set up PIN sign-in lalu enter.

9. Full Screen Start Menu

Kalau kamu terbiasa menggunakan Windows 8 atau Windows 8.1, kamu akan menemukan full screen start menu. Nah di Windows 10 ini kamu juga bisa mengaktifkan fitur ini dengan cara klik tombol Windows, ketik 'full screen' dan pilih 'Use Start full screen' lalu tekan enter dan aktifkan Use Start full screen.

27. Jelaskan perbedaan cara mematikan komputer (shutdown) pada: windows 7, windows 8 dan windows 10!..mohon di jawab kk​


Inilah perbedaan utama untuk Windows 8 : seperti pada Windows 7, kami menutup sesi pengguna, tetapi alih-alih menutup sesi kernel, kami hibernasi . Dibandingkan dengan hibernasi penuh, yang mencakup banyak halaman memori yang digunakan oleh aplikasi, data hibernasi sesi 0 jauh lebih kecil, yang membutuhkan waktu lebih sedikit untuk menulis ke disk. Jika Anda tidak terbiasa dengan hibernasi, kami secara efektif menyimpan status sistem dan konten memori ke file di disk (hiberfil.sys) dan kemudian membacanya kembali di resume dan mengembalikan konten kembali ke memori. Menggunakan teknik ini dengan boot memberi kita keuntungan yang signifikan untuk waktu boot, karena membaca hiberfile in dan menginisialisasi ulang driver jauh lebih cepat pada sebagian besar sistem (30-70% lebih cepat pada kebanyakan sistem yang telah kami uji).

34. dimana letak perbedaan cara mematikan komputer/cold booting windows xp dengan windows 8?


Langkah – langkah menyalakan komputer dengan sistem booting :
1. Cek dulu keadaan komputer, apakah sudah terhubung dengan aliran listrik
2. Nyalakan tombol power pada Stavol
1. Tekan tombol “Power” pada CPU
1. Tekan tombol “ Power “ pada Monitor (monitor dalam keadaan on ditandai dengan led menyala / switch pada posisi on)
1. Tunggu sampai monitor menyala dan mengeluarkan layar desktop windows.
Ada 2 macam isitilah booting, yaitu :
1. COLD BOOTING (Booting dingin)
Yaitu proses menjalankan computer pertama kali (computer dalam keadaan mati ).
Proses ini dapat dilakukan dengan cara menekan tombol ON
1. WARM BOOTING (Reset/ Booting Panas)
Yaitu proses menjalankan komputer ketika computer dalam keadaan hidup.
