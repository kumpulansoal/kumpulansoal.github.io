Asking and responding for attention:

1. Attention, please!

What happened?

2. May I get your attention, please?

Yes, sure.

3. Look what I’ve got here. 

What is it?

4. Please don’t be noise, attention please!

I'm sorry. Okay, i'll listen now.

5. Excuse me, can you give me a second?

I'm sorry. I'm busy now.

Asking and giving something:

1. Can you give it me?

Yes, here you are./ I'm sorry. I can't reach it.

Sorry, i don't have money right now. / yes, sure. how much do you need?

I’m sorry. I’m using it./ Of course.

25. tolong buatkan expression of asking for and giving something



1. To ask for things, we can use the following sentences:
- Can I…., please?
- May I…., please?
- Can you…., please?

2. To give the things, we can use the following sentences:
- Here you are.
- Here it is.
- Okay
- Sure.
- Certainly
- Of Course.
- I would be glad/happy to ........

3. To refuse to give the things, we can use the following sentences:
- I’m sorry. I’m using it.
- Not now. I’m using it. Sorry.
- Not now. I’m using them. Sorry
- Sorry, I’m using it right now.
- Sorry, I’m using them right now.

4. To show our gratitude, we can use the following sentences:
- Thanks.
- Thanks a lot
- Thank you
- Thank you very much

5. To answer the gratitude expression, we can use the following sentences:
- You're welcome
- It's OK
- My pleasure
- No problem
- Don't mention it
