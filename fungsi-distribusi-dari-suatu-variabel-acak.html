11. Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar. Variabel acak x menyatakan banyak kejadian muncul sisi angka. Tentukan distribusi probabilitas, fungsi probabilitas


Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar. Variabel acak x menyatakan banyak kejadian muncul sisi angka dan n adalah banyaknya percobaan.

Maka Distribusi probabilitasnya adalah

P(X = 0) = (1/3)ⁿ

P(X = 1) = n (1/3)ⁿ⁻¹ ( 2/3 )

P(X = 2) = [ n(n-1)/2 ] (1/3)ⁿ⁻² ( 2/3 )²

dan seterusnya....


Fungsi probabilitasnya adalah

P ( X = a ) = ⁿCₐ ( 2/3 )ᵃ ( 1/3 )ⁿ⁻ᵃ


PEMBAHASAN

Peluang suatu kejadian di definisikan sebagai kemungkinan terjadinya suatu kejadian terhadap segala kemungkinan yang ada ( ruang sampel ).

[tex]\textbf{Peluang = banyak kejadian yang di maksud / ruang sampel}[/tex]

Kita misalkan percobaannya adalah melempar sebuah uang koin, maka ruang sampelnya adalah munculnya Gambar(G) atau Angka(A) pada koinnya.  Jadi :

Peluang munculnya Gambar P(G) = 1/2

Peluang munculnya Angka P(A) = 1/2

Sekarang kalau percobaannya adalah melempar 2 buah uang koin , maka ruang sampelnya adalah munculnya Gambar-Gambar (GG), Gambar-Angka (GA) , Angka-Gambar (AG) , dan Angka-Angka(AA). Jadi :

Peluang munculnya 2 Gambar P (2G) = (1/2)² =1/4

Peluang munculnya 1 Gambar P (1G) = 2 x (1/2) x (1/2) = 2/4

Peluang munculnya 0 Gambar P (0G) = (1/2)² =1/4

Sekarang kalau percobaannya adalah melempar 3 buah uang koin , maka dengan cara yang sama akan di dapatkan :

Peluang munculnya 3 Gambar P (3G) = (1/2)³ = 1/8

Peluang munculnya 2Gambar P (1G) = 3 x (1/2)² (1/2) = 3/8

Peluang munculnya 1 Gambar P (0G) = 3 x (1/2)² (1/2) = 3/8

Peluang munculnya 0 Gambar P (0G) = (1/2)³ = 1/8

Distribusi probabilitas seperti ini di sebut distribusi binomial yang memiliki rumus :

P ( X = a ) = ⁿCₐ ( p )ᵃ ( 1 - p )ⁿ⁻ᵃ

Okay lah dengan menggunakan prinsip ini kita akan menyelesaikan soal yang ditanyakan.

Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar , maka bisa di tuliskan sebagai berikut :

P(A) = 2 x P(G)

Sedangkan kita tahu bahwa probabilitas bernilai maksimum 1, maka:

P(A) + P(G) = 1

2 P(G) + P(G) = 1

3 P(G) = 1

P(G) = 1/3

P(A) = 2 x P(G)

P(A) = 2 x 1/3

P(A) = 2/3

Berhubung variabel acak X menyatakan banyak kejadian muncul sisi angka  dan anggap jumlah percobaan lemparan sebanyak n maka bisa di tuliskan distribusi probabilitasnya sebagai berikut ini :

P(X = 0) = (1/3)ⁿ

P(X = 1) = n (1/3)ⁿ⁻¹ ( 2/3 )

P(X = 2) = [ n(n-1)/2 ] (1/3)ⁿ⁻² ( 2/3 )²

dan seterusnya....

Maka bisa di buat fungsi probabilitasnya adalah :

P ( X = a ) = ⁿCₐ ( 2/3 )ᵃ ( 1/3 )ⁿ⁻ᵃ

Plis tolong bantu jawab ya plis..diketahui fungsi kepadatan probabilitas diskrit sebagai berikut:P(X) = X/10 , X = 1,2,3 atau 4.pertanyaan :a. berapa probabilitas bahwa x=2 atau pr(X=2) ?b. berapa probabilitas bahwa x>2 atau pr(X>2) ?c. gambarkan grafik distribusi probabilitasnyad. hitung E(x) atau nilai harapan dari variable acak tersebute. hitung variansi dan standar deviasi dari variabel acak tersebut​


Jawaban:

a. Untuk menghitung probabilitas bahwa X = 2 atau P(X = 2), kita dapat menggunakan fungsi kepadatan probabilitas yang diberikan:

P(X) = X/10

Menggantikan X dengan 2, kita dapatkan:

P(X = 2) = 2/10 = 0.2

Jadi, probabilitas bahwa X = 2 adalah 0.2.

b. Untuk menghitung probabilitas bahwa X > 2 atau P(X > 2), kita perlu menjumlahkan probabilitas untuk semua nilai X yang lebih besar dari 2. Dalam hal ini, nilai X yang lebih besar dari 2 adalah 3 dan 4.

P(X = 3) = 3/10 = 0.3

P(X = 4) = 4/10 = 0.4

P(X > 2) = P(X = 3) + P(X = 4) = 0.3 + 0.4 = 0.7

Jadi, probabilitas bahwa X > 2 adalah 0.7.

c. Grafik distribusi probabilitas dapat digambarkan dengan memplot titik-titik probabilitas untuk setiap nilai X pada sumbu X. Dalam kasus ini, kita memiliki empat nilai X yaitu 1, 2, 3, dan 4. Probabilitasnya adalah X/10.

Grafik distribusi probabilitasnya adalah sebagai berikut:

X | P(X)

--------------

1 | 0.1

2 | 0.2

3 | 0.3

4 | 0.4

d. Untuk menghitung nilai harapan E(X) dari variabel acak, kita dapat menggunakan rumus:

E(X) = Σ(X * P(X))

Menggantikan nilai X dan P(X) dengan nilai yang diberikan, kita dapat menghitung nilai harapan sebagai berikut:

E(X) = (1 * 0.1) + (2 * 0.2) + (3 * 0.3) + (4 * 0.4)

= 0.1 + 0.4 + 0.9 + 1.6

= 3

Jadi, nilai harapan E(X) dari variabel acak tersebut adalah 3.

e. Variansi (Var) dan standar deviasi (SD) dari variabel acak dapat dihitung menggunakan rumus-rumus berikut:

Var(X) = Σ((X - E(X))^2 * P(X))

SD(X) = √Var(X)

Menggantikan nilai X, E(X), dan P(X) dengan nilai yang diberikan, kita dapat menghitung variansi dan standar deviasi sebagai berikut:

Var(X) = ((1 - 3)^2 * 0.1) + ((2 - 3)^2 * 0.2) + ((3 - 3)^2 * 0.3) + ((4 - 3)^2 * 0.4)

= (2^2 * 0.1) + (1^2 * 0.2) + (0^2 * 0.3) + (1^2 * 0.4)

= 0.4 + 0.2 + 0 + 0.4

= 1

SD(X) = √Var(X)

= √1

= 1
