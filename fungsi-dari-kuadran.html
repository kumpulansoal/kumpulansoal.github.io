5. titik puncak dari grafik fungsi kuadran dengan persamaan y= -4x +8x -3


Dari persamaan kuadrat tersebut diketahui
a = - 4, b = 8 dan c = - 3

[tex] \frac{ - b}{2a} = \frac{ - 8}{2( - 4)} = \frac{ - 8}{ - 8} = 1[/tex]
[tex] \frac{ - ( {b}^{2} - 4ac)}{4a} = \frac{ - ( {8}^{2} - 4( - 4)( - 3) }{4( - 4)} \\ = \frac{ - (64 - 48)}{ - 16} \\ = \frac{ - 16}{ - 16} \\ = 1[/tex]
Jadi titik puncaknya adalah (1, 1)

26. nilai minimum fungsi kuadran f(x)=xkuadrat-6x+4 adalah


[tex]f(x) = x^{2} - 6x + 4[/tex]
Turunan pertama
[tex]f'(x) = 2x - 6[/tex]
[tex]0 = 2x - 6[/tex]
[tex]x = 3[/tex]

Subsitusi x dalam f(x) maka kita dapat
[tex]f(3) = 3^{2} - 6(3) + 4[/tex]
[tex]f(3) = 9 - 18 + 4[/tex]
[tex]f(3) = -5[/tex]

Maka nilai minimum pada fungsi kuadrat [tex]f(x) = x^{2} -6x + 4[/tex] adalah -5
