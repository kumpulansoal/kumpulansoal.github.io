1. Fungsi yang digunakan untuk mencari hasil dengan cara membandingkan data dengan syarat tertentu adalah


Jawaban:

Fungsi LOGIKA

Penjelasan:

Fungsi yang digunakan untuk mencari hasil dengan cara membandingkan data dengan syarat tertentu adalah fungsi LOGIKA. Fungsi LOGIKA merupakan fungsi yang biasa digunakan dalam lembar kerja elektronik seperti Microsoft Excel untuk mengevaluasi suatu pernyataan dan mengembalikan hasilnya sebagai benar (TRUE) atau salah (FALSE).

Contoh penggunaan fungsi LOGIKA adalah fungsi IF, yang digunakan untuk menentukan apakah suatu pernyataan benar atau salah dan memberikan hasil yang sesuai. Misalnya, jika Anda ingin mencari data yang memenuhi syarat tertentu, Anda dapat menggunakan fungsi IF untuk membandingkan data dengan syarat yang ditentukan dan menampilkan hasil sesuai dengan apa yang Anda inginkan.

Selain fungsi IF, fungsi LOGIKA juga mencakup fungsi-fungsi lain seperti AND, OR, dan NOT yang dapat digunakan untuk mengevaluasi pernyataan yang lebih kompleks.

Program dalam bahasa C++

// AUTHOR : LordRimuruTempest

#include <iostream>

void tahunKabisat(int Tahun){

   if(Tahun % 100 == 0){

      if(Tahun % 400 == 0){

          std::cout<<"Tahun "<<Tahun<<" merupakan tahun kabisat";

      }else{

          std::cout<<"Tahun "<<Tahun<<" bukan tahun kabisat";

      }

  }else{

      if(Tahun % 4 == 0){

          std::cout<<"Tahun "<<Tahun<<" merupakan tahun kabisat";

      }else{

          std::cout<<"Tahun "<<Tahun<<" bukan tahun kabisat";

      }

  }

}

int main() {

  int Tahun;

  std::cout<<"Masukkan tahun : ";

  std::cin>>Tahun;

  tahunKabisat(Tahun);

  return 0;

}

Pembahasan

Contoh program yang dapat dibuat menggunakan nested if adalah program menentukan angka tahun kabisat. Sebelum mencoba menerapkan algoritma ke dalam program C++, maka harus diketahui algoritmanya sebagai berikut

1. Program dimulai

2. Masukkan angka tahunnya

3. Jika angka tahun habis dibagi 100 (sisa baginya 0), maka

Jika angka tahun habis dibagi 400 (sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 400 (sisa baginya bukan 0), cetak "bukan tahun kabisat".

4. Jika angka tahun tidak habis dibagi 100 (sisa baginya bukan 0), maka

Jika angka tahun habis dibagi 4(sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 4(sisa baginya bukan 0), cetak"bukan tahun kabisat".

5. Program selesai.

Algoritma diatas dalam bentuk pseudocode dapat ditulis

Algoritma tahunKabisat

program start

var tahun : integer;

if(tahun % 100 == 0) then

   if(tahun % 400 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

else then

   if(tahun % 4 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

end if

program end

Setelah algoritma diketahui, maka sisanya adalah menerapkan algoritma kedalam kode program. pertama-tama import library iostream(input output stream) ke dalam program dengan perintah #include <iostream>. Fungsi library ini adalah untuk menerima input dan mencetak output. Kemudian buatlah procedure/function bernama tahunKabisat, function tersebut bertipe void (tidak mengembalikan nilai apapun) dan memiliki parameter Tahun(bertipe integer). Dalam function tahunKabisat, dibuat nested if sesuai petunjuk/alur dari algoritma. Kemudian untuk main function, deklarasikan variabel Tahun bertipe integer dan minta user untuk memasukkan angka tahunnya dengan menggunakan cin(c input). Langkah terakhir adalah memanggil function tahunKabisat dan memasukkan nilai variabel Tahun tadi kedalam function tahunKabisat. Kode program beserta output-nya dapat dilihat pada lampiran gambar.

(If) untuk satu kondisi  

If adalah pernyataan yang digunakan untuk satu kondisi apabila kondisi yang ada telah ditetapkan bernilai TRUE maka program akan mengeksekusi yang ada didalamnya, jika bernilai FALSE maka akan diabaikan. Bentuk umum:

         

         if(kondisi1):   #statement1

                   pernyataan1

Contoh :

nilai = 9

if(nilai >= 7):

       print("Selamat anda lulus")

#hasil program diatas akan menghasilkan output Selamat anda lulus

 

(if....else) untuk satu kondisi dan tidak terpenuhinya kondisi

ini merupakan lanjutan dari perintah if, If..else adalah untuk melakukan penyeleksian kondisi dimana jika kondisi bernilai benar maka program akan mengeksekusi statemen 1. Namun jika nilai kondisi bernilai salah, maka statemen 2 yang akan dieksekusi. Bentuk umum:

if(kondisi1):   #statement1

                pernyataan1

       else:            #statement2

                pernyataan2

Contoh:

nilai = int(input("masukan nilai = "))

if(nilai<=100 and nilai>=70):

           print("anda lulus")

else:

           print("anda tidak lulus")

(if...elif.. else)  untuk dua kondisi atau lebih

Statemen if – else – elif digunakan untuk melakukan penyeleksian kondisi Diana kondisi yang diberikan lebih dari 1 kondisi atau memiliki beberapa kondisi. Jika kondisi pertama bernilai benar, maka lakukan seleksi kondisi kedua dan seterusnya. Bentuk umum:

if(kondisi1):   #statement1

                  pernyataan1

       elif(kondisi2):    #statement2

                  pernyataan2

        else:                #statement3

                  pernyataan3

Contoh:

nilai = int(input("Masukkan nilai Anda = "))

if (nilai <= 100):

      if (nilai > 85):

                 print ("Huruf mutu Anda adalah A.")

      elif (nilai > 75):

      elif (nilai > 65):

       elif (nilai <= 55):

33. Komponen yang berfungsi untuk membantu pekerjaan pengguna komputer dalam mengolah berbagai macam data komputer disebut​


Jawaban:

Hardware/ Perangkat Keras

penjelasan:

Hardware merupakan perangkat keras yang kasat mata dan memiliki wujud fisik yang dapat diraba. Hardware pada komputer sendiri masih dibagi lagi menjadi empat bagian yaitu input device, output device, processing device dan storage device.

1. Input Device/ Perangkat Masukan

Bagian pertama adalah input device atau perangkat masukan. Yang termasuk dalam input device ini adalah keyboard, mouse, webcam dan juga scanner. Perangkat tersebut dikategorikan input device karena fungsinya sebagai alat untuk memasukan berbagai data dan juga perintah pada komputer. Sebuah data atau instruksi harus di input terlebih dahulu sebelum diolah menjadi informasi yang seperti kita inginkan, oleh karena itu dibutuhkan sebuah unit input yang menjadi sebuah gerbang untuk data yang dari dunia luar masuk ke dalam sistem komputer. Bentuk-bentuk data yang telah diinput akan dikonversi menjadi kode binary sehingga dapat dibaca oleh memori utama.

2. Processing Device/ Perangkat Proses

Data yang dimasukan melalui input device selanjutnya akan diolah melalui processing device. Dengan kata lain, processing device adalah perangkat yang berfungsi untuk mengolah berbagai data yang berasal dari input device. Yang termasuk dalam processing device adalah CPU (Central Processing Unit). Peran CPU sangat penting didalam sebuah komputer karena merupakan otak dari sistem komputer. CPU sekaligus juga perangkat yang mengatur semua perangkat internal dan eksternal di komputer.

3. Output Device/ Perangkat Keluaran

Perangkat keluaran atau output device adalah perangkat yang memberikan atau mengeluarkan hasil dari data yang masuk dan telah diproses. Contoh dari output device ini adalah monitor, printer, headphone, speaker, webcam dan mikrofon. Dengan kata lain output device ini menyajikan segala informasi yang didapat dari pemrosesan sistem komputer seperti gambar, suara dan video.

4. Storage Device/ Perangkat Penyimpanan

Perangkat terakhir pada hardware adalah perangkat penyimpanan. Sesuai dengan namanya, perangkat ini adalah alat yang digunakan untuk menyimpan berbagai data. Storage device sendiri terdiri dari dua macam yaitu internal storage dan eksternal storage. Contoh dari internal storage adalah harddisk dan RAM. Sedangkan external storage misalnya adalah harddisk eksternal, CD, DVD dan flashdisk.

38. cara menggunakan fungsi IF ( logika) dalam microsoft excel


Isi di formula bar seperti ini:
=IF(nama sel,jika benar,jika salah)
Maksudnya:
- nama sel: nama sel yang menjadi acuannya (bisa diisi nama sel seperti A1, atau nama sel dengan tanda operator logika seperti A1>=75)
- jika benar: string (data/angka/tulisan)atau hasilnya jika sel yang kita jadikan acuan benar
- jika salah: string/hasilnya jika sel yang kita jadikan acuan salah
Pada jika benar dan jika salah bisa diisi dengan angka (50%,30%) atau tulisan (tapi kalau tulisan harus diawali dan diakhiri dengan tanda petik ("..."), misal "Lulus","Tidak Lulus")
- tanda , adalah tanda pemisah, bisa diganti dengan ; (tergantung versi excel yang dipakai dan settingan regional di komputermu)

Contoh:
di sel A5 ada nilai 70, sedangkan kkmnya adalah 75, jadi rumusnya memakai:
=IF(A5>=75,"Lulus","Tidak Lulus") maka hasilnya adalah Tidak Lulus
