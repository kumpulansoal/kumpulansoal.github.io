PERTANYAAN

Suatu fungsi ditentukan dengan f(x) = ax + b. Jika

diketahui nilai f(4) = 5 dan f(-2) = -7, tentukan :

▪Nilai a dan b

▪Bentuk fungsinya

PEMBAHASAN

<<Diketahui>>

Fungsi yang ditentukan = f(x) = ax + b

f(4) = 5

f(-2) = -7

<<Ditanya>>

nilai a dan b

bentuk fungsinya  

<<Jawab>>

Sebelum kita mengetahui nilai a dan b,Sebaiknya kita dapat membuat persamaannya dari f(x) = ax + b terlebih dahulu,Maka :

f(x) = ax + b

f(4) = 4x + b = 5...Untuk Pers 1

f(-2) = -7

= -2x + b = -7...Untuk Pers 2

Sesudah kita membuat Persamaannya,Selanjutnya kita dapat Eliminasi Persamaan 1 dan 2,Maka :

4a + b = 5   I ×1 I 4a + b = 5

-2a + b = -7 I ×1 I -2a + b = -7

                                                        -

                              6a = 12

                                 a = [tex]\frac{12}{6}= 2[/tex]

Setelah Kita mengetahui a,Selanjutnya kita akan mengetahui nilai b dengan cara substitusi kepersamaan 1,Maka :

4a + b = 5

4(2) + b = 5

b = 5 - 4(2) → b = 5 - 8

                         = -3

[tex]\boxed{\boxed{Maka,Nilai~a~dan~b~adalah~2~dan~-3}}}[/tex]

Setelah kita menentukan nilai a dan b,Selanjutnya kita balik kesoal lagi untuk menentukan bentuk fungsinya,Maka :

f(x) = ax + b

= ax + -3

= ax - 3

= 2x - 3

[tex]\boxed{\boxed{Maka,Bentuk~Fungsinya~adalah~2x-3}}}[/tex]

PEMBAHASAN TAMBAHAN

Fungsi adalah sebuah materi didalam matematika yang dimana suatu himpunan yang terdiri dari 2 ,yaitu himpunan C dan himpunan D. dalam Fungsi yang dimana himpunan C dapat dibilang dengan daerah sendiri.yang dimana daerah sendiri hanya memiliki didaerahnya dan juga ada yang namanya daerah lawan yang dimana himpunan D harus ada semuanya dihimpunan C.

Adapun bentuk Atau simbol dari fungsi adalah [tex]f:A \rightarrow B[/tex]

Fungsi memiliki bentuk sifat - sifatnya,Yaitu :

Fungsi surjektif Fungsi Into Fungsi Injektif Fungsi Bijektif

Adapun rumus - rumus untuk mencari fungsi :

[tex]Fungsi~linier=f(x) = ax + b=f^{-1}(x) = \frac{x-b}{a}\\Fungsi~pecahan~linier=f(x) =\frac{ax+b}{cx+d}=f^{-1}(x) = \frac{-dx+b}{cx-a}\\Fungsi~Irrasional=f(x) =\sqrt[n]{ax+b}=f^{-1}(x) = \frac{x^n-b }{a}\\Fungsi~eksponen=f(x) = a^x =f^{-1}(x) = ^a\log x[/tex]

Kita akan menggunakan metode faktorisasi untuk menyelesaikan persamaan ini:

3x² - 7x - 6 = 0

(3x + 2)(x - 3) = 0

Dalam hal ini, nilai x yang membuat g(x) = 0 adalah ketika faktor-faktor dalam kurung menjadi nol, yaitu:

3x + 2 = 0 -> 3x = -2 -> x = -2/3

x - 3 = 0 -> x = 3

Jadi, nilai x yang membuat g(x) = 0 adalah x = -2/3 dan x = 3.

x² - 6x - 10 = 17

x² - 6x - 27 = 0

Kita dapat menyelesaikan persamaan ini menggunakan metode rumus kuadrat:

x = (-b ± √(b² - 4ac)) / (2a)

x = (-(-6) ± √((-6)² - 4(1)(-27))) / (2(1))

x = (6 ± √144) / 2

x = (6 ± 12) / 2

Kita mendapatkan dua solusi:

x₁ = (6 + 12) / 2 = 18 / 2 = 9

x₂ = (6 - 12) / 2 = -6 / 2 = -3

Jadi, nilai x yang membuat h(x) = 17 adalah x = 9 dan x = -3.
