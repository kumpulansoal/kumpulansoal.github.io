Tool yang dikembangkan oleh firefox sebagai add-on browser memungkinkaan untuk melakukan debuging padaa html, css dan javascript dalam browser adalah Firebug.

Yang dimaksud dengan Firebug adalah alat pengembang web yang memungkinan pembetulan bug, penyuntingan, dan pengawasan terhadap CSS, HTML, DOM, XHR, dan JavaScript; alat ini juga menyediakan alat pengembangan web.

Firebug adalah alat pengembang web yang memungkinan pembetulan bug, penyuntingan, dan pengawasan terhadap CSS, HTML, DOM, XHR, dan JavaScript; alat ini juga menyediakan alat pengembangan web. Panel JavaScript Firebug dapat mencatat kesalahan, menampilkan pemanggilan fungsi, dan memungkinkan pengembang untuk menjalankan JavaScript yang berubah-ubah. Panel jaringan dapat mengawasi URL yang diminta peramban, seperti CSS eksternal, JavaScript, dan berkas gambar. Panel jaringan dapat menampilkan header request maupun header response untuk setiap berkas eksternal halaman; juga dapat memperkirakan waktu untuk memuat setiap URL yang diminta tersebut.

Firebug adalah perangkat lunak bebas dan sumber terbuka; berlisensi di bawah lisensi BSD. Firebug awalnya ditulis pada bulan Januari 2006 oleh Joe Hewitt, satu dari pencipta Firefox awal. Firebug Working Group melihat pengembangan sumber terbuka dan ekstensi dari Firebug. Firebug memiliki dua implementasi besar: ekstensi untuk Mozilla Firefox dan implementasi bookmarklet yang disebut Firebug Lite. Versi antar-peramban saat ini sedang dikembangkan. Saat ini, add-on Firebug telah memiliki lebih dari 3 juta pengguna harian aktif.

28. jelaskan istilah web browser, protokol, http, html, dan alamat URL


1. Web browser
Program komputer yang digunakan sebagai media untuk berselancar di internet.

2. Protokol
Aturan aturan yang ada untuk untuk melakukan tugas tertentu, misalnya, di Internet ada protokol untuk mengirimkan pesan ke komputer lain, ada protokol yang mengirimkan emai dan sebagainya.

3. Http://
Http atau kepanjangannya Hyper Text Transfer Protocol adalah kode yang dituliskan pada awal site, untuk menjelaskan pada program web browser bahwa protokol (antarmuka) yang digunakan adalah http.

4. HTML
Hyper Text Markup Language, yaitu bahasa yang digunakan untuk membuat tampilan dalam bahasa program.

5. URL adalah singkatan dari “Unifrom Resource Locator” yaitu merupakan rangkaian karakter menurut format standar tertentu, digunakan untuk menunjukan alamat dari suatu sumber misalnya seperti dokumen, file dan gambar yang terdapat di internet.
