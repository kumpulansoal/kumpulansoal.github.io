1. Rename = Artinya adalah Fitur untuk mengganti nama Pada Akun. Misalnya nama awal adalah abc kemudian akan diganti menjadi cba maka kita bisa menggunakan fitur rename ini.

2. Delete = Delete digunakan untuk menghapus beberapa file yang sudah tidak diperlukan/file sampah. Fitur ini biasanya dimanfaatkan untuk mengosongkan penyimpanan perangkat.

3. Copy = Fitur Copy digunakan untuk menyalin teks/tulisan/file untuk kemudian nantinya dipastekan lagi di tempat lain.

4. Cut = Fitur Cut berfungsi untuk memotong sebagian/beberapa bagian dari file yang tidak diperlukan.


4. Apa fungsi header, footer, save, save as, rename, delete, back space, page lay out dan print previw


- Header ialah teks yg diletakkan khusus di bagian atas halaman yang akan selalu tampil di setiap halaman. header dapat difungsikan untuk menampilkan nomor halaman , tanggal , jam  dan sebagainya.- Footer ialah teks yg diletakkan di bawah lembar kerja dokumen dan memiliki arti sama dengan header, perbedaanya hanya letaknya saja (header di bagian atas halaman, footer di bagian bawah halaman)- Save berfungsi untuk menyimpan dokumen.
- Save as berfungsi untuk menyimpan dokumen dengan nama lain, dan dapat pula berfungsi untuk mengganti ekstensi dokumen.
- Rename berfungsi untuk mengganti nama file.
- Delete berfungsi untuk menghapus satu karakter di sebelah kanan kursor.
- back space berfungsi untuk menghapus satu karakter di sebelah kiri kursor
- page layout ialah salah satu ribbon di ms. word 2007 dan berfungsi untuk menampung group-group tertentu.
- print preview adalah fitur pada ms. word yang dipakai untuk menampilkan dokumen sebelum dicetak.

# semoga membantuheader dan footer yaiut sebgai catatan yang terletak di atas dan dibwh kertas ,save= utk menyimpan dokument jika save as menyimpan file yang sdh tersimapan degn nama dan tempat yang berbeda,rename utk edit nama,delet utk mengapus carakters,backspas utk hapus beberapa carakter, dan print prewiw utk menapilan tampilan document yang mau di print sblm print out

40. dalam menu delete cells terdapat 4, sebutkan dan jelaskan fungsinya


- Shift Cells Left: untuk menghapus 1 cell, kemudian cell yang ada di samping kiri berpindah ke dalam cell yang tadi dihapus
- Shift Cells Up: untuk menghapus 1 cell, kemudian cell yang ada di atas berpindah ke dalam cell yang tadi dihapus
- Entire Row: untuk menghapus satu baris dari cell yang dipilih
- Entire Collumn: untuk menghapus satu kolom dari cell yang dipilih
