1. Tipe data STRING sama dengan tipe data….​


JAWABAN

Tipe data string merupakan salah satu jenis data yang ada dalam bahasa pemrograman. Kamu bisa mempelajarinya saat melakukan coding.

Dalam merancang MySQL Server, kamu perlu memperhatikan segala macam tipe data. Kemudian menganalisis tipe dari data yang akan disimpan pada tabel.

Secara umum, ada dua jenis tipe data, yaitu tipe string yang mendominasi karakter atau huruf, dan tipe string yang mendominasi angka.

Sebenarnya, ada berbagai tipe data yang berbeda di dalam bahasa pemrograman. Uniknya, dari beragam jenis tersebut memiliki fungsi yang sama, yaitu menyimpan satu jenis nilai pada suatu variabel. How To Tekno akan jelaskan tipe data string adalah di bawah ini.

Apa itu Tipe Data String?

Tipe data ini bisa menampung banyak karakter sekaligus. String adalah tipe data untuk teks yang terdiri dari gabungan huruf, angka, dan berbagai karakter. Fungsi ini digunakan untuk membuat identifier string atau teks.

4. Buat lah sebuah 3 fungsi overloading yang memiliki 1 parameter - Jika parameternya int maka outputnya “Tipe data int, hasilnya [parameter * 100]” - Jika parameternya double maka outputnya “Tipe data double, hasilnya [parameter * 1000]” - Jika parameternya string maka outputnya “Tipe data string, nilainya Hello [parameter]” contoh pada fungsi main : int main() { overloadingFunc(10; overloadingFunc(10.0); overloadingFunc("World); return 0; } outputnya : Tipe data int, hasilnya 10000 Tipe data float, hasilnya 1000 Tipe data string, nilainya Hello World


Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}
