4. Apa yang harus dilakukan jika ingin memindahkan data penjualan dari data yang telah dikumpulkan a klik tab file impor B klik tab file ekspor C klik tab insert dokumen D klik tab file add documet​


Jika Anda ingin memindahkan data penjualan dari data yang telah dikumpulkan, langkah-langkah yang tepat tergantung pada perangkat lunak atau platform yang Anda gunakan. Namun, berdasarkan deskripsi yang Anda berikan, berikut adalah langkah-langkah yang mungkin relevan:

1. Klik tab File Impor:

  - Jika Anda mengklik tab "File Impor", kemungkinan ada opsi untuk mengimpor data dari file eksternal ke dalam perangkat lunak atau platform yang Anda gunakan. Pilih opsi ini untuk memindahkan data penjualan dari file yang telah Anda kumpulkan ke dalam perangkat lunak.

2. Klik tab File Ekspor:

  - Jika Anda mengklik tab "File Ekspor", kemungkinan ada opsi untuk mengekspor data dari perangkat lunak atau platform yang Anda gunakan ke dalam file eksternal. Pilih opsi ini untuk memindahkan data penjualan dari perangkat lunak ke file yang dapat Anda simpan atau digunakan di tempat lain.

3. Klik tab Insert Dokumen:

  - Jika Anda mengklik tab "Insert Dokumen", kemungkinan ada opsi untuk menyisipkan dokumen ke dalam perangkat lunak atau platform yang Anda gunakan. Namun, ini tidak secara langsung berkaitan dengan memindahkan data penjualan. Jadi, dalam konteks memindahkan data penjualan, langkah ini mungkin tidak relevan.

4. Klik tab File Add Document:

  - Jika Anda mengklik tab "File Add Document", kemungkinan ada opsi untuk menambahkan dokumen ke dalam perangkat lunak atau platform yang Anda gunakan. Namun, ini juga tidak secara langsung berkaitan dengan memindahkan data penjualan.

Jika deskripsi langkah-langkah di atas tidak sesuai dengan perangkat lunak atau platform yang Anda gunakan, mohon berikan informasi lebih lanjut tentang jenis perangkat lunak atau platform tersebut. Dengan informasi tambahan, saya dapat memberikan langkah-langkah yang lebih spesifik dan sesuai dengan kebutuhan Anda.

20. apa yang di maksud dengan tab? sebutkan ikon ikon yang ada pada tab file​


Jawaban:

Tab adalah sederetan menu yang ditandai dengan teks

Ikon² yg ada pada tab file :

- Save, merupakan perintah untuk menyimpan file lembar kerja excel yang baru dibuat atau diedit, yg sebelumnya pernah atau belum pernah tersimpan di dokumen penyimpanan.

- Save As, merupakan perintah yang sama dengan Save, hanya saja jika Save as itu berfungsi untuk menyimpan file baru. Jadi, apabila file lama disimpan dengan tombol ini maka itu sama artinya dengan membuat file baru pada media penyimpanan.

- Open, merupakan perintah untuk membuka file Microsoft excel yg telah tersimpan di media penyimpanan.

- Close, merupakan perintah untuk menutup lembar kerja dan me-nonaktifkan seluruh tombol menu bar pada jendela Microsoft excel setelah itu kita baru bisa menutup jendela Microsoft excel dgn mengklik tombol close dipojok kanan atas.

- Options, merupakan ikon yang memberikan beberapa pilihan untuk mengatur lembar kerja excel.

- Exit, merupakan perintah untuk menutup jendela Microsoft excel

- Info, merupakan ikon yang berfungsi untuk memberikan informasi terkait lembar kerja yg kita gunakan.

- Recent, adalah ikon yang berfungsi untuk menampilkan file² yg baru saja kita buka/simpan.

- New, merupakan perintah untuk menampilkan lembar kerja baru yang masih kosong untuk dilakukan pengetikan dokumen baru.

- Print, merupakan perintah untuk melakukan penyetakan terhadap file yang kita kehendaki.

- Save & Send, merupakan perintah yg digunakan untuk menyimpan dan mengirimkan file² tertentu yang kita kehendaki.

- Help, merupakan ikon yg kita gunakan untuk mencari bantuan terhadap masalah yang timbul dari Ms. Excel

25. sebutkanlah tombol tombol yang terdapat dalam tab menu file kemudia jelaskanlah fungsinya masing masing!​


Jawaban:

1. New.

New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.

2. Open.

Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.

3. Save.

Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

4. Save as.

Save as adalah submenu dalam menu file yang fungsinya hampir sama dengan submenu save. Pengguna harus memilih mau diletakkan dimana dokumen atau file mereka. Pengguna juga akan bisa memilih dalam bentuk atau tipe apa dokumen atau file mereka yang akan disimpan. Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.

5. Print.

Print adalah submenu dalam menu file yang jika pengguna mengklik submenu tersebut berarti pengguna akan mencetak dokumen mereka. Dalam submenu print, terdapat beberapa hal untuk mengecek kembali apakah dokumen yang akan dicetak sudah benar atau belum. Fungsinya adalah untuk mencetak dokumen dengan printer.
